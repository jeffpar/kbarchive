DOCUMENT:Q210533  09-MAY-1999  [homegame]
TITLE   :Golf 99: Program Hangs Drawing Fifth Hole in Donald Ross Course
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbdisplay kbui kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Golf 1999 Edition, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start to play the fifth hole in the Donald Ross Memorial course in
Microsoft Golf 1999 Edition, the game may stop responding (hang) after drawing
the main view of the hole.

CAUSE
=====

This behavior can occur if all of the following conditions are true:

 - You select the Top or Top/Reverse inset view on the Game Preferences screen.

 - You select the Moderate 3 pin on the New Round screen.

 - You select the Blue 3 tee on the Edit Players screen.

RESOLUTION
==========

To prevent this issue from occurring, do not select an inset view on the Game
Preferences screen, or select a different pin or tee.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Golf 1999 edition,
version 1.0.

MORE INFORMATION
================

To resume playing Golf 99, press CTRL+ALT+DELETE to open the Close Program
dialog box, click Microsoft Golf 99, click End Task, and then restart Golf 99.

Additional query words: 1.00 msgame golf99 crash freeze lock
