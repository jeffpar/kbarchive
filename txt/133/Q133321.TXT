SUMMARY
=======

The following table lists fixed problems in Windows NT version 3.51. This is not
a comprehensive list and is current as of 21-Jul-1995. For more information
about each issue, query in the Microsoft Knowledge Base on the article
identifier (ID). For a current list of problems fixed in Windows NT version
3.51, query on KBFIX3.51 in the Microsoft Knowledge Base.
