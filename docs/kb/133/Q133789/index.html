Q133789: WT1229: Getting Started Controlling PowerPoint Through OLE

Article: Q133789
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbappnote kbfile kbole kbGraphxLink
Last Modified: 10-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft offers an Application Note titled "WT1229: Getting Started Controlling
Microsoft PowerPoint for Windows 95 Through OLE Automation" that provides
information to get you started on taking full advantage of the new object model
in the Microsoft PowerPoint presentation graphics program for Windows 95.

PowerPoint for Windows 95 is an OLE Automation server and can be controlled from
any program that is an OLE controller, such as the Microsoft Visual Basic
programming system, Microsoft Excel, Microsoft Project, or Microsoft Access.
What this means is that PowerPoint can become an integral member of any Office
solution that you or your company develops.

The following file is available for download from the Microsoft Download Center:

  DownloadDownload Wt1229.exe now
  (http://download.microsoft.com/download/powerpoint95/Document/1/WIN98/EN-US/wt1229.exe)


For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

posting to scan this file for viruses. Once posted, the file is housed on secure
