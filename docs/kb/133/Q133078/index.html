Q133078: Known Issues Scrolling in Chat Sessions

Article: Q133078
Product(s): The Microsoft Network
Version(s): 1.00 1.05 1.2 1.3
Operating System(s): 
Keyword(s): 
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0, 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists known issues with scrolling in chat sessions on The Microsoft
Network.

MORE INFORMATION
================

When you select text in the Chat History pane, you can select only the text
visible in the pane. That is, the pane does not scroll as you select. If you
want to copy and paste text, you must do so one pane at a time.

If you scroll back in the Chat History pane and then send a message, the Chat
History pane does not scroll down far enough to display the entire message that
you sent. If you use the scroll arrows to scroll enough to see the whole
message, the same behavior occurs the next time you send a message. To work
around this behavior, scroll a few lines past the end of the last message.

When you use the keyboard to move the insertion point up in the Chat History pane
and you reach the top of the pane, the insertion point disappears. This occurs
because the insertion point is still scrolling up, but into a non-visible
region. The text does not scroll as you move the insertion point up. Moving the
insertion point back down makes it reappear.
