Q133193: Black Screen Running Magic School Bus - Solar System Demo.

Article: Q133193
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the Magic School Bus - Solar System demonstration on the Windows 95
CD-ROM as part of Microsoft Exposition, the screen may turn black after the
music begins.

CAUSE
=====

You are using a color palette with more than 256 colors. The Magic School Bus -
Solar System demonstration does not work correctly with more than 256 colors.

NOTE: It also will not work correctly with less than 256 colors.

RESOLUTION
==========

Press ESC after the screen turns black to exit the demonstration.

To prevent the problem, change the color palette to 256 colors. To do so, follow
these steps:

1. Use the right mouse button to click the desktop, then click Properties on the
  menu that appears.

2. Click the Settings tab.

3. Change the Color Palette setting to 256 colors.

4. Click OK.

After you make this change, run the Magic School Bus - Solar System demonstration
again.
