Q133278: MSB Human: Freezes on Opening Animation on Macintosh

Article: Q133278
Product(s): Microsoft Home Kids Products
Version(s): MACINTOSH:1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 17-JUL-2001

1.00
MACINTOSH
kbenv kbprb kbfaq

-------------------------------------------------------------------------------
The information in this article applies to:

- Scholastic's Magic School Bus series: Explores the Human Body for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Magic School Bus Explores the Human Body for the Macintosh, the
opening animation sequence may stop and you must restart your computer.

This behavior occurs when 32-bit Memory Addressing has been disabled.

It may also occur if Virtual Memory is turned off on a computer with a small
amount of physical RAM installed.

RESOLUTION
==========

To resolve this problem, use the following steps:

Turn on 32-Bit addressing:
--------------------------

1. From the Apple menu, choose Control Panels.

2. Open the Memory Control Panel.

3. Turn 32-Bit Addressing on.

4. Restart your computer.

If you are still experiencing problems, perform the following steps:

Turn on Virtual Memory:
-----------------------

1. From the Apple menu, choose Control Panels.

2. Open the Memory Control Panel.

3. Turn Virtual Memory on.

4. Restart your computer.

After following these steps, you should be able to run Magic School Bus Explores
the Human Body successfully.


MORE INFORMATION
================

The Magic School Bus Human Body for Macintosh ReadMe file contains additional
information on how to improve the application's performance.
