Q133007: Flight Simulator 5.1: CD-ROM Add-Ons Readme.doc File Contents

Article: Q133007
Product(s): Microsoft Home Games
Version(s): 5.1
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

MS-DOS
kbusage kbsetup kbreadme kbref

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article contains the complete Readme.doc file, located in the
\Addons subdirectory of the Fight Simulator CD-ROM.

MORE INFORMATION
================

Japan Add-on
------------

Copy the contents of JAPAN directory into the SCENERY subdirectory of the
directory where you have the Japan Add-on installed.

For example, if you have the Japan Add-on installed in c:\FLTSIM5\JAPAN, and your
CD-ROM drive is D: copy d:\ADDONS\JAPAN\*.*
c:\FLTSIM5\JAPAN\SCENERY.

Caribbean Add-on
----------------

Copy the contents of CARIB directory into the SCENERY subdirectory of the
directory where you have the Caribbean Add-on installed.

For example, if you have the Caribbean Add-on installed in c:\FLTSIM5\CARIB and
your CD-ROM drive is D: copy d:\ADDONS\CARIB\*.*
c:\FLTSIM5\CARIB\SCENERY.

Europe 1 Add-on
---------------

Copy the contents of EUROPE1 directory into the SCENERY subdirectory of the
directory where you have the Europe 1 Add-on installed.

For example, if you have the Europe 1 Add-on installed in c:\FLTSIM5\EUROPE1 and
your CD-ROM drive is D: copy d:\ADDONS\EUROPE1\*.*
c:\FLTSIM5\EUROPE1\SCENERY
