Q97877: Bookshelf: Pronunciation Key Popup Window Resize

Article: Q97877
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,1991 edition,1992 edition,1993 edition,2.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf for Windows versions 1991 edition, 1992 edition, 1993 edition 
- Microsoft Multimedia Viewer, versions 1.0, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you resize the main Bookshelf window so that it is smaller than the size of
the Pronunciation Key Popup window in the American Heritage Dictionary, the
popup will also make itself smaller. The shorter you make the main Bookshelf
window, the shorter the popup becomes.

This behavior is by design. Microsoft Multimedia Viewer (the program that
displays the popups and screens for Bookshelf) resizes the popup so that it
always fits within the main Bookshelf window.

MORE INFORMATION
================

Steps to Reproduce Situation
----------------------------

1. Select a word from the American Heritage Dictionary.

2. Select the pronunciation of the word to bring up the Pronunciation Key popup
  window.

3. Note the size of this window.

4. Cancel the popup window and resize the American Heritage window to a
  significantly shorter size.

5. Select the pronunciation of the word and note the shorter Pronunciation Key
  popup window.
