Q170349: Options Command on View Menu Does Not Work in Internet Explorer

Article: Q170349
Product(s): Microsoft Windows NT
Version(s): 1.0
Operating System(s): 
Keyword(s): kbui
Last Modified: 21-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Options on the View menu in Microsoft Internet Explorer, no
dialog box may be displayed.

CAUSE
=====

The behavior can occur for any of the following reasons:

- The Zero Administration Kit (ZAK) Appstation or Taskstation that you are
  using has had this functionality disabled by the administrator.

- The system administrator has disabled this functionality using a policy
  provided with the Microsoft Internet Explorer Administration Kit (IEAK).

RESOLUTION
==========

Contact your System Administrator to enable this functionality.
