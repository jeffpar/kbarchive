Q170943: WD97: Clicking a Suggestion in Spelling And Grammar Causes IPF

Article: Q170943
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta kbproof word8 word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you check grammar in a Microsoft Word document, and you click in the
Suggestions box in the Spelling And Grammar dialog box, you may receive the
following error message:

  This program has performed an illegal operation and will be shut down.

  If the problem persists, contact the program vendor.

When you click Details (on Microsoft Windows Millennium Edition, press ALT+D),
you receive the following message:

  WINWORD caused an invalid page fault in WINWORD.EXE at 0137:3039e560.

NOTE: The actual memory address may vary.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.<A0>This problem was corrected in Microsoft
Word 2000.

MORE INFORMATION
================

For additional information about checking the grammar in a Word document, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q167655 WD97: Frequently Asked Questions About the Grammar Checker
