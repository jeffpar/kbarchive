Q170277: XCLN: Setup Repeatedly Prompts for the Same Diskette

Article: Q170277
Product(s): Microsoft Exchange
Version(s): WINDOWS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 27-JUN-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Macintosh client, version 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

After creating a floppy disk Setup from the Microsoft Exchange Macintosh client
5.0 CD and starting Setup from the newly created installation disk 1, the Setup
program may prompt for installation disk 3 or 4. After inserting the requested
disk, Setup will again prompt for the same disk. The only option available is to
cancel Setup.

WORKAROUND
==========

From the Macintosh client, run Setup for the Exchange Client from the Macintosh
Exchange client 5.0 CD.

-or-

From the Macintosh client, copy the Setup files to a MacFile volume on a Windows
NT Server and run Setup over the network.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Macintosh
client, version 5.0.

Additional query words: Disk System Apple XCLNMac
