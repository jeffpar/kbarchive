Q50699: Resetting the Screen to Graphics Mode in Flight Simulator

Article: Q50699
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

You can reset the screen to graphics mode in Flight Simulator by performing the
following steps:

1. Press 1 to access the Mode menu.

2. Press 4 to access the Review Logbook option.

3. Press 1 to use the Logbook Edit option.

4. Enter a name.

5. Press "C" to cancel request to log flight. This resets the program to
  graphics mode.
