Q50298: Standard Emulation for Star Micronics Printers

Article: Q50298
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11; Win95:
Operating System(s): 
Keyword(s): 
Last Modified: 19-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
- Microsoft Windows 95 
-------------------------------------------------------------------------------


SUMMARY
=======

This article contains the a device compatibility list (according to Star
Micronics) for Star Micronics printers.

MORE INFORMATION
================

  Star Printer         Emulation 1          Emulation 2
  ------------         -----------          -----------

  Laser8               HP LaserJet Plus     Epson EX800

  XB-2410/2415+        Windows Driver       IBM Proprinter X24
  NX-2400              Epson LQ             IBM Proprinter X24
  XR-1000/1500         Epson EX             IBM Proprinter II
  XR1020/1520          Epson EX800/EX1000   IBM Proprinter III
  NX-1000 Rainbow*     Epson JX             IBM Proprinter II
  NX-1000*             Epson JX             IBM Proprinter II
  NX-1000 II+          Windows Driver

  NB-15*               Epson LQ             IBM Graphics
  NB24-10/15*          Epson LQ             IBM Graphics/IBM Proprinter
  NR-10/15*            Epson LQ             IBM Graphics/IBM Proprinter
  ND-10/15*            Epson LQ             IBM Graphics/IBM Proprinter
  NL-10*               Epson LQ             IBM Graphics
  NX-10/15*            Epson LQ             IBM Graphics

  LV-2010/2015*        IBM Graphics
  LV-1610/1615*        IBM Graphics
  LV-1210/1215*        IBM Graphics

  SB-10*               None
  SR-10/15*            IBM Graphics

  Powertype*           None
