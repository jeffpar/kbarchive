Q173257: Where Are the Practice Files in VB Deluxe Learning Edition?

Article: Q173257
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Visual Basic 5.0 Deluxe Learning Edition ISBN 1-57231-551-2 
- MSPRESS Microsoft Visual Basic 6.0 Deluxe Learning Edition ISBN 1-57231-873-2 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes where to find the practice files for each component of
"Microsoft Visual Basic Deluxe Learning Edition."

MORE INFORMATION
================

"Microsoft Visual Basic Deluxe Learning Edition" is divided into three parts:

- Microsoft Visual Basic Learning Edition Version (program)

- Learn Visual Basic Now (program)

- "Microsoft Visual Basic Step by Step" (book)

The Microsoft Visual Basic Learning Edition and Learn Visual Basic Now programs
are contained on two CD-ROMS in a single CD jewel case. To use the Learn Visual
Basic Now CD-ROM, open the jewel case. Grasp the black CD holder and carefully
lift it from the jewel case. The Learn Visual Basic Now CD-ROM is fastened to
the back of the CD holder.

To install Learn Visual Basic Now, follow the steps written on the CD-ROM.

The following information applies only to version 5.0 of the Microsoft Visual
Basic Deluxe Learning Edition.

"Microsoft Visual Basic 5.0 Step by Step" includes a third CD-ROM with practice
files for use with the tutorials in the book. This third CD-ROM can be found in
a cardboard sleeve between the last page and the back cover of the "Microsoft
Visual Basic 5.0 Step by Step" text.

To install the "Microsoft Visual Basic 5.0 Step by Step" practice files, follow
the steps on pages xvii-xxiii.

The following information applies only to version 6.0 of the Microsoft Visual
Basic Deluxe Learning Edition.

The files for "Microsoft Visual Basic 6.0 Step by Step" are located on the Learn
Visual Basic 6.0 Now CD-ROM. To install these files, please follow the
directions on page xv of "Microsoft Visual Basic 6.0 Step by Step."

The Microsoft Visual Basic 6.0 Deluxe Learning Edition also includes two MSDN
CD-ROMs in a single jewel case. To install MSDN using these CD-ROMs, please read
the "Getting Started" section in the booklet enclosed in the MSDN jewel case.
