Q173256: Flight Simulator for Win95 Inside Moves, Corrections

Article: Q173256
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 22-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Flight Simulator for Windows 95: Inside Moves 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains corrections, comments, and information on known errors
relating to the Microsoft Press book "Microsoft Flight Simulator for Windows 95:
Inside Moves."

Contents:

- Page 222: Incorrect navigation chart

- Page 277: 2 lines missing from bottom of page

MORE INFORMATION
================

Page 222
--------

Page 222 should show the navigation and airport charts for Newark International
Airport (EWR). The airport chart is correct. However, the navigation chart is
the navigation chart for New Orleans International Airport (MSY), repeated from
page 221.


Page 277
--------

The last sentence on page 277 is not continued on page 278. Two lines of text are
missing.

The missing lines of text are:

737-400 featured in Flight Simulator uses such an engine, while the Learjet 35A
sticks with the basic turbojet.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
