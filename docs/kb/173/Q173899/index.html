Q173899: XWEB: OWA Points to Non-Local Exchange Server Computer

Article: Q173899
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 09-MAR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======


When you install Microsoft Exchange Server with the Outlook Web Access (OWA)
component, Setup will by default point OWA to the local computer. But, if you
are running custom setup and installing OWA alone, Setup prompts you for a
server name to point to. If you choose later to install Exchange Server on this
computer, Setup will not change the Server that was selected in the previous
step. As a result, when you run OWA it will not point to the local Exchange
Server computer. This behavior is by product design.

For additional information about how to point OWA to a new Exchange server, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q244850 XWEB: OWA Does Not Start When Server Is Removed from Site
