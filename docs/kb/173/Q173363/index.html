Q173363: Earlier MSN Sign-In Screen Appears When You Connect to the Web

Article: Q173363
Product(s): The Microsoft Network
Version(s): 2.5,2.51,2.52,2.6,3.0,3.01,3.02,4.0,5,5.01,5.5
Operating System(s): 
Keyword(s): kbenv
Last Modified: 13-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 2.5, 2.51, 2.52, 2.6 
- Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0, 5 for Windows 95 
- Microsoft Internet Explorer versions 5, 5.01, 5.5 for Windows 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to connect to a Web address with either MSN, The Microsoft
Network, or Microsoft Internet Explorer, the Sign-In screen for an earlier
version of MSN may appear.

CAUSE
=====

This issue can occur if Internet Explorer is not configured to use the MSN
Dial-Up Networking connection.

RESOLUTION
==========

To resolve this issue, configure Internet Explorer to use the MSN Dial-Up
Networking connection. To do so, use the following steps:

Internet Explorer 3.0, 3.01, 3.02
---------------------------------

1. Right-click the Internet icon on the desktop, and then click Properties.

2. On the Connection tab, click the Connect to the Internet as needed check box
  to select it.

3. In the Use the following Dial-Up Networking connection box, click MSN, and
  then click OK.

Internet Explorer 4.0
---------------------

1. Right-click the Internet Explorer icon on the desktop, and then click
  Properties.

2. On the Connection tab, click Connect to the Internet using a modem.

3. Click Settings, and then click MSN in the Use the following Dial-Up
  Networking connection box.

4. Click OK, and then click OK again.

Internet Explorer 5.x
---------------------

1. Right-click the Internet Explorer icon on the desktop, and then click
  Properties.

2. On the Connection tab, and under Dial-up Settings, select one of the
  following options:

   - dial whenever a network connection is not present (use if you switch
     between using a LAN and Modem to connect)

   - always dial my default connection (if you only want to connect by
     modem)with the MSN connectoid you wish to use for the connection selected
     as the default connection.

3. Click OK, and then click OK again.
