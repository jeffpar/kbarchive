Q173732: PPT7: Text Prints Slanted, Truncated to Tektronix Phaser Printer

Article: Q173732
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kb3rdparty kbprint kbdta
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print to the Tektronix Phaser color laser printer, using the Phaser 320
printer driver, the text on the printout is slanted and truncated.

CAUSE
=====

This problem occurs when you use Comic Sans or Lucida Sans fonts.

RESOLUTION
==========

Install the Tektronix Phaser 540 color printer driver instead of the 320 driver.

REFERENCES
==========

For additional information, please see the following article in the Microsoft
Knowledge Base:

  Q162306 PowerPoint Articles Available by Fax or E-Mail: Printing

The Phaser color laser printer is manufactured by Tektronix, a vendor independent
of Microsoft; we make no warranty, implied or otherwise, regarding this
product's performance or reliability.
