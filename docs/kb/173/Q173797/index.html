Q173797: Toshiba 300CT Laptop Computer Uses a Windows Modem

Article: Q173797
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): kbhw win95 kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The modem built into the Toshiba 300CT laptop computer is a Windows modem. The
modem works only with Windows 95.

MORE INFORMATION
================

Other operating systems may offer support for this modem at a later date. To
determine if this modem is supported by an operating system, consult with the
manufacturer of the operating system. If you need to use an operating system
other than Windows 95 with the Toshiba 300CT laptop computer, Toshiba recommends
installing a Personal Computer Memory Card International Association (PCMCIA) or
PC Card modem. For information about installing a PCMCIA or PC Card modem into
the Toshiba 300CT laptop computer, contact Toshiba or refer to the documentation
included with the computer.
