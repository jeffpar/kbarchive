Q76874: MSCDEX CD-ROM Driver and Windows with Multimedia Extensions

Article: Q76874
Product(s): Miscellaneous Windows Products
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 20-DEC-2000

1.00

WINDOWS

kbmm

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows with Multimedia Extensions, version 1.0 
-------------------------------------------------------------------------------

The MSCDEX driver is a file system driver for MS-DOS that provides
extensions for CD-ROM hardware. It is suggested that version 2.2 of
this driver be used in conjunction with Windows with Multimedia
Extensions version 1.0. However, Windows with Multimedia Extensions
version 1.0 only requires version 2.0 or later of the MSCDEX CD-ROM
driver.

This information does not apply to later versions of Windows.

This driver is NOT supplied by Microsoft. Individual hardware
manufactures supply this driver. Please contact the manufacturer of
the CD-ROM drive for any questions or updates to this driver.
