Q76992: International and National Standard Paper Sizes

Article: Q76992
Product(s): Microsoft Windows Printing Issues
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

The following is a list of International and National standard paper dimensions
available for use in printers and commercial typesetters:

  Description           Inches                         Cm
                   Width     Length              Width     Length
  ---------------------------------------------------------------
  A3               11.583    16.38               29.7      42.0
  A4                8.27     11.69               21.0      29.7
  A5                5.772     8.19               14.8      21.0
  B5                7.098    10.02               18.2      25.7
  Com-10 / 
    Envelope #10    4.125     9.5                10.4775   24.13
  DL                4.29      8.58               11.0      22.0
  Executive         7.5      10.0                19.05     25.4
  Folio             8.5      13.0                21.59     33.02
  Ledger           17.0      11.0                43.18     27.94
  Legal             8.5      14.0                21.59     35.56
  Letter            8.5      11.0                21.59     27.94
  Monarch           3.875     7.5                 9.8425   19.05
  Note              8.5      11.0                21.59     27.94
  Quarto            8.385    10.73               21.5      27.5
  Tabloid          11.0      17.0                27.94     43.18

MORE INFORMATION
================

Conversion equivalent: 1 inch = 2.54 centimeters

  1 centimeter = 0.39 inches
