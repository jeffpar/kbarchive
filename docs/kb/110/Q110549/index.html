Q110549: Flight Simulator: Flashing Screen with Orchid Fahrenheit 1280

Article: Q110549
Product(s): Microsoft Home Games
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

kbusage

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

You may receive a flashing screen when using the Orchid Fahrenheit 1280 video
card with Flight Simulator 5.0 in SVGA mode.

MORE INFORMATION
================

The Orchid Fahrenheit 1280 is based on the S3 chipset and will function using
Flight Simulator's S3 86C911 option; however, the refresh rate must be
calibrated correctly to do so. To set the refresh rate for the Fahrenheit 1280,
use Orchid's FSCAN.EXE utility that ships with the Fahrenheit 1280. For more
information about the FSCAN.EXE utility and setting refresh rates, consult the
Fahrenheit 1280 user's guide.

Orchid's phone numbers:

- Orchid Technology (510) 683-0323 technical support
- Orchid Technology (510) 683-0327 2400 baud BBS
