Q110820: Mac Wkst: Resizable Forms Available with Upgrade to 3.1d

Article: Q110820
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:3.1d
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, version 3.1d 
-------------------------------------------------------------------------------

SUMMARY
=======

With version 3.1d of Microsoft Mail for AppleTalk Networks, all workstations
automatically have resizable Note and 80-Column forms available.

The resizable forms automatically appear with no modifications to any
workstations.

MORE INFORMATION
================

A feature of version 3.1d of Microsoft Mail is the availability of resizable
Note and 80-Column forms. These forms are provided to the workstation by the
server.

Because of this feature, all workstations connecting to a version 3.1d server
automatically receives the resizable forms -- there is no need to upgrade the
workstations to receive this new feature.

The Note form can be resized both vertically and horizontally.

The 80-Column form can only be resized vertically. The 80-Column form is designed
to be a fixed-width form and therefore does not allow horizontal resizing.
