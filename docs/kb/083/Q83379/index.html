Q83379: CorelDRAW! Gives WALDO Error Using Extrude or Blend

Article: Q83379
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 12-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When you try to perform the extrude or blend functions in CorelDRAW! version 2.0
with Windows version 3.1, a problem occurs in CorelDRAW!'s WALDO program module.
The following error message displays:

  WALDO has created a fault at . . .

You cannot perform the extrude or blend functions, but you have a choice of
proceeding anyway or exiting the program. This problem occurs only in CorelDRAW!
version 2.0 and was corrected in version 2.001.


MORE INFORMATION
================

Corel Corporation has confirmed this to be a problem with CorelDRAW! 2.0 running
under Windows 3.1 and provides an upgrade option to registered users of
CorelDRAW!.

For more information on this upgrade, contact Corel Technical Support.

Additional query words: 3.10 3.11 3.1 Corel Draw
