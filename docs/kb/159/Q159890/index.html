Q159890: Mac Encarta Setup: QuickTime Musical Instrument Error

Article: Q159890
Product(s): Microsoft Home Multimedia Titles
Version(s): 1996 edition
Operating System(s): 
Keyword(s): kberrmsg kbmm kbsetup
Last Modified: 24-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Macintosh 
- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to install Microsoft Encarta Encyclopedia for the Macintosh, you
receive the following error message:

  "The file "QuickTime Musical Instruments" on disk "<diskname>" cannot
  be replaced by the installer. No installation will take place. Quit the
  installer, move the file to another folder and try again."

After you click OK, you return to the Easy Install dialog box.

CAUSE
=====

You may have a later version of QuickTime installed.

RESOLUTION
==========

To resolve the problem and install Encarta, do the following:

1. Double-click the Install Encarta Encyclopedia icon on the Encarta CD-ROM.

  You should get an Easy Install dialog box.

2. Click the Customize button (located at the bottom left-hand corner of the
  dialog box).

3. Press and hold down the SHIFT key, and then click the items you want to
  install.

  NOTE: Do NOT select QuickTime.

4. Click Install.

Encarta 96 should now install and run correctly.

MORE INFORMATION
================

Encarta should work with later versions of QuickTime Musical Instruments.

NOTE: This error message can occur if you have Microsoft Encarta Encyclopedia for
the Macintosh, 1997 edition, already installed when you try to install 1996 or
1995 edition. Encarta 97 installs QuickTime 2.5.
