Q159758: WD97: Caption Text Shifted Right or Missing from Caption Box

Article: Q159758
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): 
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Caption box is empty or the label text is shifted to the right. (To locate
this option, click Caption on the Insert menu.)


CAUSE
=====

Either the normal style or the caption style is formatted with center or right
alignment.

WORKAROUND
==========

To work around this problem, format the normal style or caption style with left
paragraph alignment and manually format the caption text with right or center
alignment. To change the style formatting, use these steps:

1. On the Format menu, click Style.

2. In the List box, click All Styles.

3. In the Styles list, select Normal (or Caption).

4. Click Modify.

5. In the Modify Style dialog box, click Format.

  NOTE: To add the style change to your template, click to select the Add To
  Template check box, and then click the Format button.

6. Click Paragraph.

7. In the Paragraph dialog box, click the Indents and Spacing tab.

8. In the Alignment drop-down box, select Left.

9. Click OK two times.

10. Click Close.

11. Apply direct formatting to the caption text. To do this, select the caption
  and then click the Centered or Right Align button on the Formatting toolbar.

STATUS
======

Microsoft has confirmed this to be a problem in the products listed at the
beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
