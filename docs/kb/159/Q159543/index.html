Q159543: Intel ProShare PC Card May Hang Computer

Article: Q159543
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 2.0
Operating System(s): 
Keyword(s): kbhw osr2 win95 kbHardwarekbfixlist
Last Modified: 25-MAR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 OEM Service Release, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you have a computer that provides CardBus hardware support, the computer
may stop responding (hang) unexpectedly when you are using an Intel ProShare PC
Card.

CAUSE
=====

With some CardBus controllers, the computer hangs when it is accessing memory on
the CardBus card if certain control bits are set on the controller.


RESOLUTION
==========


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.

MORE INFORMATION
================
