Q175690: The Winmsd.exe Program Shows Multiple Users Are Logged On

Article: Q175690
Product(s): Microsoft Windows NT
Version(s): WINNT:4.0
Operating System(s): 
Keyword(s): kbnetwork kbtool
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view the Network tab in the Windows NT Diagnostics program for Windows
NT 4.0, there may appear to be more than one user logged on to your computer.

RESOLUTION
==========

To work around this problem, restart your computer.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation version
4.0, and Windows NT Server version 4.0.
