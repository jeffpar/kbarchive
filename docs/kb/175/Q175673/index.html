Q175673: How to Modify Private Information in Microsoft Wallet

Article: Q175673
Product(s): The Microsoft Network
Version(s): WINDOWS:2.5,4.0
Operating System(s): 
Keyword(s): kbtool kbmsn
Last Modified: 14-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.5 
- Microsoft Internet Explorer version 4.0 for Windows 95 
- the operating system: Microsoft Windows 98 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Wallet provides a secure place to store the private information you
use to shop on the Internet. To modify your private information in the wallet,
use the following steps:

1. Right-click the MSN icon on the desktop, and then click Properties.

2. Click the Content tab.

3. Under Personal Information, do the following:
   - Click Addresses to add, edit, or delete the addresses stored in the
     wallet.

   - Click Payments to add, edit, or delete the credit cards and payment
     methods available to the wallet.

4. Click OK or Close until you return to Windows.

Additional query words: 4.0 msnet msnetwork address payment selector
