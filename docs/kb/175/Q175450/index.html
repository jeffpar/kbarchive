Q175450: INFO: Visual Basic 5.0 Fixes in Visual Studio 97 Service Pack 3

Article: Q175450
Product(s): Microsoft Visual Basic for Windows
Version(s): WINDOWS:5.0
Operating System(s): 
Keyword(s): kbVBp500 kbVS97sp1fix kbVS97sp2fix kbFAQ kbhowto kbvbp500sp1fix kbvbp500sp2fix kbvbp500
Last Modified: 18-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Learning Edition for Windows, version 5.0 
- Microsoft Visual Basic Professional Edition for Windows, version 5.0 
- Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Visual Studio 97 Service Pack 3 contains several fixes for Visual
Basic 5.0. This article lists all of the Microsoft Knowledge Base articles
documenting bug fixes that affect Visual Basic 5.0.

MORE INFORMATION
================

The following list contains the updated components and the bugs that were fixed
for Visual Basic 5.0 in the Visual Studio 97 Service Pack 3. You can find all
these articles quickly in the Microsoft Knowledge Base by searching on
VB5FixlistSP3. If you are only interested in a specific article below, you can
find it in the Microsoft Knowledge Base by searching on the article ID (for
instance, Q171546.)
