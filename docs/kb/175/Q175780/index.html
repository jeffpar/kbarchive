Q175780: XCON: Sending to Nested DLs Fails

Article: Q175780
Product(s): Microsoft Exchange
Version(s): WinNT:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 13-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

In a scenario where two Microsoft Exchange Server sites are connected using an
X.400 Connector with no replication, sending mail to recipients that are in
nested distribution lists may randomly fail. The sending Exchange Server
computer reports DS errors and the receiving side reports ASN.1 decoding errors.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.0. This problem has been corrected in the latest U.S. Service Pack for
Microsoft Exchange Server version 5.0. For information on obtaining the Service
Pack, query on the following word in the Microsoft Knowledge Base (without the
spaces):

  S E R V P A C K
