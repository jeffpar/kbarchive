Q175750: How to Install the Full Installation of Internet Explorer 4.0

Article: Q175750
Product(s): The Microsoft Network
Version(s): WINDOWS:2.51,4.0
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 05-JAN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.51 
- Microsoft Internet Explorer version 4.0 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install MSN, The Microsoft Network, version 2.51, Microsoft Internet
Explorer 4.0 is installed with only the basic components needed to run MSN. To
install the Active Desktop and the additional communication and multimedia
components of Internet Explorer 4.0, you must install the full installation of
Internet Explorer 4.0.

MORE INFORMATION
================

To install the full installation of Internet Explorer 4.0, which includes the
Active Desktop and additional communication and multimedia components, follow
these steps:

1. Restart the computer. When you see the "Starting Windows 95" message, press
  the F8 key, and then choose Command Prompt Only from the Startup menu.

2. Start Windows 95 by typing "win" (without the quotation marks) and holding
  down the SHIFT key for the duration of Windows startup. This prevents any
  programs from loading automatically at startup.

3. Press CTRL+ALT+DELETE to open the Close Program dialog box.

4. Quit all programs except Explorer and Systray (which are parts of Windows
  95). To quit a program, click the program, and then click End Task.

5. Repeat steps 3-4 until you have quit all programs except Explorer and
  Systray.

6. Disable any anti-virus or disk tool programs installed on the computer. For
  information about how to disable these programs, see the printed or online
  documentation for the program.

7. Insert the MSN 2.51 CD-ROM into the CD-ROM drive. If the Auto Insert
  Notification option is turned on, close the Welcome To MSN window that opens
  on the desktop, and then click Cancel to exit MSN Setup. When you are
  prompted to confirm you want to exit Setup, click Yes.

8. Double-click My Computer.

9. Right-click the MSN (CD-ROM drive) icon, and then click Explore.

10. Double-click the Ie40 folder.

11. Double-click the Ie4setup.exe file.

12. Follow the instructions on the screen. When you are prompted to select the
  installation option you prefer, click Full Installation, and then click
  Next.

13. Click Upgrade Only Newer Items, and then click OK.

14. If you are prompted to restart the computer, click Yes.
