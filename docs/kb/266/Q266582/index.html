Q266582: Encarta Researcher: Researcher Window Is Repeatedly Displayed

Article: Q266582
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbui kbimu
Last Modified: 27-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Africana, Third Edition 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Interactive World Atlas 2001 
- Microsoft Encarta Reference Suite 2001 
- Microsoft Encarta World English Dictionary 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you add information to Microsoft Researcher in any of the programs listed
at the beginning of this article, you may be prompted with the "Add to
Researcher" window each time you add information.

CAUSE
=====

This behavior can occur if you quit Microsoft Internet Explorer before you add
all the information to Researcher.

RESOLUTION
==========

To resolve this issue, do not quit Internet Explorer until you add all the
information that you want to Researcher.
