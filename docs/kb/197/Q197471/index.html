Q197471: WD97: Options That Are Unavailable in Print Preview

Article: Q197471
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

Although you can edit your document in print preview, it is an "exclusive" mode,
so you are unable to perform any action that switches focus out of print
preview.

For example, you cannot click New on the File menu while in print preview. You
must click Close on the File menu before clicking New.

MORE INFORMATION
================

The following options are unavailable (dimmed) while in print preview:

  Menu      Commands unavailable
  --------------------------------------------------

  File      New
            Open
            Versions
            Send To: Mail Recipient
                     Routing Recipient
                     Exchange Folder
                     Fax Recipient
            MRU list (Most recently used file list)

  Edit      Paste As Hyperlink
            Object

  Tools     Mail Merge
            Merge Documents
            Templates and Add-Ins
  Table     (Show) Gridlines

  Window    Everything

In addition, if you click Envelopes And Labels on the Tools menu, the New
Document button is unavailable on the Labels tab.
