Q188537: WD97: Styles Not in Alphabetical Order in Style List

Article: Q188537
Product(s): Word 97 for Windows
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbList kbdta kbwordvba
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

Styles are not listed alphabetically in the Style list on the Formatting
toolbar.

CAUSE
=====

This behavior is by design.

The styles are listed in predefined groups. The first group includes the caption,
heading and title styles. The second group consists of the normal style. The
third group lists any user-defined styles. However, the third group does not
include preinstalled styles that you have modified. The fourth group is composed
of block text, body text, and plain text styles. The fifth group is made up of
the list bullet and list number styles. The sixth group includes all other
preinstalled styles. The styles are listed alphabetically within each of these
groups.

The following list includes all of the normal, predefined styles. To display all
predefined styles in the Style list, hold down SHIFT and click the drop-down
button for the list.
