Q111065: Flight Simulator: No Sound in Some New York Add-On Situations

Article: Q111065
Product(s): Microsoft Home Games
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

kbother

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Some of the situations that ship with the New York Scenery add-on do not have
any sound.

RESOLUTION
==========

Pressing the Q key on the keyboard re-enables sound. These situations were saved
with the Mute option enabled.

This occurs with the following situations:

+----------------------------------------------+
| Situation                     | File         | 
+----------------------------------------------+
| NYC - Follow the Hudson River | NYC-FOLL.STN | 
+----------------------------------------------+
| NYC - Statue of Liberty       | NYC-STAT.STN | 
+----------------------------------------------+


Additional query words: 5.00 flightsim fltsim dos NY FS5 muted silenced disabled
