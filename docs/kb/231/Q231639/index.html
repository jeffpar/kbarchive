The information in this article applies to:

- MSPRESS Academic Learning Series Microsoft Windows NT Network Administration ISBN 1-57231-913-5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book "Academic Learning Series Microsoft Windows
NT Network Administration", ISBN 1-57231-913-5.

The following topics are covered:

- Page 302: Incorrect answer to question 11

MORE INFORMATION
================

Page 302: Incorrect Answer to Question 11
-----------------------------------------

The answer to question 11 states that you should be able to purge print documents
with the Manage Documents permission. However, this is incorrect.

Replace answer with:
"No, because the Manage Documents permission does not give you permission to
purge documents."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
