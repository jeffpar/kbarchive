Q154918: MicroGrafx Draw 4.0 Prints Garbled Output to PostSript Printer

Article: Q154918
Product(s): Microsoft Windows NT
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kb3rdparty kbprint kbPrinting
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print from MicroGrafx Draw 4.0 in Windows NT 4.0 to a PostScript
printer, you may receive garbled or irregular output. The output can be anything
from blank paper to an incorrectly spaced font.


STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.


MORE INFORMATION
================

MicroGrafx Draw 4.0 is manufactured by MicroGrafx, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
