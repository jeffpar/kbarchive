Q162596: WD97: Products That Qualify for the Word Upgrade

Article: Q162596
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbpolicy kbsetup kbdtakbfaq
Last Modified: 20-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Word 97 is available in two versions: a retail version which can be
installed on any compatible computer, and an upgrade version which requires that
you have a qualifying product installed on your computer or that you have Disk 1
from the qualifying product available when you run the Setup program.

NOTE: The retail version does not require a qualifying product.

This article lists the products that qualify you to use the upgrade version of
Microsoft Word 97.

NOTE: This list applies to the U.S. English version of Microsoft Word 97 only.

When you install Microsoft Word 97, the Setup program may not correctly recognize
that you have one or more of the qualifying products installed on your computer.
If this occurs and you are unable to install Microsoft Word 97, contact
Microsoft Product Support Services. When you speak to a support professional,
inform them that you have a qualifying product as listed in Microsoft Knowledge
Base article Q162596.

For information about how to contact Microsoft Support, see the following Web
site:
