Q162569: Monster Truck Madness: Patch is for the English Version Only

Article: Q162569
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbsetup kbimu
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Monster Truck Madness, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Monster Truck Madness (MTM) patch is only for the English version of Monster
Truck Madness. The following article discusses the results of installing the MTM
patch over a localized version of the program.

MORE INFORMATION
================

Installing the patch over a localized version will have the following impact:

If you install it over the French or German version of MTM
----------------------------------------------------------

- The graphical areas of the user interface (UI) will remain in the localized
  language.

- Menus and dialog boxes will be in English, but the yellow buttons will remain
  in the localized language.

- During game play, pit board overlays will be in English.

- The English version of DirectX 3.0 is installed, which causes the connection
  types to be in English.

- The Help file is not affected.

- The graphical multiplayer dialogs remain in the localized language.

- The loading dialog box and status text will be in English.

- The commentary overlay text and audio will remain in the localized language.

If you install it over the Japanese version of MTM
--------------------------------------------------

- Monster Truck Madness will not work because the English version is not
  double-byte character set (DBCS) enabled.
