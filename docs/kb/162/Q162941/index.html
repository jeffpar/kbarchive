Q162941: Flight Simulator 95: Missing Boeing 737-400 Adventure

Article: Q162941
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

6.0
WINDOWS
kbusage kbbug kbother

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for Windows 95, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

In the Preflight Briefings section of Pilot's Help in Flight Simulator for
Windows 95, there is a listing for "Boeing 737-400 Introductory Flight" under
Adventures. This flight does not exist on the Flight Simulator compact disc.

MORE INFORMATION
================

The Boeing 737-400 Introductory Flight is listed on the Flight Simulator Help
menu under Pilot's Help, Preflight Briefings, Adventures. The details of the
flight are included in the Help topic, but the actual adventure is missing from
the Flight Simulator compact disc.

This Adventure was not included on the compact disc because of quality issues. It
was pulled from the disc just prior to shipping, but the Help file was not
updated.
