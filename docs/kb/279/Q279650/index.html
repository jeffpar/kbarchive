Q279650: Microsoft Casino: Dealer Incorrectly States Two of a Kind

Article: Q279650
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu msgame
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Casino, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Caribbean Stud in the Bellagio casino in Microsoft Casino, the
dealer's voice may incorrectly state that the dealer's hand contains two of a
kind.

CAUSE
=====

This behavior occurs when the dealer's hand contains two pairs.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Casino, version 1.0.

MORE INFORMATION
================

Despite this incorrect statement, the dealer pays out correctly at the end of
the hand. This issue does not affect game play in any way.
