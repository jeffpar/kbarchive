Q174847: Arcade for the Macintosh Hangs or Quits

Article: Q174847
Product(s): Microsoft Home Games
Version(s): MACINTOSH:1.0
Operating System(s): 
Keyword(s): kbenv kbusage
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Arcade for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start one of the programs that come with Arcade for the Macintosh, the
program freezes or quits unexpectedly.

CAUSE
=====

Arcade for the Macintosh may be running on a version of the Macintosh Operating
System (Mac OS) that is newer than version 7.5.5. The version of Mac OS can be
determined by selecting "About This Computer..." from the Apple menu.

Arcade was designed for System 7.0 and you may experience compatibility issues
when you attempt to run it on more advanced operating systems.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Arcade for the
Macintosh, version 1.0.

MORE INFORMATION
================

The games included with Arcade for the Macintosh are:

- Asteroids
- Battlezone
- Centipede
- Missle Command
- Tempest
