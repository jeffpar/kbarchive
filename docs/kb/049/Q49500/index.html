Q49500: List of Antivirus Software Vendors

Article: Q49500
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): 
Operating System(s): 
Keyword(s): kbdtakbfaq
Last Modified: 06-AUG-2002

SUMMARY
=======

This article provides a list of antivirus software vendors.

MORE INFORMATION
================

Antivirus software is software that is specifically designed to detect and
prevent viruses. Microsoft recommends that you use antivirus software on your
computer. This article contains a list of independent antivirus software
vendors.

For a list of antivirus products that are designed to work with Microsoft
Exchange Server, visit the following Microsoft Web site:

  http://www.microsoft.com/exchange/partners/antivirus.asp
