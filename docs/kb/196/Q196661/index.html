Q196661: Windows NT 4.0 Does Not Support Universal Serial Bus

Article: Q196661
Product(s): Microsoft Windows NT
Version(s): 2000,4.0
Operating System(s): 
Keyword(s): kbhw kbHardware
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server, Enterprise Edition version 4.0 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows 2000 Professional 
- Microsoft Windows 2000 Server 
- Microsoft Windows 2000 Advanced Server 
-------------------------------------------------------------------------------

SUMMARY
=======

Windows NT 4.0 does not support Universal Serial Bus (USB) host controllers.
There are no plans to provide USB support in Windows NT 4.0 in a future service
pack.

MORE INFORMATION
================

Microsoft Windows 2000, Microsoft Windows 98, and Microsoft Windows 95 OEM
Service Release versions 2.1 and 2.5 provide support for the Open Host
Controller Interface (OHCI) and Universal HCI (UHCI) specifications. Microsoft
provides full supportfor USB under the new Windows Driver Model (WDM).

For more information about Microsoft Windows 98 support, see the following
article in the Microsoft Knowledge Base:

  Q184400 Supported Universal Serial Bus Host Controllers for Windows 98 and
  Windows Me

For more information about USB specifications, visit the following Microsoft Web
site:

  http://www.microsoft.com/hwdev/usb/
