Q196352: Inside the Win98 Registry: Chapter 4 Files Missing

Article: Q196352
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 20-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Inside the Microsoft Windows 98 Registry ISBN 1-57231-824-4 
-------------------------------------------------------------------------------

SYMPTOMS
========

The \Chapter4 folder, and all of the files referenced in the book in Chapter 4,
may be missing from the companion CD-ROM.

WORKAROUND
==========

Microsoft Press has created a patch file containing the missing Chapter 4
files.

The following file is available for download from the Microsoft Download Center:

  Inreg98.exe
  (http://download.microsoft.com/download/win98/mspress/2.0/W9XNT4/EN-US/Inreg98.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

MORE INFORMATION
================

To install the Inreg98.exe patch file, follow these steps:

1. Download the patch file to your hard disk.

2. Copy the patch file to the folder where you have installed your companion CD
  files.

3. Run the Inreg98.exe self-extracting executable file. The Chapter 4 folder and
  files will automatically extract themselves into the appropriate location.

Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
