Q196656: DOC: vbOLEDBProvider Value for DataSourceBehavior Doesn’t Exist

Article: Q196656
Product(s): Microsoft Visual Basic for Windows
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): kbdocfix kbdocerr kbVBp600 kbGrpDSVBDB
Last Modified: 11-JAN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Visual Basic 6.0 documentation incorrectly states that you can set a class
module's DataSourceBehavior property to 2 - vbOLEDBProvider. The correct values
for DataSourceBehavior are 0 - vbNone and 1 - vbDataSource.

REFERENCES
==========

This documentation error is described in the Visual Basic 6.0 ReadMe file,
ReadMeVB.htm in the "Data Access Issues and Databinding Tips" section under the
"Incorrect References for Creating OLE DB Providers" topic.
