Q134374: Misleading Message for Properties in Favorite Places

Article: Q134374
Product(s): The Microsoft Network
Version(s): 1.00 1.05 1.2 1.3
Operating System(s): 
Keyword(s): 
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Properties on the File menu in Favorite Places, you may receive
the following error message:

  This task cannot be completed at this time. Please try later.

CAUSE
=====

The Properties command does not function in Favorite Places if you do not have
an item selected.
