Q281479: User Wants To Use MSN Companion To Visit Chat Rooms

Article: Q281479
Product(s): The Microsoft Network
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

- Customer tries to use a Chat site service such as MSN or AOL but the device
  doesn't work on these Web sites.

CAUSE
=====

- Chat rooms usually require special ActiveX controls. Because MSN Companion
  does not support downloaded ActiveX controls, many chat sites will not work
  with MSN Companion.

RESOLUTION
==========

- Chat isn't supported on MSN Companion. However, you can use the Messenger
  feature to send instant messages to your friends and family. Just click the
  messenger button in the toolbar at the bottom of the screen to use this
  feature.
