Q116251: PC2X 8-Bit Bernoulli Controller May Cause System to Hang

Article: Q116251
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your computer stops responding (hangs) while you are using an Iomega PC2X 8-Bit
Bernoulli Controller card.

CAUSE
=====

Iomega Technical Support has confirmed that the Iomega PC2X 8-Bit Bernoulli
Controller card may not function properly on 486/33 megahertz (MHz) and faster
computers.

For setup purposes, you may need to run Setup with the ignore hardware detection
parameter, /I, (use the command SETUP /I).

WORKAROUND
==========

To work around this problem, use a slower computer or a different Bernoulli
controller card.


MORE INFORMATION
================

The PC2X 8-Bit Bernoulli Controller card is manufactured by Iomega Corporation,
a vendor independent of Microsoft; we make no warranty, implied or otherwise,
regarding this product's performance or reliability.

For more information about the Iomega PC2X Bernoulli controller card, contact
Iomega Technical Support.
