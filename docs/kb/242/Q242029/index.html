Q242029: MS Exchange Server 5.5 Administrator’s Companion Corrections

Article: Q242029
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 16-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Exchange Server 5.5 Administrator's Companion ISBN 0-7356-0646-3 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Exchange Server 5.5
Administrator's Companion, ISBN 0-7356-0646-3.

The following topics are covered:

- Page iii: Typographical Error In Dedication

- Pages v & vi: Typos In Contents At A Glance Section

- Pages vii, xiv, & xvi: Typos In Table Of Contents

- Page 93: Error In Figure 6.3

- Page 517: Typo In Part VI Opener

MORE INFORMATION
================

Page iii: Typographical Error In Dedication
-------------------------------------------

On page iii, in Rick's dedication, change:
"...even corny song's lyrics come true."

To:
"...even corny songs' lyrics come true."


Pages v & vi: Typos In Contents At A Glance Section
---------------------------------------------------

Contents at a Glance:

Page v:
Chapter 3 begins on page 36, not page 37.

Change:
"3 Integration with the Operating System 37"

To: "3 Integration with the Operating System 36"

Page vi:
Chapter 18 is titled "Using Microsoft Outlook 98."

Change:
"18 Using Outlook"

To:
"18 Using Microsoft Outlook"

Chapter 26 is titled "Using Collaboration Data Objects."
Change:
"26 Using Collaborative Data Objects"

To:
"26 Using Collaboration Data Objects"


Pages vii, xiv, & xvi: Typos In Table Of Contents
-------------------------------------------------

Page vii: Dedication is on page iii, not page xvii.

Page xiv: Chapter 20 begins on page 460, not 430.

Page xvi: Chapter 23 begins on page 518, not 508.


Page 93: Error In Figure 6.3
----------------------------

On page 93 there is an error in Figure 6.3. The Transitive Connection should be
drawn between sites A and C rather than sites A and B.


Page 517: Typo In Part VI Opener
--------------------------------

Change the Chapter 26 title from:
"Using Collaborative Data Objects"

To:
"Using Collaboration Data Objects"


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
