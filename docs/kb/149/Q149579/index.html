Q149579: XADM: MTA Error: Database Object Not in Use

Article: Q149579
Product(s): Microsoft Exchange
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 23-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When the Microsoft Exchange Message Transfer Agent (MTA) is restarted, the MTA
may stop with the following exception error:

  Database Object Not in Use

This error may occur even if the MTA was stopped successfully without errors.

CAUSE
=====

The Microsoft Exchange MTA experienced a startup error which resulted in this
exception.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
Service Pack 2. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

  S E R V P A C K
