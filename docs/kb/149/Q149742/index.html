Q149742: Magic School Bus: Out of Memory Printing to Laser Printer

Article: Q149742
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Scholastic's Magic School Bus series: Explores the Human Body for Windows, version 1.0 
- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
- Scholastic's Magic School Bus series: Explores the Solar System for Windows, version 1.0 
- Scholastic's Magic School Bus series: Explores Inside the Earth for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a full page landscape image to a laser printer with Magic School
Bus products, such as the diver's license in MSB Ocean, you may receive a
printer memory error. The message varies depending on the printer manufacturer,
but Out of Memory and Memory Overflow are examples. Only part of the image
prints.

MORE INFORMATION
================

Because these images contain large graphics and are being printed in landscape
mode, these images may require more memory than the printer has available. Below
is a chart of tested printers and their memory requirements. With the exception
of the Hewlett-Packard (HP) 4L, all 300 DPI (dots per inch) laser printers
required more than 1 MB RAM to print the image. For 600 DPI printers, more than
2 MB RAM are required.

Brand/Model                     Memory   DPI           Result
=============================================================

Hewlett Packard 4L                 1     300          Printed
Hewlett Packard 4ML                4     300          Printed
Hewlett Packard 4 Plus             6     600          Printed
IBM LaserPrinter 4039 12R          4     300          Printed
TI Microlaser Pro 600              6     600          Printed

Canon LBP-1260                     2     600           Failed
Canon LBP-430                      1     300           Failed
Panasonic KX-P4440                 1     300           Failed
Panasonic KX-P4430                 1     300           Failed
Okidata OL410e                     2     600           Failed
