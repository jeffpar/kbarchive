Q142485: Streets 4.0: Highways Do Not Print Using HP Color Deskjet

Article: Q142485
Product(s): Microsoft Automap
Version(s): 4.00
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 25-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Automap Streets for Windows, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a map to a Hewlett-Packard (HP) Deskjet 500C, 540, or 550C color
printer, the Highway and Interstate lines do not print.

STATUS
======

This problem occurs when you use Microsoft Automap Streets version 4.0 and a
Hewlett-Packard (HP) Deskjet 500C, 540, or 550C color printer. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.

WORKAROUND
==========

To resolve this problem, set the Road Drawing Style either to Regular or to
Enhanced. To change this setting, use the following steps:

1. Start Automap Streets.

2. Click Display Automap Streets.

3. On the Map menu, click Options.

4. Click the Drawing tab and set the Road Drawing Style either to Regular or to
  Enhanced. Click OK.

To successfully print the Highways and Interstate lines as a non-color image,
install the HP Deskjet 500 printer driver.

MORE INFORMATION
================

For additional information, please see the following article in the Microsoft
Knowledge Base:

  Q125653 Automap: Printing Problems with Canon BJC Series Printers
