Q142921: Streets 4.0: Some Street Numbers and Labels Do Not Print

Article: Q142921
Product(s): Microsoft Automap
Version(s): 4.00
Operating System(s): 
Keyword(s): 
Last Modified: 25-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Automap Streets for Windows, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a map, some of the Street Numbers and Labels displayed on the
screen do not print.

RESOLUTION
==========

To resolve this problem, use one of the following methods:

NOTE: The following steps set the number of Street Labels that appear on your
area map. For best results, use Method 1.

Method 1
--------

Use the following steps to set the Street Labeling option to Many.

1. On the Map menu, click Options.

2. Click the Drawing Tab.

3. Move the Street Labeling Option slider all the way to Many.

Method 2
--------

Use the following steps to set the Road Drawing Style to Regular or Enhanced:

1. On the Map menu, click Options.

2. Click the Drawing tab.

3. Select Regular or Enhanced.

4. Click OK.
