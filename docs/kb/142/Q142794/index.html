Q142794: Availability of Microsoft Windows 95 Service Pack 1

Article: Q142794
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:2.0,95
Operating System(s): 
Keyword(s): kbfile win95 msiew95 kbgraphxlinkcritical
Last Modified: 21-DEC-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Internet Explorer version 2.0 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The latest Microsoft Windows 95 service pack, Service Pack 1, includes an
update, system administration tools, additional components, and drivers for
Windows 95. For additional information about the contents of Windows 95, click
the article numbers below to view the articles in the Microsoft Knowledge Base:

  Q143003 Description of Microsoft Windows 95 Service Pack 1 Updates

  Q143004 Description of Microsoft Windows 95 Service Pack 1 Tools

  Q146449 Description of Microsoft Windows 95 Service Pack 1 Components

Additional information is also available in the Administrator's Guide
(http://www.microsoft.com/windows/downloads/bin/W95acmindoc.EXE).

NOTE: This download is not intended for use on computers running Microsoft
Windows 95 A, B, or C or Microsoft Windows 98. To determine if you are running
Windows 95 A or later, click Start, point to Settings, click Control Panel, then
double-click System. If the version number reported is 4.00.950 A or later, do
not install this update. Your system already contains the appropriate fix. For
additional information about how to determine what version of Windows you have
installed, click the article number below to view the article in the Microsoft
Knowledge Base:

  Q158238 How to Determine the Version of Windows 95/98/Me in Use

MORE INFORMATION
================

Click the site below to download just the fixed files included with Windows 95
Service Pack 1, all the files included with Windows 95 Service Pack 1, or to
order the Windows 95 Service Pack 1 CD-ROM:

  http://www.microsoft.com/windows95/downloads/contents/WURecommended/S_WUServicePacks/W95SvcPack1/Default.asp

Click the appropriate file(s) below to download only the individual support tool
or component. Please note that these support tools and components are included
in the complete download of Windows 95 Service Pack 1 (available via the link
above). For additional information about the support tools and components, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q143004 Description of Microsoft Windows 95 Service Pack 1 Tools

  Q146449 Description of Microsoft Windows 95 Service Pack 1 Components

- Download Windows 95 Support Assistant (Assist.exe) now

- Download Batch Installation Utility (Version 2.0) (Batch20.exe) now

- Download SLIP and Scripting Support for Dial-Up Networking (Dscrpt.exe) now

- Download INF Generator Utility (Ig.exe) now

- Download INF Installation Tool (Infinstd.exe) now

- Download 32-bit DLC Protocol (Msdlc32.exe) now

- Download Microsoft Windows NT Server Management Tools (Nexus.exe) now

- Download SLIP and Scripting Support for Dial-Up Networking (Script.exe) now

- Download Folder to update a Windows 95 network distribution Point
  (Sp1nsf.exe) now

- Download Silent update for network administrators (Sp1upd.exe) now

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
