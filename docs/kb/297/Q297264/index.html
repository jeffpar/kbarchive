Q297264: A+ Certification Training Kit, Third Edition Comments And Correc

Article: Q297264
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 13-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS A+ Certification Training Kit, Third Edition ISBN 0-7356-1265-X 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book A+ Certification Training Kit, Third
Edition, ISBN 0-7356-1265-x.

The following topics are covered:

- Page 30: Incorrect Information About Chipset

- Page 81: Incorrect Connector Orientation In Figure 5.1

- Pages 139 and 638: Extended RAM Should Be Upper Memory Area

- Page 205: Incorrect Format Switch

- Page 275: Change 65-KB To 65,536

- Page 336: 128bps Should Be 128Kbps

- Page 408: Incorrect Information About Windows Me Compatibility with NTFS

- Page 420: Incorrect Information About C Partition For Windows 2000 and
  Windows 98 Dual Boot

- Page 471: Incorrect File Name

- Page 503: Expand.com Should Be Expand.exe

- Page 553: Switch Steps 9 And 10

- Page 633: Clarification For Answers To Questions 5 And 7

- Page 645: Incorrect Term For CHS

MORE INFORMATION
================

Page 30:  Incorrect Information About Chipset
---------------------------------------------

On page 30, in the Description field for "Chip set", it incorrectly states that
the chipset inlcudes the CPU.

Change:
"This set includes the CPU and other chips that control the flow of data
throughout the system"

To:
"This set includes chips that control the flow of data throughout the system"


Page 81: Incorrect Connector Orientation In Figure 5.1
------------------------------------------------------

On page 81, in Figure 5.1, both black wires of connectors P8 and P9 are shown
facing away from each other. This is incorrect. The two black wires on each
connector should be side by side against each other.


Pages 139 and 638: Extended RAM Should Be Upper Memory Area
-----------------------------------------------------------

On page 139, in the last paragraph, and on page 638, in the answer to question
2,

Change:
"Shadow RAM rewrites (or shadows) the contents fo the ROM BIOS and/or video BIOS
into extended RAM (between the 640-KB boundary and 1 MB).

To:
"Shadow RAM rewrites (or shadows) the contents fo the ROM BIOS and /or video BIOS
into the upper memory area (between the 640-KB boundary and 1 MB).


Page 205: Incorrect Format Switch
---------------------------------

On page 205,

Change:
"format a: /s to"

To:
"format a: /s"


Page 275: Change 65-KB To 65,536
--------------------------------

On page 275, in the table,

Change:
"65-KB Color"

To:
"65,536 Color"


Page 336: 128bps Should Be 128Kbps
----------------------------------

On page 336, in the second paragraph under "Modem Basics",

Change:
"128bps"

To:
"128Kbps"


Page 408: Incorrect Information About Windows Me Compatibility with NTFS
------------------------------------------------------------------------

On page 406, in the third bulleted item from the bottom, it is stated that
Windows ME is compatible with the NTFS file system. This is incorrect.

Change:
"NTFS offers many enhanced features over other PC file systems, but it is only
compatible with Windows NT, Windows Me, and Windows 2000."

To:
"NTFS offers many enhanced features over other PC file systems, but it is only
compatible with Windows NT and Windows 2000."


Page 420: Incorrect Information About C Partition For Windows 2000 and Windows 98 Dual Boot
-------------------------------------------------------------------------------------------

On page 420, in the second bulleted item, change:

"The C drive must be a FAT16 partition and include..."

To:

"The C drive must be either a FAT 16 or a FAT32 partition and include..."


Page 471: Incorrect File Name
-----------------------------

On page 471, in the second bulleted item,

Change:
"CDLDR"

To:
"CMLDR"


Page 503: Expand.com Should Be Expand.exe
-----------------------------------------

On page 503, in the last paragraph,

Change:
"EXPAND.COM"

To:
"EXPAND.EXE"


Page 553: Switch Steps 9 And 10
-------------------------------

On page 553, steps 9 and 10 should be switched in order.

Change:
"9. Replace the cover and power up the computer.
10. Reconnect any peripherals (keyboard, mouse, monitor)."

To:
"9. Reconnect any peripherals (keyboard, mouse, monitor).
10. Replace the cover and power up the computer."


Page 633: Clarification For Answers To Questions 5 And 7
--------------------------------------------------------

On page 633, clarification is needed in the answers to questions 5 and 7.

Change the third sentence in the answer for question 5 from:
"A microphone converts analog pulses into a digital format that can be used or
stored by the computer."

To:
"A microphone accepts analog pulses that can be converted into a digital format,
which can be used or stored by the computer."

Change the last part of the answer for question 7 from:
"...and the speakers convert digital information into an analog form that is
understood as sound by the human ear."

To:
"...and the speakers receive analog information from a sound card that has
converted digital information into an analog form that is understood as sound by
the human ear."


Page 645: Incorrect Term For CHS
--------------------------------

On page 645, in question 22,

Change:
"tracks per sector"

To:
"sectors per track"


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
