Q297350: Upgraded Free-To-Paid Hotmail Accounts Unable To Connect Through

Article: Q297350
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 03-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

You upgraded a hotmail.com e-mail address to a paying MSN Internet Access
through the free-to-paid feature in MSN Explorer (MSN Version 6.1). Though you
can establish a dial-up connection using the MSN Explorer software, you cannot
establish a connection through a manually created dial-up networking connection.

CAUSE
=====

This is a known issue with MSN Explorer (MSN Version 6.1) and the development
team are currently working to correct it.
This issue happens only when connecting with a manually created connection file
in Dial-up Networking. In this specific case, the user name credentials don't
work.

RESOLUTION
==========

Install MSN Explorer 6.1

To install MSN Explorer 6.1

- Go to http://explorer.msn.com.
