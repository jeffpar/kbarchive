Q74450: Upgrading to Microsoft Mouse Driver Version 8.1

Article: Q74450
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 21-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Mouse driver version 8.1 is available to current Microsoft Mouse
owners. For information, call Microsoft Sales Information Center (MSIC) at (800)
426-9400.

The version 8.1 driver is available at no charge to Microsoft Mouse owners
currently using version 8.0 of the Mouse driver. There is a $25 charge (plus
shipping and handling) to upgrade from a driver version earlier than 8.0.

A free version of the driver is made available to MS-DOS 5 Upgrade owners who
have Microsoft Mouse driver version 6.2 or earlier and own a Microsoft Mouse.
For information about obtaining the driver, call Microsoft MS-DOS product
support or Microsoft Sales Information Center (MSIC) at the number above.
MOUSE.COM version 7.04 is included with the 5.0a version of MS-DOS 5 Upgrade.

MORE INFORMATION
================

The Microsoft Mouse driver version 8.1 is to be used ONLY with Microsoft mice.
Microsoft does not support this driver with other brands of mice.
