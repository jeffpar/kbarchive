Q226367: Content Expiration Shows Date as Question Marks in Some Language

Article: Q226367
Product(s): Internet Information Server
Version(s): winnt:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 04-FEB-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Internet Information Services version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you set up content expiration using the HTMLA Web Administration Forms in
Internet Information Services (IIS) 5.0, in some languages, the date may appear
as question marks.

This is true for the following languages:

  Arabic
  Russian
  Hindi

When you use these languages, the date appear as five question marks (?????).

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Internet Information
Services version 5.0.

Additional query words: HTMLA Globalization iis
