Q166065: WV: Cannot Paste to the Desktop Folder from Word Viewer

Article: Q166065
Product(s): Word 97 for Windows
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbui kbdta kbviewer word97
Last Modified: 19-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word Viewer 97-2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Microsoft Word Viewer 97-2000, the Paste command may be unavailable if you
attempt to paste a file to the Desktop folder directly from the Open dialog box
(to display this dialog box, click Open on the File menu).

CAUSE
=====

This problem occurs if you right-click the file in the Open dialog box, click
the Cut or Copy command, and then attempt to paste the file to the Desktop
folder.

WORKAROUND
==========

Use one of the following methods to work around this problem:

- After cutting or copying the file, switch to the Windows desktop (outside of
  Word Viewer) and paste the file.

- After cutting or copying the file, go to the Desktop folder by
  double-clicking the Windows folder and then double-clicking the Desktop
  folder. Then paste the file.

- Perform the cut or copy and paste the file from another program such as
  Windows Explorer or My Computer.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

REFERENCES
==========

For more information about Word Viewer 97-2000, please see the following
Knowledge Base article:
