Q166737: Windows NT Does Not Show Mapped Drives Using Chameleon 6.0.1

Article: Q166737
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kb3rdparty
Last Modified: 25-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Chameleon NFS 6.0.1 on Windows NT 4.0, any mapped drives to
the NFS volumes will disappear from the File Open Dialog Box after you click
Open from the File menu to view available drives. This occurs with 16- bit
applications like Visual Basic 4.0 and Visual C++ 1.5.

NOTE: This may also occur in other applications running on Windows NT as well as
other 16-bit applications. This issue does not occur with Chameleon NFS 5.0.1 in
Windows NT 4.0.


RESOLUTION
==========

Contact NetManage INC at (408) 973-7171.
