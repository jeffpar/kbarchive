Q273607: Microsoft Exchange 2000 Server Administrator’s Companion Comment

Article: Q273607
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 30-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Exchange 2000 Server Administrator's Companion ISBN 0-7356-0938-1 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Exchange 2000 Server
Administrator's Companion, ISBN 0-7356-0938-1.

The following topics are covered:

- Incorrect CD Label Text

- Page 205: Incorrect Instruction On Creating Mailbox For Existing User

MORE INFORMATION
================

Incorrect CD Label Text
-----------------------

On the companion CD label, remove the following text:

"To install the sample files from this disc to your hard disk, see the
installation instructions in the 'Using the Companion CD' section of the book."

This book does not come with sample files and does not have a "Using the
Companion CD" section.


Page 205: Incorrect Instruction On Creating Mailbox For Existing User
---------------------------------------------------------------------

On page 205, in the paragraph below Figure 9-2, the instruction for creating a
mailbox for an existing user is incorrect.

Change:
"Simply select any existing user in the Users folder in Active Directory Users
and Computers, and choose Add Mailbox from the Action menu. This command opens
the Create Exchange Mailbox screen of the wizard, allowing you to add and
configure the mailbox for the user."

To:
"Simply select any existing user in the Users folder in Active Directory Users
and Computers, and choose Exchange Tasks from the Action menu. This command
opens the Exchange Task Wizard screen, allowing you to add and configure the
mailbox for the user."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
