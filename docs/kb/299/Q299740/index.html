Q299740: Train Simulator: Keyboard Stops Responding

Article: Q299740
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 25-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Train Simulator, version 1.0, used with:
   - the operating system: Microsoft Windows 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are running Microsoft Train Simulator, the keyboard may stop
responding.

CAUSE
=====

This behavior can occur if DirectX has become corrupted.

RESOLUTION
==========

There are two possible resolutions to this issue. The first is to install
DirectX 8.0a. DirectX 8.0a is on the Microsoft Train Simulator installation CD.
Use the following steps:

1. Insert the installation CD into the CD-ROM drive. Press and hold the SHIFT
  key to prevent the program from starting.

2. Double-click My Computer.

3. Right-click the letter of the drive that has the installation CD, and then
  click Install DirectX 8.0a.

4. Follow the prompts.

If installing DirectX 8.0a does not help, work around this issue by pressing
ALT+TAB to minimize the screen, and then pressing ALT+TAB again to maximize the
screen.

The second resolution is to download and install the latest version of DirectX
from the following Microsoft Web site:

  http://www.microsoft.com/directx/homeuser/downloads/default.asp

For additional information about how to obtain and install the latest version of
DirectX, click the article number below to view the article in the Microsoft
Knowledge Base:

  Q179113 How to Download and Install DirectX
