Q322729: XADM: -1018 and -1022 Errors on Dell PowerEdge Servers

Article: Q322729
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 25-JUN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Microsoft Exchange Server 5.5 on a Dell PowerEdge 2400, 2450, or
4400 server, Exchange Server 5.5 may stop responding (hang) under heavy disk
use. You may see -1018 or -1022 error messages in the Application log. The
errors may occur during:

- A defragmentation of a database

- A repair of a database

- Heavy disk use under typical operation


RESOLUTION
==========

For additional information about a firmware revision that can resolve this
problem, click the article number below to view the article in the Dell Computer
Corporation Knowledge Base:

  TT1056235 Why does Microsoft Exchange 5.5 hang (lock up) under heavy disk
  activity (I/O) on Dell PowerEdge 2400, 2450, and 4400 servers?
