Q103385: PC Forms: Word for Windows Left Running After Installation

Article: Q103385
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:1.0,3.0,3.2
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Electronic Forms Designer, version 1.0, used with:
   - Microsoft Mail for PC Networks, versions 3.0, 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

If the Setup program for version 1.0 of Microsoft Electronic Forms Designer
detects Microsoft Word for Windows on your machine, it gives you the option of
installing a Route command in the application. If you choose to have the Route
command installed, Microsoft Word remains on the screen after the Setup program
is finished.

STATUS
======

Microsoft has confirmed this to be a problem in version 2.0c of Microsoft Word
for Windows. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.
