Q98413: Beethoven Err Msg: The Wrong Disc Is in the CD-ROM Drive…

Article: Q98413
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Multimedia Beethoven version 1.0 
- Microsoft Multimedia Schubert, version 1.0 
- Microsoft Multimedia Stravinsky, version 1.0 
- Microsoft Multimedia Mozart, version 1.0 
- Microsoft Visual Basic Professional Edition for Windows, version 3.0 
- Microsoft Visual Basic Standard Edition for Windows, version 3.0 
- MSPRESS Asymetrix Toolbook for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

The following error message may occur if one of the programs listed at the top
of this article (or another CD audio application) is started when the wrong disc
is in the drive:

  Windows 3.x:

  The Wrong Disc is in the CD-ROM drive. Please insert the Multimedia Beethoven
  CD.

  Click "Retry" to try again, click "Continue" to proceed without the CD-ROM
  disc, or click "Exit" to leave Multimedia Beethoven.

  Windows 95:

  No CD-ROM drive is detected. Make sure your CD-ROM drive is turned on and the
  Multimedia Beethoven disc is in the drive.

  Click "retry" to try again, click "Continue" to proceed without the CD-ROM
  disc, or click "Exit" to leave Multimedia Beethoven.

This error message may also appear if the CD-ROM drive is not capable of playing
CD audio.

MORE INFORMATION
================

Some CD-ROM drives are not capable of playing CD audio due to the SCSI
controller. This can be verified by starting Media Player and choosing CD Audio
from the Device menu. The following error message will result if the CD-ROM
drive is not capable of playing CD Audio:

  There is an undetectable problem in loading the specified device driver.

Any music CD in the drive will produce the above error message in Media Player.

Although you cannot play the tracks from the Beethoven CD-ROM, Setup for
Beethoven usually runs to completion without problems.
