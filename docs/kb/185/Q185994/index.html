Q185994: GPS Err Msg: This Program Was Designed for a Previous Version…

Article: Q185994
Product(s): Microsoft Automap
Version(s): WINDOWS:5.0
Operating System(s): 
Keyword(s): kbenv kberrmsg kbtool kbimukbfaq
Last Modified: 19-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Automap Streets Plus for Windows, version 5.0 
- Microsoft Automap Trip Planner for Windows, version 5.0 
- Microsoft Expedia Streets 98 
- Microsoft Expedia Trip Planner 98 
- the operating system: Microsoft Windows 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start the Global Positioning System (GPS) tool in Control Panel on a
computer running Microsoft Windows 98, you may receive the following error
message:

  This program was designed for a previous version of Windows. You should
  upgrade to the latest version of Windows.

RESOLUTION
==========

To resolve this issue, click OK to close the error message dialog box. The GPS
tool in Control Panel should function properly.

STATUS
======

This behavior is by design.

MORE INFORMATION
================

When you start the GPS tool in Control Panel, the GPS tool checks your version
of Windows. If you are not running Windows 95 or Windows NT 4.0, you receive
this error message.
