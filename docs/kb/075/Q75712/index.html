Q75712: Alphabetical Listing of the MS-DOS 5.0 Working Disks Contents

Article: Q75712
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 22-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a cross-reference of MS-DOS 5.0 files. The files are listed in
alphabetical order followed by their location on the 5.25-inch and 3.5-inch
working disks. These disks are created when installing on floppy disks using the
SETUP /f command.
