Q313945: Encarta 2001: IPF Err Msg When Program Reads CD-ROM

Article: Q313945
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 29-JAN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When your installation of Microsoft Encarta Reference Suite 2001 attempts to
read data from the Encarta Reference Suite CD-ROM, you may receive an error
message similar to the following:

  Enc2001 performed an Illegal Operation and will be shut down.

When you click the Details button, you receive the following error message:

  Invalid page fault in module MVTL53N.DLL at 015f:

You typically receive the first error message after the CD-ROM speeds up and then
slows down repeatedly.

CAUSE
=====

This behavior occurs when your Encarta installation is unable (for a variety of
possible reasons) to read from the CD-ROM.

RESOLUTION
==========

For additional information about how to troubleshoot CD-ROM read issues, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q218617 How to Troubleshoot CD-ROM Read Errors
