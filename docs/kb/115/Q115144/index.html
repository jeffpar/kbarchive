Q115144: Cannot Print Faxes from AWFAX Fax Viewer with the HP 1200C

Article: Q115144
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.11
Operating System(s): 
Keyword(s): 
Last Modified: 10-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to print from the Fax Viewer in Microsoft At Work PC Fax to a
Hewlett-Packard DeskJet 1200C color printer, a blank sheet of paper is ejected
from the printer.

CAUSE
=====

The Hewlett-Packard DeskJet 1200C color printer driver version 2.20 does not
support printing from the Fax Viewer.

RESOLUTION
==========

Hewlett-Packard has a new printer driver (version 3.30) that allows the 1200C
color printer to print from the Fax Viewer. You can obtain this new printer
driver from Hewlett-Packard.
