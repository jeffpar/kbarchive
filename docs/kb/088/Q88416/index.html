Q88416: ROM BIOS Needed for Tandon Laptops

Article: Q88416
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When running Microsoft Windows on Tandon Laptop computers, you experience
keyboard failures, Unrecoverable Application Errors (UAEs), or general
protection (GP) faults with some BIOS versions. To avoid these problems:

- All 386sx laptop computers should use Tandon BIOS and not Phoenix BIOS

- BIOS version should be at least the following:

  BIOS Version   Tandon Computers
  ------------   ----------------

  3.61           PCA non-discrete, PAC 286, Targa
                 386/16,386/20,386/25,386/33, 386 SX, 486
  3.62           PAC/12, PCA/12
  3.52           286 Laptop
  3.64           386 laptop
  2.22           PCA discrete

MORE INFORMATION
================

To obtain a new Tandon BIOS, call the Tandon dealer that sold the machine or
call Tandon directly.

NOTE: A Tandon BIOS, unlike a Phoenix BIOS, is not identified on the screen when
you start your computer. The BIOS version should be displayed at the top-right
corner of the CMOS setup screen.
