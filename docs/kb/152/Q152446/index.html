Q152446: XCON: Dead Letter Recipient Button is Missing

Article: Q152446
Product(s): Microsoft Exchange
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 28-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Microsoft Exchange Books Online and the help for the MTA Site Configuration
General tab, in the Microsoft Exchange Administrator program, states that there
is a button called Dead letter recipient, but this button does not exist.

RESOLUTION
==========

This option was available in beta releases of Microsoft Exchange Server but was
removed before the final release. The documentation is incorrect.
