Q148782: Err Msg: "Modem Is Busy or Cannot Be Found" Using Aptiva Guide

Article: Q148782
Product(s): The Microsoft Network
Version(s): WINDOWS:1.2,1.3,2.0,2.5,2.51,2.52,2.6
Operating System(s): 
Keyword(s): kb3rdparty kberrmsg kbmsn
Last Modified: 03-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use a computer with the Aptiva Guide program in the Startup folder, you
may receive the following error message when you attempt to connect to MSN, The
Microsoft Network:

  Modem is busy or cannot be found.

CAUSE
=====

The Aptiva Guide program has partial control of the modem, monitoring it for
incoming voice messages.

RESOLUTION
==========

To resolve this issue, remove the Aptiva Guide program from the Startup folder
and then restart your computer. To do so, follow these steps:

1. Right-click Start, and then click Open.

2. Double-click the Programs folder, and then double-click the StartUp folder.

3. Drag the Aptiva Guide icon from the StartUp window to the desktop.

4. Close all open windows, and then restart your computer.
