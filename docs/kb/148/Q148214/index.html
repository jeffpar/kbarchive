Q148214: Deleting Roles Causes Duplicate Button Faces

Article: Q148214
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:7.0; Win95:7.0
Operating System(s): 
Keyword(s): 
Last Modified: 21-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you delete roles from the Roles tab, the Seven Habits Tools causes
duplicate button faces. This will cause permanent data loss for some or all of
the roles that have been entered into the Roles tab.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.


Additional query words: schedule plus 7.00
