Q315600: Asheron’s Call: Err Msg: One of Your Characters Is Still in…

Article: Q315600
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call: Dark Majesty, version 1.0 
- Microsoft Asheron's Call, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you log on to a world, you may receive an error message similar to one of
the following:

  One of your characters is still in the world. Please try again in a few
  minutes.

  -or-

  A save of this character is still in progress.

  -or-

  This characters starting server is experiencing difficulty. Please try again
  in a few minutes.

CAUSE
=====

This behavior can occur if the server has not released the data for this
character.


RESOLUTION
==========

To resolve this issue, use one of the following methods.

Method 1 
--------

Wait a few minutes before you attempt to log on with this character.

Method 2
--------

1. Log on with another character, and then click "Options... Urgent Assistance".

2. Wait for an administrator to respond. Explain the situation to the
  administrator, and then ask him or her to boot the stuck character.

Method 3
--------

Wait for a server reset (this occurs at least monthly with the content update).

MORE INFORMATION
================

For more information about the status of the Asheron's Call servers, please
visit the following Web site:

  http://zone.msn.com/asheronscall/news/ASHEnewsnetwork.asp
