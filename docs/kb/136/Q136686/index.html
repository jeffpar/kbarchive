Q136686: Fury3 Mission Complete Text Not Displayed After Some Missions

Article: Q136686
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

1.00
WINDOWS
kbprb

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Fury 3 for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

In missions that have no Jumpzone to fly to, such as Ares or Kroy, the mission
complete text may not be displayed, or, it may only be displayed for a very
short period of time.

STATUS
======

Microsoft has confirmed this to be a problem in Fury 3 for Windows, version 1.0.

Additional query words: fury 3 fury-3 1.00
