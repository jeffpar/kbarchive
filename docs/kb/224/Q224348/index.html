Q224348: Wacom PenPartner Tablet Does Not Work Correctly

Article: Q224348
Product(s): Microsoft Windows NT
Version(s): winnt:4.0,4.0 SP1,4.0 SP2,4.0 SP3,4.0 SP4
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbHardware
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 4.0, 4.0 SP1, 4.0 SP2, 4.0 SP3, 4.0 SP4 
- Microsoft Windows NT Server versions 4.0, 4.0 SP1, 4.0 SP2, 4.0 SP3, 4.0 SP4 
- Microsoft Windows NT Server, Enterprise Edition versions 4.0, 4.0 SP4 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install the driver for your Wacom PenPartner tablet (WTEST), your
PenPartner tablet may not work correctly, or at all.

CAUSE
=====

This issue can occur for either of the following reasons:

- The communications settings for your COMM port may be incorrect.

- The legacy Wacom keys in the registry may contain damaged data.

RESOLUTION
==========

To resolve this issue, verify that the communications settings for your COMM
port are correct, and if they are, contact Wacom to inquire about the
avilability of a fix for this issue:

1. For information about how to verify that the communications settings for your
  COMM port are correct, view the "VERIFY YOUR COM PORT(S)" section of the
  following article in the Microsoft Knowledge Base:

  Q161516 Troubleshooting Modem Problems Under Windows NT 4.0

2. Test to determine if the issue is resolved, and if it is not, contact Wacom
  to inquire about the avilability of a fix for this issue.
