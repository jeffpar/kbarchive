Q150562: Bookshelf 96-97 Scales Incorrectly on PostScript Printer

Article: Q150562
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf 1996-97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the graphics scaling option for Postscript printers, Microsoft
Bookshelf prints smaller than expected.

For example, if you set the printer scaling at 50 percent, Bookshelf prints the
graphics, and sometimes the text, at 33 percent of the original size.

RESOLUTION
==========

Copy and paste the article or image into a word processor, then print it from
the word processor.

For more information about copying and pasting, please see Bookshelf Online Help.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Bookshelf '96-'97 for
Windows. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.
