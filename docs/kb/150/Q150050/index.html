Q150050: How to See More than 9 Users in SMS Security Manager

Article: Q150050
Product(s): Microsoft Systems Management Server
Version(s): winnt:1.0,1.1
Operating System(s): 
Keyword(s): kbnetwork smsgeneral smssecman kbArtTypeINF
Last Modified: 31-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


The drop-down list box in SMS Security Manager displays up to nine logon entries.
If there are more than nine, no scroll bars are displayed. To see all the logon
entries, use the up and down arrow keys or use the mouse to drag the highlight
bar up or down the list.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server version
1.2.
