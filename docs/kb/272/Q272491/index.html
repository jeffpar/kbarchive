Q272491: How to change passwords with MSN Companion

Article: Q272491
Product(s): The Microsoft Network
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

How to change the password on the MSN Companion system.

MORE INFORMATION
================

If you know your password
-------------------------

To change an MSN Companion password

1. On the MSN Companion Start page, click settings, and then click password.

2. On the password settings page type your current password and click continue.

3. Type the new password in the first text box, retype the new password in the
  ...confirm it box, and then click done to save settings.

A password must be at least eight (8) characters long, composed of letters (a-z)
and numbers (0-9), but may contain no spaces.

Note: if you select lock this MSN Companion with my password so other people
can't use it, then you will be asked for your password each time you connect to
MSN, The Microsoft Network.

If you do not know your password
--------------------------------

Call MSN Customer Service at 1 (888) 765-2411 and ask for for a password reset.

Customer Service will direct you to type a reset password into the device using
the FIXIT pages. For more information on the FIXIT pages see Q278481 MSN
Companion FIXIT Pages

If you have the password lock feature turned on you will turn the unit off and
back on, and then type your reset password into the password lock box.

After connecting to MSN with your MSN Companion using your reset password, follow
the steps above to change your password.
