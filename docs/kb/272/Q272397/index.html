Q272397: WD97: Photo Editor Doesn’t Recognize Default Printer Change

Article: Q272397
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta word8 word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you change your default printer in Microsoft Windows, Microsoft Photo
Editor does not recognize the change and continues to use the previous default
printer.

WORKAROUND
==========

Change the printer in Microsoft Photo Editor to the printer that you want. To do
this, follow these steps:

1. On the File menu, click Print.

2. In the Print dialog box, change the Printer setting to the printer that you
  want.

3. Click OK.

NOTE: You need to repeat these steps each time you print from Microsoft Photo
Editor.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

To change the default printer in Microsoft Windows, follow these steps:

1. Click Start, point to Settings, and then click Printers.

2. In the Printers dialog box, right-click the printer that you want to set as
  the default printer, and then click "Set as Default Printer".
