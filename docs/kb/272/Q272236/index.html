Q272236: Games: Video Problems with Intense3D Wildcat Video Adapter

Article: Q272236
Product(s): Microsoft Home Games
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): kbdisplay kberrmsg kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires, version 1.0 
- Microsoft Age of Empires Expansion: The Rise of Rome, version 1.0 
- Microsoft Age of Empires, Gold Edition 
- Microsoft Age of Empires II Expansion: The Conquerors 
- Microsoft Age of Empires II: The Age of Kings, version 2.0 
- Microsoft CART Precision Racing, version 1.0 
- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
- Microsoft Crimson Skies 
- Microsoft Flight Simulator 98 
- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
- Microsoft Links LS 2000 
- Microsoft StarLancer, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start one of the programs listed at the beginning of this article on a
Windows NT 4.0-based computer, you may experience one or more of the following
symptoms:

- Your computer may stop responding (hang).

- Graphics in the game may be distorted or fuzzy.

- The screen may turn black.

- Horizontal lines may appear across the screen.

- The game may quit unexpectedly immediately after you start it.

- You may receive a DirectX-related or DirectDraw-related error message.

- You may receive an "Invalid Page Fault in module <unknown>" error
  message when you start the game.

CAUSE
=====

This behavior can occur if an Intense3D Wildcat video adapter is installed in
your computer.

The Intense3D Wildcat line of video adapters do not support DirectX at this
time.


RESOLUTION
==========

There is no know resolution to this issue at this time.

MORE INFORMATION
================

You may be able to resolve this issue by downloading and installing the latest
version of the video driver for your video adapter from the following Intense3D
Web site:

  http://www.Intense3D.com
