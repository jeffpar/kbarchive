Q171404: INFO: ODBC Files Shipped with Visual C++ 5.0

Article: Q171404
Product(s): Microsoft C Compiler
Version(s): winnt:5.0
Operating System(s): 
Keyword(s): kbDatabase kbMFC kbODBC kbVC
Last Modified: 17-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual C++, 32-bit Enterprise Edition, version 5.0 
- Microsoft Visual C++, 32-bit Professional Edition, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the ODBC files that shipped with Visual C++ 5.0 and the
components that the files are associated with.

NOTE: When you are checking to see if you have all of the proper files installed,
you should check the ODBC files as well as the driver-specific files. Some
files, such as MSVCRT40.DLL, are used by several drivers.
