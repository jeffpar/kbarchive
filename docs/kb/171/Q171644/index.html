Q171644: BUG: Reporting Progress Template Not On Practice Disk For PowerP

Article: Q171644
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 21-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft PowerPoint 97 Step by Step ISBN 1-57231-315-3 
-------------------------------------------------------------------------------

SUMMARY
=======

Page 6 of Microsoft PowerPoint 97 Step by Step asks you to select the Reporting
Progress template, which is used in the steps on pages 6 through 17. This
template is not included on the Microsoft PowerPoint 97 Step by Step disk.

The following file is available for download from the Microsoft Download Center:

  Ppt97sbs.exe
  (http://download.microsoft.com/download/powerpoint97win95/mspress/2.0/WIN98Me/EN-US/Ppt97sbs.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

MORE INFORMATION
================

The following installation instructions are copied from the README.TXT file
included in Ppt97sbs.exe.

INSTALLATION INSTRUCTIONS
-------------------------

1. Download PPT97SBS.EXE to your Microsoft PowerPoint Presentations directory.
  For most users, this directory will be

  C:\Program Files\Microsoft Office\Templates\Presentations

  Note: If you cannot download to this directory, you will need to move
  PPT97SBS.EXE after downloading using the Microsoft Windows Explorer (or other
  file management software) before doing step 2.

2. Double-click (or run) PPT97SBS.EXE in your Presentations directory to extract
  the necessary files.

3. Delete PPT97SBS.EXE.

4. Open Microsoft PowerPoint 97.

5. In the PowerPoint Startup dialog box, select AutoContent Wizard and then
  click OK.

6. Click the General button.

7. Click Add.

8. From the list, select Reporting Progress, and then click Open.

9. Click Cancel and close PowerPoint.
