Q171340: Requirements for Using RAS to Connect to NetWare Connect

Article: Q171340
Product(s): Microsoft Windows NT
Version(s): WINDOWS:3.51,4.0
Operating System(s): 
Keyword(s): kbnetwork
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server versions 3.51, 4.0 
- Microsoft Windows NT Workstation versions 3.51, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the server and client requirements for using Windows NT
Remote Access Service (RAS) and the IPX/SPX-compatible protocol to connect to
Novell NetWare Connect.

MORE INFORMATION
================

Server Requirements
-------------------

- Novell NetWare Connect version 2.0.28 or later

- IPX protocol

Client Requirements
-------------------

- One of the following NetWare network clients:

   - Microsoft Client Service for NetWare

   - Novell Client for Windows NT

- Remote Access Service (RAS)

- IPX/SPX-compatible protocol

For information about installing RAS, please see the following articles in the
Microsoft Knowledge Base:

  Q171267 How to Install Remote Access Service in Windows NT 3.51

  Q171015 How to Install and Configure Windows NT 4.0 Dial-Up Networking

For information about NetWare Connect, please contact Novell.
