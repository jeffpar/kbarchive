Q232383: XGEN: The Exchange Server 5.5 Evaluation and Migration Kit

Article: Q232383
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

The information in this article is taken from the Home.html file on the
Microsoft Exchange Server 5.5 Evaluation and Migration Kit CD.

MORE INFORMATION
================

  This kit provides all the tools and information you need to evaluate
  Microsoft Exchange Server and to decide whether Exchange is right for your
  organization. These resources are designed to make migration to Exchange
  Server from your current messaging system as seamless as possible. In the kit
  you will find a 120-day trial version of Exchange Server as well as technical
  white papers and tools to ensure smooth migration and deployment.

  For the latest information on Exchange Server and additional migration
  resources visit

  http://www.microsoft.com/exchange/techinfo/deployment/default.asp

  Migration 
  ---------

  In this section you will find the latest migration tools and technical white
  papers to help you move from Lotus cc:Mail, Microsoft Mail, Lotus Notes and
  Novell GroupWise to Exchange. Migration content includes:

- New Migration Tools for Lotus cc:Mail such as Microsoft Importer for Lotus
  cc:Mail Archives, cc:Mail Directory Migration Toolkit, and Excel Macros for
  migration of cc:Mail Distribution Lists.

- New Microsoft Importer for Lotus Notes Mail, Microsoft Exchange Application
  Services for Lotus Notes, Microsoft Exchange Connector for Lotus Notes, and
  the updated Microsoft Exchange Migration Wizard.

- White papers, best practices, and case studies help you migrate to Microsoft
  Exchange Server and Outlook.

  Improving Deployment
  --------------------

  This section provides technical white papers and presentations that discuss
  best practices to help you improve reliability and availability of your
  existing Exchange Servers as well as information to help you improve
  performance and scalability issues. In the "Improving Deployment" section of
  the CD you will find:

- Key white papers such as the "Best Practices for Exchange Database
  Management," "Disaster and Recovery Planning," and others.

- Key presentations from Microsoft Exchange Conference 98 such as "Exchange
  Troubleshooting - How Exchange Reports on Itself," "50 Mighty Registry
  Tweaks," and "My Server Is Down and Can't Get Back Up."

- Performance and Scalability information, including tools from Compaq to help
  you size your servers.

- References to books and training materials.

  Collaboration 
  -------------

  To help you get the most out of your Exchange infrastructure, this section
  provides sample applications and technical white papers to help you develop
  and deploy collaboration solutions. The "Collaboration" section contains:

- Sample Outlook forms.

- White papers such as "Building Solutions with Microsoft Exchange and
  Microsoft Outlook," and "Microsoft Outlook 98 and Exchange Server Tips
  Tricks."

- Key presentations from the Microsoft Exchange Conference 98.

- Information on Compaq Work Expeditor, a single tool that focuses on the
  automation of the workflow process.

- References to books and training materials.

  Become Year 2000 Compliant - Exchange Server and Microsoft Mail
  ---------------------------------------------------------------

  This section offers detailed information on how to bring Microsoft Exchange
  Server and Microsoft Mail products Year 2000 compliant. In the "Year 2000"
  section you will find:

- Service Pack 5 for Exchange Server 4.0

- Service Pack 2 for Exchange Server 5.0

- Updates for Microsoft Mail 3.5

  For more information on Year 2000 and Microsoft products, visit
  http://www.microsoft.com/year2000/

The Microsoft Exchange Server 5.5 Evaluation and Migration Kit CD can be ordered
at http://www.microsoft.com/exchange/55/downloads/Trial.htm.

For questions about ordering the CD or the status of your order, please call
(888) 658-7230.
