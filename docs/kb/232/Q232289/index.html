Q232289: XFOR: Exchange 5.5 SP2 Notes Connector Doesn’t Support Notes R5

Article: Q232289
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 09-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Microsoft Exchange Server 5.5 Service Pack 2 (SP2) Notes Connector does not
currently support Lotus Notes version R5.

RESOLUTION
==========

To resolve this issue, obtain the latest service pack for Exchange Server 5.5.
For additional information, click the article number below to view the article
in the Microsoft Knowledge Base:

  Q191014 XGEN: How to Obtain the Latest Exchange Server 5.5 Service Pack


MORE INFORMATION
================

There are some interoperability issues, which Microsoft is working on resolving.
No date has been set for when the Exchange Notes Connector will support Notes
R5.
