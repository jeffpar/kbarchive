Q165333: New Features Available in Windows NT 4.0 Service Pack 3

Article: Q165333
Product(s): Microsoft Windows NT
Version(s): 4.0 SP3
Operating System(s): 
Keyword(s): kbnetwork kbother kbDirectPlay kbSDKPlatform kbGrpDSNetkbfaq
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 SP3 
- Microsoft Windows NT Server version 4.0 SP3 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists new features available in Windows NT 4.0 Service Pack 3. For
additional information on Service Pack 3, please see the Service Packs page on
the following Microsoft web site:

- http://www.microsoft.com/NTServerSupport/

  -or-

- http://www.microsoft.com/NTWksSupport/

MORE INFORMATION
================

Windows NT 4.0 Service Pack 3 includes the following new features:

- Changes for clustering

- Creative Labs SoundBlaster AWE32 driver
  For more information, please see the following article in the Microsoft
  Knowledge Base:

  ARTICLE-ID: Q143155
  TITLE : No Audio with Crystal Audio or SoundBlaster AWE32

- DirectX 3.0, including the following feature enhancements:

   - DirectDraw

   - DirectSound

   - DirectInput

   - DirectPlay

   - Direct3D (software emulation)

For more information, please see the Windows NT 4.0 Service Pack 3 Readme.txt
file.

- Hooks for Microsoft NetMeeting support

- Microsoft Cryptography Application Programming Interface (CryptoAPI) 2.0.

  For more information, please see the Windows NT 4.0 Service Pack 3 Readme.txt
  file.

- Microsoft Open Database Connectivity (ODBC) API 3.0

  For more information, please see the Windows NT 4.0 Service Pack 3 Readme.txt
  file.

- New security features, which include:

   - Defining a Windows NT System Key lets you implement strong encryption

     For more information, please see the following article in the Microsoft
     Knowledge Base:

  ARTICLE-ID: Q143475
  TITLE : Windows NT SP3 System Key Permits Strong Encryption of the SAM

   - Password Filtering

     For more information, please see the Windows NT 4.0 Service Pack 3
     Readme.txt file.

   - Restricting Information Available to Anonymous Logon Users

     For more information, please see the following article in the Microsoft
     Knowledge Base:

  ARTICLE-ID: Q143474
  TITLE : Restricting Information Available to Anonymous Logon Users

   - New version of Update.exe lets you install Service Pack 3 and Hotfixes

   - Server Message Block (SMB) Signing

     For more information, please see the Windows NT 4.0 Service Pack 3
     Readme.txt file.

   - Two new Win32 APIs:

      - InitializeCriticalSectionAndSpinCount()

      - SetCriticalSectionSpinCount()

   - Updated versions of the remote procedure call (RPC) run-time dynamic- link
     library (DLL) files

     For more information, please see the Windows NT 4.0 Service Pack 3
     Readme.txt file.

Additional query words: 4.00 sp3
