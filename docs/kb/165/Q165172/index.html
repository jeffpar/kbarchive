Q165172: Gphsamp.exe Shows Automation to DataSheet in MS Graph 8

Article: Q165172
Product(s): Microsoft FoxPro
Version(s): WINDOWS:5.0,5.0a,6.0
Operating System(s): 
Keyword(s): kbfile kbole kbsample kbAutomation kbvfp500 kbvfp600
Last Modified: 24-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Gphsamp.exe is a file that demonstrates how to use OLE Automation to Microsoft
Graph to change the data that Graph 8.0 is using for a graph.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

  Gphsamp.exe
  (http://download.microsoft.com/download/vfox60/Howto1/1/W9X/EN-US/Gphsamp.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Microsoft Graph 8.0 that ships with Microsoft Office 97 exposes the DataSheet to
OLE automation. The Graphsmp form, which is available for download, is an
example of how to use OLE automation to change values in the Graph 8.0
DataSheet. The sample uses the TestData database (Vfp\Samples\Data) that ships
with Visual FoxPro. It uses the Employee and Order tables to chart the total
yearly and quarterly sales for each salesperson. For more information on doing
OLE automation to Microsoft Graph 8.0 please see the Vbagrp8.hlp file that ships
with Microsoft Office 97.
