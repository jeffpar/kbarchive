Q298699: MSN Explorer (MSN Version 6.1) Sign-in And Sign-up Error Message

Article: Q298699
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 23-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SUMMARY
=======

For any of the specific sign-in and sign-up error messages below, use the
corresponding solution. In this KB article, the sign-in and sign-up error
messages are categorized by type, and are listed alphabetically under the type
heading.

MORE INFORMATION
================

Error Message: MSN Internet Access Price plan,"Sorry, We were not able to retrieve the billing plan text at this time.\nPlease try again .later.\n\ninfocode:" + GetXmlNodeValue(root, "//Error/@Code"), true);
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

This error also refers to a problem in retrieving the billing plan text from the
sign-up server. Continue trying to set up the account. If the issue persists
over time, call Customer Service.

Error Message: The e-mail name you have chosen contains a forbidden word.  Please choose a different e-mail name and try again.
-------------------------------------------------------------------------------------------------------------------------------

The e-mail name the user is attempting to set up contains an obscene word or a
service related word (MSN, Hotmail, service). Have the user choose another
e-mail name, and then try again.

Error Message: The e-mail name you have chosen is too similar to your password.  Choose another e-mail name or a different password and try again.
--------------------------------------------------------------------------------------------------------------------------------------------------

This happens if the e-mail name specified contains the password you are
attempting to use. Have the user choose a different password. This system helps
ensure the security of user passwords.

Error Message: The first name you entered contains a forbidden word.  Please try again.
---------------------------------------------------------------------------------------

The first name the user entered contains an obscene word or a service related
word (MSN, Hotmail, service). Have the user change the name in the First name
box, then and try again.

Error Message: The last name you entered contains a forbidden word.  Please try again.
--------------------------------------------------------------------------------------

The last name the user entered contains an obscene word or a service related word
(MSN, Hotmail, service). Change the name in the Last name box, and then try
again.

Error Message: Sorry, an error occurred while trying to upgrade this account. Would you like to try again?
----------------------------------------------------------------------------------------------------------

Try again later.

Error Message: Sorry, we are unable to retrieve your profile information at this time. Would you like to try again?
-------------------------------------------------------------------------------------------------------------------

Ask the user to log out and try again. If problem persists, escalate to
MSNIANOC.

Error Message: Sorry, we are unable to retrieve your profile information at this time. Would you like to try again?
-------------------------------------------------------------------------------------------------------------------

Log out and try again.

Error Message: Sorry, we are unable to verify the user at this time. Please try again.
--------------------------------------------------------------------------------------

Try again later.

Error Message: Sorry, we are not able to create your e-mail address right now due to an unknown error. Would you like to try again?
-----------------------------------------------------------------------------------------------------------------------------------

This occurs when there is an error in creating a free Passport account. Try again
later.

Error Message: We are unable to signup a new user right now, please try again later.
------------------------------------------------------------------------------------

An issue with the sign-up server is the most likely cause of this error. Try
again. If unsuccessful, contact Customer Service to have an account created over
the phone, and then go through Setup as an existing user.

Error Message: Sorry, we were unable to retrieve your Internet access subscription plan information. Please try again later.
----------------------------------------------------------------------------------------------------------------------------

This error refers to a problem in retrieving the billing plan text from the
sign-up server. Continue trying to set up the account. If the issue persists
over time, call Customer Service to create an account over the phone. Go through
the Setup for MSN Explorer as an "existing user".

Error Message: We were not able to retrieve the billing plan text at this time.\n\ninfocode
-------------------------------------------------------------------------------------------

This generally occurs because of an issue with the sign-up server. Try again. If
unsuccessful, contact Customer Service to create an account over the phone, then
go through Setup as an existing user.
