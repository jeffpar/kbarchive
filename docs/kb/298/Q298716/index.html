Q298716: MSN Explorer (MSN Version 6.1) Sign-in And Sign-up Error Message

Article: Q298716
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 23-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SUMMARY
=======

For any of the specific sign-in and sign-up error messages below, use the
corresponding solution. In this KB article, the sign-in and sign-up error
messages are categorized by type, and are listed alphabetically under the type
heading.

MORE INFORMATION
================

Error Message: Numbers and '-' are the only valid characters for this field, please try again.
----------------------------------------------------------------------------------------------

This error occurs when the user has entered invalid characters in the Credit Card
box. You can only type numbers and the "-" character in this box. Enter the
correct credit card number, and then click Continue.

Error Message: Please choose an option before continuing.
---------------------------------------------------------

This error occurs when a user neither accepts nor declines the pricing plan.
Click I accept, and then click Continue.

Error Message: Please choose the type of credit card you are using.
-------------------------------------------------------------------

This error occurs when the user has not chosen a credit card type. Choose the
correct credit card type from the list, and then click Continue.

Error Message: Please enter the first name of the credit card holder.
---------------------------------------------------------------------

This error occurs when the user has not entered the first name of the credit card
holder. Type the first name, and then click Continue.

Error Message: Please enter the last name of the credit card holder.
--------------------------------------------------------------------

This error occurs when the user has not entered the last name of the credit card
holder. Type the last name, and then click Continue.

Error Message: Please enter your credit card number.
----------------------------------------------------

This error occurs when the user has left the credit card number box empty. Type
the credit card number, and then click Continue.

Error Message: Please make a selection.
---------------------------------------

This error occurs when the user has left either the Expiration Month or
Expiration Year box blank. Have the user select the expiration month and year
from the drop-down menus, and then click Continue.

Error Message: There is a maximum of 16 digits for a valid credit card number, please try again.
------------------------------------------------------------------------------------------------

This error occurs when a user has entered a credit card number longer than 16
digits, which is invalid. Have the user enter the correct credit card number,
and then click Continue.

Error Message: We were unable to authenticate your credit card, please check your information and try again
-----------------------------------------------------------------------------------------------------------

To resolve this issue:

1. Check all of the credit card information to be sure that it is correct.

2. If this does not work, try another credit card.
