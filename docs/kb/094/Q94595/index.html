Q94595: Voice Pilot: Limit to Number of Words in Vocabulary Files

Article: Q94595
Product(s): Miscellaneous Windows Products
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 11-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Sound System, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Voice Pilot application included with the Windows Sound System version 1.0
comes with 17 vocabulary files (.VCB) for some of the most popular Windows-based
applications. You can add more words or commands to these vocabulary files.
However, there is a limit of 4000 words per vocabulary file.

MORE INFORMATION
================

Voice Pilot comes with 17 vocabulary files to help you access application menus
faster and without the use of your mouse in many cases. Each vocabulary file has
a limit of 4000 words. After you have exceeded this limit, the application you
are using may behave erratically and Windows may perform slowly. Some of the
larger vocabulary files included with Voice Pilot have close to 1000 words. The
following is a list of some of the larger predefined vocabularies included with
Voice Pilot and the applications to which they apply:

  Application                                       Vocabulary Words
  ------------------------------------------------------------------

  WordPerfect for Windows, version 5.1                900
  Microsoft Word for Windows, version 2.0 or later    400
  Lotus 1-2-3 for Windows, version 1.0                400
  Microsoft Excel version 4.0                         330
  Aldus PageMaker version 4.0                         300
  Micrografx Designer version 3.1                     300

For more information about built-in vocabularies included with Voice Pilot, see
page 153 of the "Microsoft Windows Sound System User's Guide."
