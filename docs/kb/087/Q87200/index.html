Q87200: PROFS: When Is Gateway Component Successfully Installed?

Article: Q87200
Product(s): Microsoft Mail For PC Networks
Version(s): 3.0
Operating System(s): 
Keyword(s): 
Last Modified: 11-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When the PROFS gateway component is installed successfully, this message is
displayed at the bottom of the screen:

  Microsoft Mail Gateway to IBM PROFS and OfficeVision installed.

A similar message is displayed when the Gateway Access component is installed.

During the installation process, the screen displays the names of directories
being created in the mail system database and the files being copied from the
disks to the server. When the installation is complete, the Install program
exits to MS-DOS.

REFERENCES
==========

"Microsoft Mail Gateway to IBM PROFS and OfficeVision LAN Administrator's Guide"
