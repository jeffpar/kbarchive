Q87440: Money Manager by Expert Software May Cause GP Faults

Article: Q87440
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

Expert Software has confirmed that earlier versions of Money Manager may
cause general protection (GP) faults when running under Windows 3.1.
According to Expert Software, it does not use version numbers on most of
its products. If the file MM.VIP is dated earlier than 3/25/92, then it
will need to be upgraded to run correctly under Windows 3.1.

For more information, contact Expert Software technical support.

The Money Manager product included here is manufactured by Expert Software,
a vendor independent of Microsoft; we make no warranty, implied or
otherwise, regarding this product's performance or reliability.
