Q264725: SMS: Systems Management Server Online Support

Article: Q264725
Product(s): Microsoft Systems Management Server
Version(s): winnt:1.2,2.0
Operating System(s): 
Keyword(s): kbsms200 kbsms120 kbsmsAdmin kbUpgrade
Last Modified: 09-JUN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server versions 1.2, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Online information about current Systems Management Server (SMS) support issues
and updates is available at the following Microsoft Web sites:

- FAQs and highlights for SMS 2.0:

  http://support.microsoft.com/support/default.asp?PR=SMS200

- FAQs and highlights for SMS 1.2:

  http://support.microsoft.com/support/default.asp?PR=sms

Product highlights include current hot issues with detailed technical
descriptions and links to relevant articles in the Microsoft Knowledge Base.

MORE INFORMATION
================

For additional information about SMS, see the following Microsoft Web sites:

- Systems Management Server home page:

  http://www.microsoft.com/smsmgmt

- Microsoft TechNet SMS Technology Center:

  http://www.microsoft.com/technet/sms
