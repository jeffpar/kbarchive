Q286002: Combat Flight Simulator 2: How to Print a Flight Checklist

Article: Q286002
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbprint kbimu
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to print a flight checklist in Microsoft Combat
Flight Simulator 2.

MORE INFORMATION
================

To print a Combat Flight Simulator 2 flight checklist:

NOTE: The following steps use the default folder as the path to which you
installed Combat Flight Simulator 2.

1. Right-click Start, and then click Explore.

2. In the left pane, double-click the Program Files folder.

3. Double-click the Microsoft Games folder.

4. Double-click the Combat Flight Simulator 2 folder.

5. Click the Aircraft folder.

6. In the right pane, double-click the aircraft folder for which you want to
  print the checklist.

For example, to print the flight checklist for the F4U-1A Corsair aircraft:

   - Double-click the F4UIA_Corsair folder.

7. Double-click the aircraft checklist file.

For example, to print the flight checklist for the F4U-1A Corsair aircraft:

   - Double-click the F4ula_corsair_check file.

     NOTE: If you are prompted to select a program to open the file, use
     Notepad.

8. On the File menu, click Print.
