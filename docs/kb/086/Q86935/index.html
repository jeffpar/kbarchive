Q86935: 3COM: What Directories Are Created During Installation

Article: Q86935
Product(s): Microsoft Mail For PC Networks
Version(s): 3.0
Operating System(s): 
Keyword(s): 
Last Modified: 20-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

During the installation of the Microsoft Mail Gateway to 3Com 3+Mail, no
directories are created. You must create the directory for the gateway files
before installation.

REFERENCES
==========

"Microsoft Mail Gateway to 3Com 3+Mail Administrator's Guide"
