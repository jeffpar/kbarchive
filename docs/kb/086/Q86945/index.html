Q86945: 3COM: How to Control Polling for New Mail

Article: Q86945
Product(s): Microsoft Mail For PC Networks
Version(s): 3.0
Operating System(s): 
Keyword(s): 
Last Modified: 18-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

You can set the frequency of polling for the Microsoft Mail Gateway to 3Com
3+Mail by setting the FireInterval parameter in the GATEWAY.INI file. This
parameter specifies the elapsed time (in minutes) the gateway waits before it
checks for queued mail and starts transferring. The default is 10 minutes.

REFERENCES
==========

"Microsoft Mail Gateway to 3Com 3+Mail Administrator's Guide"
