Q86422: CalComp Printer Drivers for Windows

Article: Q86422
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

CalComp printers are not supported under Microsoft Windows. However, CalComp has
a printer driver, Zenographics SuperPrint, that supports printing in Windows.

MORE INFORMATION
================

The Zenographics SuperPrint driver is available directly from CalComp. You need
the following part number when ordering the driver:

  16205-0017

This printer driver by Zenographics supports the following printers:

- ColorMaster 5902

- ColorMaster Plus 6603VRC

- ColorMaster Plus 6613VRC

- ColorView 5912

- ColorView 5913

- PlotMaster 5902A

According to CalComp technical support, the driver supports printing TrueType
fonts in Windows 3.1. It also supports color printing in Windows 3.0 and 3.1.

For more information on the installation and use of this printer driver, contact
CalComp technical support.

This driver is manufactured by Zenographics and is distributed by CalComp.
Microsoft makes no warranty, implied or otherwise, regarding this product's
performance or reliability.
