Q107352: CD Sampler: Err Msg: Not Present Fault

Article: Q107352
Product(s): Microsoft Home Multimedia Titles
Version(s): Fall 1993 edition
Operating System(s): 
Keyword(s): 
Last Modified: 12-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Home CD Sampler Fall 1993 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

It has been reported that when running Microsoft Home CD Sampler Fall 1993
edition, intermittent Not Present Faults occur. The only way to recover from the
problem is to restart Windows.

RESOLUTION
==========

To attempt to resolve the situation, make sure the latest video driver
(available from the video manufacturer) is installed.

MORE INFORMATION
================

Microsoft has confirmed this to be a problem in Microsoft Home CD Sampler Fall
1993 edition. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.
