Q107816: WFWG 3.11 Setup Installs Older Logitech Mouse Driver

Article: Q107816
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.11
Operating System(s): 
Keyword(s): 
Last Modified: 29-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your Logitech three-button mouse loses some functionality after you install
Windows for Workgroups version 3.11.

CAUSE
=====

The Logitech Mouse driver supplied with Windows for Workgroups 3.11 is version
6.02. Windows for Workgroups 3.11 Setup overwrites newer versions of this driver
with version 6.02.

RESOLUTION
==========

To correct this situation, reinstall the latest version of the Logitech Mouse
software.

Additional query words: 3.11 set up LOGITECH DRIVER 6.02 6.3 6.12 5/22/92 lmouse wbuttons.exe lmouse.drv wbuttons middle button buttons 3 doubleclick double-click
