Q107058: OEM Modems May Be Supported by Windows for Workgroups

Article: Q107058
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.11
Operating System(s): 
Keyword(s): 
Last Modified: 02-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------


Many original equipment manufacturer (OEM) modem vendors sell modems
that are licensed from other vendors. For example, some Gateway
internal modems are actually U.S. Robotics Sportster modems.

If you plan to use an unsupported modem with Windows for Workgroups
Remote Access, you should contact your OEM and determine the true modem
brand and model. Then, choose the actual modem type when you install
Remote Access.
