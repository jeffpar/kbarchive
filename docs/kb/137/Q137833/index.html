Q137833: Error Occurs When Attempting to Remove an Autorun Program

Article: Q137833
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,1995-1996 edition
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 09-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Cinemania for Windows, 1992, 1993, 1994, 1995, 1996, 1997 editions 
- Microsoft Complete Basketball for Windows 1995-1996 edition 
- Microsoft Dogs for Windows, version 1.0 
- Microsoft Encarta 96 World Atlas for Windows 
- Microsoft Music Central for Windows, 1996, 1997 edition 
- Microsoft Oceans for Windows, version 1.0 
- Microsoft Reader's Digest Complete Do-It-Yourself Guide for Windows, version 1.0 
- Microsoft World of Flight for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Add/Remove Programs tool in the Windows Control Panel to remove
an installation of one of the programs listed above, you may receive the
following error message

  An Error Occurred while trying to remove Microsoft <progname>

  Uninstallation has been canceled.

where <progname> is the program's name.

CAUSE
=====

This error message can occur if you attempt to remove, change the Setup options,
or reinstall one of the programs listed earlier in this article and the
program's CD-ROM is not in the CD-ROM drive.

RESOLUTION
==========

To resolve this problem, insert the programs CD-ROM into the CD-ROM drive before
you attempt to make any installation changes.

MORE INFORMATION
================

Windows 95 and Windows 98 contain a feature called Auto Insert Notification. The
Auto Insert Notification feature can start Setup for the programs listed at the
top of this article automatically when you insert the program's CD-ROM. This
feature forces Windows to update program files and the Windows registry.

The Registry entries updated by the Auto Insert Notification feature are used by
the Add/Remove Programs tool in Control Panel.
