Q233046: Running MS Office 2000 Professional Comments and Corrections

Article: Q233046
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 08-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Running Microsoft Office 2000 Professional ISBN 1-57231-936-4 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Running Microsoft Office 2000 Professional,
ISBN 1-57231-936-4.

Contents:

- PhotoDraw Is Not Included In Office 2000 Professional

- FrontPage Is Not Included In Office 2000 Professional

- Chapter 43: Heading Refers To Access Instead Of Outlook.

- Page 978: Text Correction

- Page 989: Text Corrections

- Page 1019: Replace The Word Office With Outlook

MORE INFORMATION
================

PhotoDraw Is Not Included In Office 2000 Professional
-----------------------------------------------------

Delete the references to Microsoft PhotoDraw that are found on pages 5, 11, and
1263.


FrontPage Is Not Included In Office 2000 Professional
-----------------------------------------------------

Delete the references to Microsoft FrontPage that are found on pages 11 and
1256.


Chapter 43: Heading Refers To Access Instead Of Outlook
-------------------------------------------------------

On pages 1016 - 1044, the heading refers to Access but the chapter is about
Outlook.

Change the heading:
"Part V Microsoft Access"

To:
"Part VI Microsoft Outlook"


Page 978: Text Correction
-------------------------

On page 978, in the first paragraph, the following text corrections are
necessary.

Change:
"However, the Corporate Or Workgroup installation lacks some of the Internet
e-mail and Internet collaboration features provided by the Internet Only
installaion - for example, support for IMAP e-mail, group scheduling on the
Internet using the iCalendar protocol, and the ability to verify e-mail
addresses using directory services."

To:
"However, the Corporate Or Workgroup installation lacks some of the Internet
e-mail and Internet collaboration features provided by the Internet Only
installaion - for example, support for IMAP e-mail and group scheduling on the
Internet using the iCalendar protocol.


Page 989: Text Corrections
--------------------------

On page 989, in the first bulleted paragraph, make the following changes in the
first sentence.

Change:
" You can reopen the folder you previously had open by pointing to Go To on the
View menu and choosing it, or by clicking the Previous Folder button on the
Advanced toolbar."

To:
" You can reopen the folder you previously had open by clicking the Previous
Folder button on the Advanced toolbar."

On the same page, modify the second bulleted paragraph as follows.

Change:
"After you've gone back one or more times, you can go forward again by choosing
Next Folder from the Go menu or by clicking the Next Folder button on the
Advanced toolbar."

To:
"After you've gone back one or more times, you can go forward again by clicking
the Next Folder button on the Advanced toolbar."


Page 1019: Replace The Word Office With Outlook
-----------------------------------------------

On page 1019, in the second paragraph, in the second sentence:

Change:
"... (which runs after the Office 2000 Startup Wizard)..."

To:
"... (which runs after the Outlook 2000 Startup Wizard)..."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
