Q238052: Encarta 2000: Empty Window Opens When Starting Lesson Collection

Article: Q238052
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbenv kbinterop kbimu
Last Modified: 27-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Encarta Lesson Collection on the Features menu in Microsoft
Encarta Reference Suite 2000, Microsoft Internet Explorer starts and opens an
empty window.

CAUSE
=====

This behavior can occur if the following conditions are true:

- The original version of Microsoft Windows 95 (version 4.00.950) is installed
  on your computer.

- Microsoft Internet Explorer version 3.02 or earlier is installed on your
  computer.

RESOLUTION
==========

To resolve this issue, download and install the latest version of Internet
Explorer from the following Microsoft Web site:

  http://www.microsoft.com/windows/ie/download/default.htm

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Encyclopedia
2000.
