Q238042: Research Organizer: Program Is Listed Twice in Control Panel

Article: Q238042
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:; :
Operating System(s): 
Keyword(s): kbenv kbsetup kbimu
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Reference Suite 99 
- Microsoft Encarta Reference Suite 2000 
- Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to remove Microsoft Encarta Research Organizer, the program may
be listed twice in the list of installed programs in the Add/Remove Programs
tool in Control Panel.

CAUSE
=====

This behavior can occur if both Microsoft Encarta Reference Suite 99 and
Microsoft Encarta Reference Suite 2000 are installed on your computer.

RESOLUTION
==========

To work around this issue, remove Encarta Reference Suite 99.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

To prevent this issue from occurring, remove Encarta Reference Suite 99 before
you install Encarta Reference Suite 2000.
