Q179929: Games: Multiplayer Connection Types Are Not Available

Article: Q179929
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0,2.0
Operating System(s): 
Keyword(s): kbenv aoe ccabtf fs98 kbimu msgame KbDirectXkbfaq
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires, version 1.0 
- Microsoft CART Precision Racing, version 1.0 
- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 98 
- Microsoft Close Combat: A Bridge Too Far, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start a multiplayer game in one of the programs listed at
the top of this article, no connection types may be available.

CAUSE
=====

This behavior can occur if Microsoft DirectX is not installed correctly.

RESOLUTION
==========

To resolve this issue, reinstall DirectX. To do this, follow these steps:

1. Quit the program.

2. On the desktop, double-click My Computer.

3. Right-click the CD-ROM drive, and then click Reinstall DirectX. If Reinstall
  DirectX is not available, remove and then reinstall the program.

If the issue continues to occur, download and install the latest version of
DirectX from the following Microsoft Web site:

  http://www.microsoft.com/directx

For more information about how to obtain and install the latest version of
DirectX, please see the following article in the Microsoft Knowledge Base:

  Q179113 How to Download and Install DirectX
