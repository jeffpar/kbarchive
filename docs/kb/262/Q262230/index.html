Q262230: Microsoft Installer Programs and Systems Management Server 2.0

Article: Q262230
Product(s): Microsoft Systems Management Server
Version(s): 2.0,2.0 SP1
Operating System(s): 
Keyword(s): kbsms200
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server versions 2.0, 2.0 SP1 
- Microsoft Systems Management Server Installer version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Windows Installer is an installation technology that streamlines the
program installation process and offers continued resiliency for programs,
including the ability to install additional features on demand, automatically
repair programs, uninstall programs cleanly, and roll back programs reliably.
This technology is included with Windows 2000 and Systems Management Server
(SMS) 2.0.

When you use SMS 2.0 to deploy Windows Installer Setup packages, you gain
additional change and configuration benefits. For example, you can use SMS to
deploy Windows Installer Setup packages (.msi files) throughout your
organization.

Various white papers and Microsoft Knowledge Base articles exist that detail
general deployment and usage issues for Windows Installer.

MORE INFORMATION
================

Running an MSI script from an SMS advertisement may produce "Incorrect command
line parameters" error messages.

For additional information, review the following white papers and Microsoft
Knowledge Base articles:
