Q262070: Motocross Madness 2: Minimum System Requirements

Article: Q262070
Product(s): Microsoft Home Games
Version(s): 2.0
Operating System(s): 
Keyword(s): kbref kbimu msgame
Last Modified: 09-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a list of the minimum system requirements that are needed
to play Microsoft Motocross Madness 2.

MORE INFORMATION
================

To play Microsoft Motocross Madness 2, you need:

- A Multimedia PC with a Pentium II 300 MHz equivalent or higher processor or a
  Multimedia PC with a Pentium II 233 MHz equivalent or higher processor with 8
  megabytes (MB) hardware acceleration or better.

- Microsoft Windows 95 operating system or later, or Microsoft Windows 2000
  operating system.

- 64 megabytes (MB) of random access memory (RAM).

- 200 MB of available hard disk space for a minimum installation (with an
  additional 50 MB of available hard disk space for the swap file).

- A quad-speed or faster CD-ROM drive.

- A Super-VGA, 16-bit color monitor.

- A Microsoft Direct3D 7.0-compliant 3-D accelerator card with 8 MB of video
  RAM.

- A Microsoft mouse or compatible pointing device.

Optional:

- A Microsoft DirectSound 7.0 API-compatible sound card with speakers or
  headphones for audio.

- A joystick or game controller (Motocross Madness 2 supports force feedback
  input devices that are compatible with the Microsoft DirectInput 7.0 API).

- A 28,800 bits-per-second (bps) or faster modem, a local area network (LAN)
  with the TCP/IP or IPX protocol, or a null-modem cable to play multiplayer
  games.

- Microsoft Internet Explorer 3.02 or later or Netscape Communicator 4.0 or
  later to play multiplayer games on the Zone.com.

  NOTE: Internet access may require the payment of a separate fee to an Internet
  service provider. Connect time charges may apply.
