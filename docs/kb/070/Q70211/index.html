Q70211: Problem with Kodak DiconiX 150+ Windows Printer Driver

Article: Q70211
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SYMPTOMS
========

You are using the DiconiX printer driver supplied in the Supplemental Driver
Library for Microsoft Windows 3.0. The Printer Setup dialog box in the Control
Panel lets you choose a 22-inch paper size in the landscape mode, but the driver
only prints up to 17 inches on the first page and prints the remaining 5 inches
on the second page. You may also receive the error message:

  Unable to print to device selected.

RESOLUTION
==========

Kodak has confirmed this problem with the DiconiX printer driver.

Kodak is researching this problem and is currently working on a solution. For
more information, contact Kodak Product Support.
