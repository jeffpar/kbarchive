Q235013: PRB: Mastering Series Needs Installation of Internet Explorer 3

Article: Q235013
Product(s): Microsoft Mastering Series
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 15-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Mastering MFC (Microsoft Foundation Classes) Development 
- MSPRESS Microsoft Mastering Microsoft Visual Basic 5 
- MSPRESS Microsoft Mastering Microsoft Visual Basic 5 Fundamentals 
- MSPRESS Microsoft Mastering Web Site Development 
-------------------------------------------------------------------------------

SYMPTOMS
========

With Microsoft Internet Explorer 5 installed and when clicking a URL in any of
the Mastering products and Classroom courses, the URL link does not work. You
receive an error message stating that you must install Microsoft Internet
Explorer 3.0 or later.

CAUSE
=====

The Mastering Series is not aware of Internet Explorer 5.

RESOLUTION
==========

Fixes are available for download depending on which version of Mastering you
have.

MORE INFORMATION
================

For more details and downloadable fixes, click on one of the following links for
the Mastering product with which you are having this problem.

  http://www.microsoft.com/downloads/release.asp?ReleaseID=30267
