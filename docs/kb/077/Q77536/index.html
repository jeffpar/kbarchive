Q77536: Cannot Use PC Tools Diskcopy Command with Windows 3.0

Article: Q77536
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

If you run Central Point Softwares' PC Tools versions 4.0 through 5.01 in
standard or enhanced mode of Windows 3.0, the Diskcopy function in PC Tools is
not accessible.

There is currently no workaround for this problem.

MORE INFORMATION
================

For more information, contact Central Point Software.

The product included here is manufactured by a vendor independent of Microsoft;
we make no warranty, implied or otherwise, regarding this product's performance
or reliability.
