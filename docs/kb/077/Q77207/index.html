Q77207: Using Math Works MATLAB with MS-DOS 5.0

Article: Q77207
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.x
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

According to Math Works, the MATLAB software should be upgraded to version 3.5J
when using Microsoft MS-DOS 5.0. MATLAB version 3.5J is the latest version that
has been tested with MS-DOS 5.0. To obtain MATLAB 3.5J, contact Math Works.

MORE INFORMATION
================

MATLAB, a product of Math Works Inc., is a specialized fourth-generation
language designed for numerical analysis, data processing, and graphics
visualization. This application is primarily used for mathematical
implementations dealing with specific scientific and engineering situations. In
short, the MATLAB software, based on linear algebra, provides sophisticated
scientific number crunching. It should also be noted that Math Works provides
various builds of the MATLAB software for hardware- specific situations. These
include PC-MATLAB vx.x, AT-MATLAB vx.x, and 386- MATLAB vx.x, where x.x is the
specific build version number.

or reliability.
