Q167860: FP97: Cannot Choose Collate or Selection in Print Dialog Box

Article: Q167860
Product(s): Word Front Page
Version(s): 
Operating System(s): 
Keyword(s): kbprint kbdta
Last Modified: 26-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FrontPage 97 for Windows 
-------------------------------------------------------------------------------

For a Microsoft FrontPage 98 version of this article, see Q194281.

SYMPTOMS
========

The Collate Copies and Selection options are unavailable (dimmed) in the Print
dialog box (click Print on the File menu) when you attempt to print a FrontPage
file.

CAUSE
=====

The Print dialog box in Microsoft FrontPage is a Windows 95 common dialog box
that does not support the printing of either selected text or collating.

WORKAROUND
==========

To work around this problem, copy your text to an application that supports
collating or printing a selection of text.
