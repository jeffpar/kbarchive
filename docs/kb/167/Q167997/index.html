Q167997: WD97: Invalid Page Fault in KERNEL32.DLL with Browser Control

Article: Q167997
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kberrmsg kbtool word97kbfaq
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you add the Microsoft Web Browser Control (on the Control Toolbox toolbar,
click the More Controls button), and then click Exit Design Mode (on the Control
Toolbox), you receive the following error message when you quit Word:

  This program has performed an illegal operation and will be shut down. If the
  problem persists, contact the program vendor.

If you click Details, you will receive the following error message:

  WINWORD caused an invalid page fault in module KERNEL32.DLL

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
