Q131190: No Notification of Replies to Bulletin Board Messages

Article: Q131190
Product(s): The Microsoft Network
Version(s): WINDOWS:1.2 1.3
Operating System(s): 
Keyword(s): kbmsn
Last Modified: 14-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 1.2, 1.3 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Microsoft Network does not notify you when you have replies to messages you
have posted on bulletin boards.

RESOLUTION
==========

You must check your posted messages to see if they have replies. To save time,
you may want to create shortcuts to your messages.

STATUS
======

This behavior is by design, and is under review for possible change in a future
release.
