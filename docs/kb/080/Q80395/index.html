Q80395: TurboCom and Windows

Article: Q80395
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 17-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Pacific Commware's TurboCom is an advanced serial port communications driver for
Microsoft Windows.

MORE INFORMATION
================

TurboCom provides software support for 16550-style UARTs for Windows-based
applications, and provides MS-DOS-based applications access to the 16550's
features when the MS-DOS-based applications are run with Windows.

TurboCom provides these benefits in all modes of Windows. In addition, TurboCom
supports bit rates as high as 57.6 kilobits per seconds (Kbps), and provides
support for COM3 and COM4 on ISA bus computers, enabling the sharing of hardware
interrupts IRQ3 and IRQ4.

For additional information, contact Pacific Commware.
