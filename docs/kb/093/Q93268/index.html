Q93268: "Not Enough Global Memory" Using Crosstalk for Windows 1.2.2

Article: Q93268
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 17-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When attempting to dial from within Crosstalk for Windows version 1.2.2, you may
receive the following error message:

  Error 16-15
  Not Enough Global Memory

CAUSE
=====

This error message will occur if the Dial check box in the Action menu's Dial
command dialog box is not selected.

RESOLUTION
==========

To correct this problem:

1. Select the Dial check box.

2. In the phone-entry box to the right of the Dial check box, enter a phone
  number to be dialed.

3. Choose the Dial button.


For additional information, contact DCA, Inc.

Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
