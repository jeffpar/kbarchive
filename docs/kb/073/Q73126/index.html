Q73126: SysEdit Creates Backup Files with SYD Extension

Article: Q73126
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 01-NOV-1999

3.00 3.00a 3.10 3.11
WINDOWS
kbtool

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
- Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

After editing the AUTOEXEC.BAT, CONFIG.SYS, WIN.INI, or the SYSTEM.INI
files, SysEdit creates a backup of the original file and places an .SYD
extension on the file. For example, if a change was made to WIN.INI,
SysEdit saves the original file as WIN.SYD and the new file is saved as
WIN.INI. This also is true with the other files that are edited with
SysEdit.
