Q91719: Suppressing the Windows Logo

Article: Q91719
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 01-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
- Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Earlier versions of Microsoft Windows allow you to suppress the logo screen by
typing "win" (without the quotation marks) followed by a space. This does not
work with Microsoft Windows operating system version 3.0 or 3.1.

To suppress the opening logo screen in Windows 3.0 or 3.1, do the following:

1. Type "win" (without the quotation marks).

2. Press the SPACEBAR.

3. Press F7 and then press ENTER. (This sequence appears as WIN ^@ on the
  screen.)

You can also suppress the logo screen by having Windows automatically execute a
program at startup, by typing the following (where "program" is the program
name):

  WIN program

WARNING: Microsoft strongly discourages starting Windows by typing "win :".
Starting Windows with the WIN : command may cause problems and result in an
unstable system. WIN : was used in Windows versions 2.x to start the equivalent
of the run-time version of Windows with a given application. Also, WIN : allows
you to bypass the logo screen in versions 2.03, 2.1, 2.11, 3.0, and 3.1.

Additional query words: 3.00 3.00a 3.10 initiating disable screen startup
