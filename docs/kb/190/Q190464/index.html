Q190464: HOWTO: Get to the Visual FoxPro Registration Wizard

Article: Q190464
Product(s): Microsoft FoxPro
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): 
Last Modified: 11-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you install the standalone version of Visual FoxPro 6.0 and do not select the
Register Now check box, or you get an error when connecting to the registration
web page, this article describes how you can access the registration page.

MORE INFORMATION
================

To register Visual FoxPro 6.0, navigate to following Web site from Internet
Explorer 4.0 or later:

  http://register.microsoft.com/regwiz/forms/ 
    form69.asp?WizID=17&RM=&f=69&f=290&f=870&f=390&f=419&f=906&CP=1&LP=0
    &FU=http%3A%2F%2Fwww%2Emicrosoft%2Ecom%2FCU=http%3A%2F%2Fwww%
    2Emicrosoft%2Ecom%2F
