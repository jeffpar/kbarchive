Q169185: Monster Truck Madness Trial Version Is Available

Article: Q169185
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 20-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Monster Truck Madness, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The trial version of Microsoft Monster Truck Madness, version 1.0 is now
available for free download. The file is approximately 12 MB in size.

TO OBTAIN THIS FILE
-------------------

You can find Mtmtrial.exe, a self-extracting file, on the following services:

- Microsoft World Wide Web site on the Internet

  http://www.microsoft.com/games/monster

  Click the Downloads icon. Click the Trial Version link to download the file.

- Internet (anonymous FTP)

  FTP://FTP.microsoft.com
  Change to the Deskapps/Games/Public/Monster directory
  Get MTMTRIAL.EXE

Additional query words: 1.00 monstertruck MTM monster-truck trucks monstertrucks madness army armstrong big foot snake bite grave digger boogie van power wheels crusher
