Q169220: Hellbender Trial Version Is Available

Article: Q169220
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 20-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Hellbender for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The trial version of Microsoft Hellbender for Windows, version 1.0 is now
available for free download. The file is approximately 11 MB in size.

There is also an update to the trial version, which can be downloaded separately.
The update file is approximately 7 MB in size.

TO OBTAIN THESE FILES
---------------------

You can find Heltrial.exe and Upgrade.exe, both self-extracting files, on the
following services:

- Microsoft World Wide Web site on the Internet

  http://www.microsoft.com/games/hellbender

  Click the Demo Downloads icon. Click the Hellbender Trial Version link to
  download the file.

- Internet (anonymous FTP)

  FTP://FTP.microsoft.com
  Change to the Deskapps/Games/Public/Hellbender directory
  Get HELTRIAL.EXE - for the trial version
  Get UPGRADE.EXE - for the trial version upgrade
