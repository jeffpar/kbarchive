Q141521: Encarta Encyclopedia Mindmaze: Sound On Versus Sound Off

Article: Q141521
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:; :1996 edition
Operating System(s): 
Keyword(s): 
Last Modified: 24-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 96 Encyclopedia for Windows 
- Microsoft Encarta 97 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
- Microsoft Encarta 96 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

The options for Sound and Music in MindMaze appear to be reversed, but actually
work in this manner:

- If the Sound Effects are currently ON, you see the option to turn Sound
  Effects Off.

- If the Sound Effects are currently OFF then you see the option for Sound
  Effects On.

The settings for Music On and Music Off work similarly.

MORE INFORMATION
================

This method for changing the Sound or Music options is referred to as a toggle
option. To change these options, click the MindMaze menu.

For additional information, please see the following article in the Microsoft
Knowledge Base:

  Q158087 No Sound in Collages or MindMaze
