Q141852: Closing MS Info Version 2.5 Disables Sound

Article: Q141852
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,1996 edition
Operating System(s): 
Keyword(s): 
Last Modified: 06-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Cinemania for Windows 1996 edition 
- Microsoft Complete Gardening for Windows, version 1.0 
- Microsoft Encarta 96 Encyclopedia for Windows 
- Microsoft Encarta 96 World Atlas for Windows 
- Microsoft Music Central for Windows 1996 edition 
- Microsoft Reader's Digest Complete Do-It-Yourself Guide for Windows, version 1.0 
- the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you start and close Microsoft System Information version 2.5 from one of
the applications appearing at the top of this article and the computer system is
running Windows 95, all system sounds become disabled. Sounds appear to be
playing, yet no sound is heard.

NOTE: Encarta 97, Cinemania 97, Music Central 97, and World Atlas 97 use MS Info
version 2.51 which fixes this problem.

RESOLUTION
==========

To resolve this problem, use one of the following methods:

METHOD 1
--------

On the Task Bar, click the sound icon one time and click the Mute option. After
clicking the Mute box a small check mark will appear in the box.

NOTE: This method causes the system to turn the sound feature on. The Volume
Control dialog box indicates all sounds are muted and the Volume Control icon on
the Task Bar indicates sound functionality has been turned off.

METHOD 2
--------

Start Microsoft System Information for a second time and the sound settings will
return to normal.

METHOD 3
--------

Before closing Microsoft System Information, click the Taskbar sound and place a
check mark in the Mute option.

METHOD 4
--------

Restart the computer to reset system defaults.

MORE INFORMATION
================

The sound failure is system dependent. Computer exhibiting the reported problem
are systems with an installed Media Vision Pro Audio Studio 16 sound card.


Encarta Encyclopedia for Windows, 1996 edition has two versions of System
Information. Version 1.0 is in the Aamsstp folder of the CD disk and version 2.5
is in the Aamstp32 folder. Only the 32-bit version 2.5 causes the sound
problem.
