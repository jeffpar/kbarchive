Q141903: Text Missing from Bottom of Page Viewing Fax with Fax Viewer

Article: Q141903
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view or print a fax in Fax Viewer, text may be missing from the bottom
of one or more pages.

CAUSE
=====

This problem can occur if a fax is sent to a Windows for Workgroups-based
computer, and then forwarded to a Windows 95-based computer.

RESOLUTION
==========

To work around this problem, view or print the fax on the Windows for
Workgroups-based computer that originally received the fax.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.


MORE INFORMATION
================

Windows for Workgroups receives Class 2 faxes in DCX format. When faxes that
have been converted to .dcx files are forwarded to a Windows 95-based computer
and opened in Fax Viewer, text may be missing from the bottom of one or more
pages.
