Q141000: Multimedia: Black Screen or No Sound When Starting Program

Article: Q141000
Product(s): Microsoft Home Multimedia Titles
Version(s): 1995
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 500 Nations, version 1.0 
- Microsoft Cinemania for Windows, 1992, 1993, 1994, 1995, 1996, 1997 editions 
- Microsoft Dogs for Windows, versions 1.0, 1.0a 
- Microsoft Music Central for Windows, 1996, 1997 edition 
- Microsoft Oceans for Windows, version 1.0 
- Microsoft World of Flight for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The first time you start any of the Multimedia programs listed above, the screen
may turn black and you may not hear any sound.

CAUSE
=====

This issue can occur if the default startup mode is set to full screen. The full
screen startup mode displays the program at a resolution of 640 x 480 pixels and
the Windows display resolution may be configured for a different resolution.

RESOLUTION
==========

To resolve this issue, use the following steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Display.

3. Click the Settings tab, note the Desktop Area or Screen Area setting, and
  then move the Desktop Area or Screen Area slider all the way to the left (the
  640 x 480 setting).

4. Click OK, and then follow the instructions on the screen to keep the new
  display resolution setting.

5. Close Control Panel.

6. Start the Multimedia program.

7. Click Options, and then click the Run Full Screen check box to clear it.

8. Close the Multimedia program.

9. Repeat steps 1-4 to change the display resolution back to the original
  setting you noted in step 3.
