Q95385: QMS PS PostScript Printer Driver Version 2.43.04

Article: Q95385
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

QMS has a Windows 3.0 and 3.1 PostScript driver available for its QMS
ColorScript and PostScript laser printers. You can get this driver from QMS
technical support or through the QMS bulletin board service. As of 11/15/93, the
version number of this driver is 2.43.04.

Microsoft has not tested this driver. Please contact QMS if you have any problems
or questions concerning its use.


MORE INFORMATION
================

This driver supports the following printers:

  QMS ColorScript 100 Model 10
  QMS ColorScript 100 Model 10p
  QMS ColorScript 100 Model 30/30i
  QMS ColorScript 100 Model 30si
  QMS ColorScript 210
  QMS ColorScript 230
  QMS-PS 410
  QMS-PS 430J
  QMS-PS 460J
  QMS-PS 800
  QMS-PS 800+
  QMS-PS 810
  QMS-PS 810 turbo
  QMS-PS 820
  QMS-PS 820 turbo
  QMS-PS 815
  QMS-PS 825
  QMS-PS 815 MR
  QMS-PS 825 MR
  QMS 860 Print System
  QMS-PS 1500
  QMS-PS 1700
  QMS-PS 2000
  QMS-PS 2200
  QMS-PS 2210
  QMS-PS 2220
  QMS-PS 3200
  QMS IS X320T (Imageserver X320T)

The products included here are manufactured by QMS, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding these products'
performance or reliability.

For more information, contact QMS technical support.
