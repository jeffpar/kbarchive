Q271535: MCM2: Computer Hangs When You Quit and Restart Game

Article: Q271535
Product(s): Microsoft Home Games
Version(s): 2.0
Operating System(s): 
Keyword(s): kbenv kbimu msgame
Last Modified: 12-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you quit a multiplayer game of Microsoft Motocross Madness 2, your computer
may stop responding (hang) when you restart the game.

CAUSE
=====

This behavior can occur if both of the following conditions are true:

- The multiplayer game used a TCP/IP connection.

- Motocross Madness 2 is installed on a Microsoft Windows 2000-based computer.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Motocross Madness 2,
version 2.0.

Additional query words: 2.00 msgame mcm2 motorcross motorcycle other multiple players
