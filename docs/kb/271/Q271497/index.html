Q271497: Golf 2001: Multiplayer Options Are Unavailable

Article: Q271497
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kbenv kbui kbimu msgame
Last Modified: 22-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf 2001 Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to play Microsoft Golf 2001 Edition with other players, the
multiplayer options in the game may be unavailable or inactive.

CAUSE
=====

This behavior can occur if Golf 2001 Edition or the DirectPlay component of
Microsoft DirectX are not properly registered.

RESOLUTION
==========

To resolve this issue, shut down and restart the computer, uninstall Golf 2001
Edition, and then reinstall Golf 2001 Edition.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Golf 2001 Edition.

MORE INFORMATION
================

This issue is most likely to occur both of the following conditions are true:

- The retail version of Microsoft Windows 95 is installed on the computer.

- An early version of DirectX is installed on the computer.

Additional query words: msgame msgolf golf2001 dimmed grayed greyed dplay direct play
