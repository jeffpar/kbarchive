Q260155: Visual FoxPro 6.0 Sample: How To Use Animation Control

Article: Q260155
Product(s): Microsoft FoxPro
Version(s): WINDOWS:6.0
Operating System(s): 
Keyword(s): kbfile kbsample kbActiveX kbCtrl kbvfp600 kbGrpDSFox kbDSupport
Last Modified: 11-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Animate.exe contains a sample Visual FoxPro form that shows the use of the
Microsoft Animation Control to play Audio-Video Interleaved (.AVI) files.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  Animate.exe
  (http://download.microsoft.com/download/vfox60/vfpanim/6/w9x2k/en-us/Animate.exe)

Release Date: Apr-26-2000

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

The Animate.exe file contains the following files:

+--------------------+
| File name   | Size | 
+--------------------+
| AviForm.scx | 2KB  | 
+--------------------+
| AviForm.sct | 7KB  | 
+--------------------+
| ReadMe.txt  | 1KB  | 
+--------------------+

1. Start Visual FoxPro.

2. Run the form called AviForm.scx from the Command window (DO FORM AVIFORM.SCX
  ).

3. Use the OPEN, PLAY, and STOP buttons to open, play, and stop .AVI files.
