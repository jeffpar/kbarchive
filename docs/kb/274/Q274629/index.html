Q274629: Games Hangs or Slow Performance on AMD System w/ VIA Chipset

Article: Q274629
Product(s): Microsoft Home Games
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 30-JUN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call: Dark Majesty, version 1.0 
- Microsoft Asheron's Call, version 1.0 
- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
- Microsoft Dungeon Siege, version 1.0 
- Microsoft Flight Simulator 2002 
- Microsoft Flight Simulator 2002 Professional Edition 
- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
- Microsoft Midtown Madness 2, version 2.0 
- Microsoft MechWarrior 4: Vengeance 
- Microsoft MechWarrior 4: Black Knight 
- Microsoft Train Simulator, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When using one of the Microsoft Games listed at the beginning of this article on
a system with a motherboard based on the VIA Technologies chip set, users may
experience one or more of the following symptoms:

- Slow game performance or your mouse may move very slowly when navigating
  in-game.

- Game may appear to stop responding (freeze, hang).

- Your screen may go black.

- When playing a multiplayer game, you may experience slow performance in your
  game (lag).

- Your game may run but you cannot shut down the game and restart the game
  multiple times without a restart.

CAUSE
=====

This behavior may occur if you have not installed the most recent drivers for
your VIA Technologies chip set motherboard.

RESOLUTION
==========

To resolve this issue, install the latest VIA 4-in-1 driver from the following
VIA Technologies Web site:

  http://www.viatech.com

IMPORTANT: Contact your hardware manufacturer before you install the latest VIA
4-in-1 driver on your computer or make any modifications to your computer's BIOS
settings.

For information about how to contact your hardware manufacturer, click the
appropriate article number in the following list to view the article in the
Microsoft Knowledge Base:

  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K

  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P

  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z

MORE INFORMATION
================

How to Identify Your Hardware Chipset using Device Manager
----------------------------------------------------------

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click System.

3. Microsoft Windows 98 or Microsoft Windows Millennium Edition, click the
  Device Manager tab.

  Microsoft Windows 2000 or Microsoft Windows XP, click the Hardware tab, and
  then click Device Manager.

4. Expand the list next to the Hard Disk controllers or IDE ATA/ADAPI Controller
  folder.

5. If VIA Bus Master IDE Controller is present, your motherboard is based on VIA
  chipset.
