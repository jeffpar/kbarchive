Q274605: CFS2: Computer Hangs When You Start Game or Mission Builder

Article: Q274605
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kbdisplay kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Combat Flight Simulator 2: WWII Pacific Theater 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Combat Flight Simulator 2: WWII Pacific Theater or
Mission Builder (included with Combat Flight Simulator 2), the screen may turn
black, and your computer may stop responding (hang).

CAUSE
=====

This behavior can occur if your video adapter, 3D accelerator card, or monitor
does not support one of the following screen resolutions in full-screen mode:

- 800 by 600 pixels
- 1024 by 768 pixels

The menu screens in Combat Flight Simulator 2 can only be displayed at a screen
resolution of 800 by 600 pixels.

Mission Builder can only run at a screen resolution of 1024 by 768 pixels.


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Combat Flight
Simulator 2: WWII Pacific Theater, version 1.0.
