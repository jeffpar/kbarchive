Q274455: MSN Explorer Setup Or Update Fails When Not Enough Disk Space

Article: Q274455
Product(s): The Microsoft Network
Version(s): 6.0,6.1
Operating System(s): 
Keyword(s): 
Last Modified: 02-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 6.0, 6.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

MSN Explorer ceases to function after an update or version upgrade.

CAUSE
=====

Limited disk space will cause the install or upgrade packets to fail while
loading.

RESOLUTION
==========

Uninstall MSN Explorer, free up additional disk space, and then reinstall MSN
Explorer.

1. For instructions on freeing up more disk space, click Start and click Help.

2. Under the Search tab type Disk Cleanup and then click List Topics.

3. Follow the instructions under the listed topics for removing unneeded
  programs and using Disk Cleanup.

MORE INFORMATION
================
