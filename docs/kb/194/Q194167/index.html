Q194167: Encarta: Description of Compound Interest InterActivity

Article: Q194167
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:; :
Operating System(s): 
Keyword(s): kbtool kbui kbimukbfaq
Last Modified: 27-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Encyclopedia 99 
- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
- Microsoft Encarta Reference Suite 99 
- Microsoft Encarta Reference Suite 2000 
- Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes some of the limitations of the Compound Interest
InterActivity in Microsoft Encarta Encyclopedia.

MORE INFORMATION
================

You can only type whole numbers in the Monthly Contribution and Interest Rate
boxes.

The monthly contribution must be a number between 1 and 9999, and the interest
rate must be a number between 1 and 99999.

You cannot type a COMMA (,) or PERIOD (.) in the Monthly Contribution box or the
Interest Rate boxe.

If the amount of time required to accumulate $1,000,000 is very long (for
example, over 100 years), you are unable to see the entire results chart.

NOTE: The Compound Interest InterActivity calculates the correct number of years
you need to save $1,000,000, but it rounds the value to the nearest whole year.
