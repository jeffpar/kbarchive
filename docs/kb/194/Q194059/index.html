Q194059: Encarta Virtual Globe 99: How to Print the Map Legend

Article: Q194059
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbprint kbui kbimu
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to print the map legend in Microsoft Encarta Virtual
Globe 99.

MORE INFORMATION
================

When you click the Legend tab in Encarta Virtual Globe 99 to display the map
legend, the legend is shown inside an expanded tab overlying the map displayed
on the screen. To print the map legend, right-click inside this expanded tab,
click Print, and then click OK.

NOTE: If you want to print both the map legend and the map on the same page, copy
both the map legend and the map into a graphics program such as Microsoft Paint,
and then print the page from this program.
