Q194752: Golf 98/99: Distances Are Not Measured in Expected Units

Article: Q194752
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbenv kbui kbimu msgame golf98
Last Modified: 22-MAY-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf 1998 Edition, version 1.0 
- Microsoft Golf 1999 Edition, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Golf, distances may not be measured in the units you
expect. For example, distances may be measured in meters instead of in yards.

CAUSE
=====

This behavior can occur if your computer is configured to use a different
measurement system.

RESOLUTION
==========

To resolve this issue, specify the measurement system you want to use in the
Regional Settings tool in Control Panel. To do this, follow these steps:

1. Quit Golf if it is running.

2. Click Start, point to Settings, and then click Control Panel.

3. Double-click Regional Settings.

4. Click the Number tab.

5. Click the measurement system you want to use in the Measurement System box,
  and then click OK.

6. Close Control Panel.

Additional query words: 1.00 msgame golf98 international metric metres
