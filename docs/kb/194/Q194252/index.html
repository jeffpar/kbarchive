Q194252: WD97: Table with Borders and Shading Prints Slowly

Article: Q194252
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbprint kbdta winword word97
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print a Word 97 document that contains a large table with borders
and/or shading to a Hewlett-Packard (HP) LaserJet 4 printer, the document may
print very slowly.

Changing the printer driver settings does not correct the problem; in fact, this
may make the delay even more noticeable.

This problem has also been reported on the Okidata OL-400, Okidata OL-830, NEC
Pinwriter 6300, NEC 6300, Panasonic KXP 4420, and Epson LQ 2550.

This problem does not occur when printing to an HP LaserJet IIIsi or 4si.


WORKAROUND
==========

Use the HP LaserJet IIIsi or 4si printer driver.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

Additional query words: slow slower slows slowed down
