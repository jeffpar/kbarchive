Q164897: Zone: Description and Policies of the Web site

Article: Q164897
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): kburl igz kbimu msgamekbfaq
Last Modified: 30-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Zone.com 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains a brief description of the Zone.com Web Site and
information regarding its polices.

MORE INFORMATION
================

Description:

Zone.com is an Internet web site where you can play classic, retail, and premium
games with other players. The Zone also provides forums where individuals can
chat and find other people interested in playing a wide variety of computer
games. The Zone is located at:

  http://zone.msn.com

Terms of Use:

To view the Terms of Use for the Zone, please see the following MSN Web site:

  http://www.msn.com/help/legal/terms.htm

Privacy:

To view the Privacy policy for the Zone, please see the following MSN Web site:

  http://www.msn.com/help/legal/privacy.htm

Code of Conduct:

To view the Code of Conduct for the Zone, please see the following MSN Web site:

  http://zone.msn.com/zzzz/help/zonehelpcodeofconduct.asp
