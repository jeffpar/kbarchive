Q164502: WD97: Some Files Remain After "Remove All"

Article: Q164502
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbsetup word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

Setup leaves certain files because they are shared by other programs. This
article lists those folders and files that remain after you select "Remove All"
during Word Setup.

The following folder is not removed:

  Program Files\

The files listed in the following table are not removed.
