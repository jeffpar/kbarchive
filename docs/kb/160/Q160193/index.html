Q160193: Encarta 97 Err Msg: Visual C++ Runtime Error

Article: Q160193
Product(s): Microsoft Home Multimedia Titles
Version(s): 1997
Operating System(s): 
Keyword(s): 
Last Modified: 12-AUG-1999

1997
WINDOWS
kbmm kberrmsg kbusage kbprb

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 97 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

While you are in Guided Tours, you may receive the following error message:

  Visual C++ Runtime Error.

This occurs when you click Cancel in the Insert Compact Disc dialog box after you
eject the Encarta compact disc from the CD-ROM drive:

RESOLUTION
==========

Do not eject the Encarta CD-ROM while you are trying to access a guided tour. If
this occurs, return the disc to the drive and click OK.

STATUS
======

Microsoft has confirmed this to be a problem with the 1997 edition of Encarta
Encyclopedia for Windows. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
