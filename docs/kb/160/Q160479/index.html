Q160479: Encarta 97: Computer Hangs with Sound Galaxy Pro 16

Article: Q160479
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 97 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
- the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Encarta Encyclopedia in Windows 3.X and you are using an Aztech
Sound Galaxy Pro 16 sound card (driver version 1.02), the computer stops
responding (hangs).

CAUSE
=====

This occurs because of a Direct Memory Access (DMA) conflict with your sound
card configuration.

RESOLUTION
==========

Change the Direct Memory Access (DMA) settings for your sound card. You may also
need an a sound driver update. Contact Aztech Labs to find the most recent
version available for your particular card.
