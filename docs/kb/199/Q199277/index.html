Q199277: Installing Windows NT 4.0 SP4 Requests Windows NT 4.0 SP3 CD-ROM

Article: Q199277
Product(s): Microsoft Windows NT
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbenv kbsetup
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server, Enterprise Edition version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you upgrade Windows NT 4.0 Enterprise Edition to the 128-bit version of
Windows NT 4.0 Service Pack 4 (SP4), you may need to insert the Windows NT 4.0
Service Pack 3 (SP3) CD-ROM to complete the installation.

RESOLUTION
==========

To work around this issue, follow these steps:

1. Uninstall Windows NT 4.0 SP4.

2. Install Windows NT 4.0 SP3.

3. Install Windows NT Server 4.0 Enterprise Edition.

4. Install Windows NT 4.0 SP4, 128-bit version.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Windows NT 4.0,
Enterprise Edition.Microsoft is researching this problem and will post more
information here in the Microsoft Knowledge Base when it becomes available.
