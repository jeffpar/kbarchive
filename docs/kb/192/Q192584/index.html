Q192584: Tutor: Online Registration Wizard Does Not Start

Article: Q192584
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbinterop kbtool mskids kbimu
Last Modified: 04-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft My Personal Tutor: Sam's Hide and Seek Adventure, version 1.0 
- Microsoft My Personal Tutor: Sky's Space Station Voyage, version 1.0 
- Microsoft My Personal Tutor: Turru's Daring Sea Quest, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to use the Online Registration wizard to register one of the My
Personal Tutor titles listed in the applies to section of this article, the
Online Registration icon may be a generic Microsoft Windows icon, and the Online
Registration wizard may not start.

CAUSE
=====

This behavior can occur if a Web browser is not installed on the computer.

RESOLUTION
==========

To resolve this issue, install a Web browser on the computer. You can download
and install the latest version of Microsoft Internet Explorer from the following
Microsoft Web site:

  http://www.microsoft.com/ie

MORE INFORMATION
================

To start the Online Registration wizard, click Start, point to Programs, point
to Microsoft My Personal Tutor, and then click Online Registration.

Additional query words: mskids mpt square box blank run open
