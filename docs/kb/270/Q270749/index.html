Q270749: ELL: Windows Taskbar Covers Encarta Language Learning Window

Article: Q270749
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbenv kbimu
Last Modified: 28-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Language Learning French 
- Microsoft Encarta Language Learning French Deluxe 
- Microsoft Encarta Language Learning Spanish 
- Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start any of the programs listed at the beginning of this article, the
Microsoft Windows taskbar may cover part of the Encarta Language Learning
window.

CAUSE
=====

This behavior can occur if both of the following conditions are true:

- The Windows taskbar is docked to the top of the screen or the left side of
  the screen.

- Windows is configured to use a resolution of 1024 x 768 pixels or higher.

RESOLUTION
==========

To resolve this issue, use one of the following methods.

Enable the Auto-hide Feature
----------------------------

To enable the auto-hide feature for the Windows taskbar:

1. Click Start, point to Settings, and then click Taskbar & Start Menu.

2. Click to select the "Auto hide" check box.

3. Click OK.

Move the Windows Taskbar
------------------------

To move the Windows taskbar, drag the Windows taskbar to the bottom of the
screen.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
