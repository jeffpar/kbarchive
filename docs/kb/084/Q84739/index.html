Q84739: Mac Fax: FaxINIT Versions

Article: Q84739
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The FaxINIT extension is specifically designed to work with a certain fax modem.
You cannot interchange FaxINITs between different fax modems. The FaxINIT can be
identified by checking the version number that appears in the Get Info
information from the Finder.

FaxINIT Versions      Compatible Modem
----------------      ----------------

FaxINIT 1.2.1         AppleFAX modem
FaxINIT 1.2.1o        Orchid modem
FaxINIT 1.2.1cr       FlexFax modem
FaxINIT 1.2.1t        TEFAX modem

Additional query words: 2.00 2.00a 2.00b
