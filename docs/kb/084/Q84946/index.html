Q84946: Okidata Microline 320, 321 Correct Print Quality Selection

Article: Q84946
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

The Okidata 320 and 321 series of printers have High Speed Draft (HSD), Utility
(UTL), and Near Letter Quality (NLQ) print modes available. The printer drivers
that are included with Microsoft Windows version 3.1 do not list these settings
accurately.

MORE INFORMATION
================

The Okidata 320 and 321 are 9-pin printers that Okidata configures with
Microline, Epson, or IBM emulations. The Microsoft Windows printer drivers
support the Microline and IBM emulations directly.

The following chart outlines the correct settings available on the printer and
the corresponding print settings available on the Microsoft Windows 3.1 printer
drivers.

Emulation               Printer                 Windows 3.1
---------               -------                 -----------
Okidata Microline       High Speed Draft        Draft
                       Utility                 Near Letter Quality
                       Near Letter Quality     Letter Quality

Okidata ML-IBM          High Speed Draft        No Option
                       Utility                 No Option
                       Near Letter Quality     No Option
