Q84266: Err Msg: This DLL Can Only Be Used with ToolBook Evaluation…

Article: Q84266
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,1.0a; WINDOWS:3.x
Operating System(s): 
Keyword(s): 
Last Modified: 01-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Multimedia Beethoven, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Multimedia Beethoven may display the following error message if you
have the Multimedia ToolBook evaluation edition installed into the system:

  This DLL can only be used with the ToolBook evaluation edition.

MORE INFORMATION
================

Multimedia Beethoven uses the ToolBook runtime version 1.5. To alleviate the
problem, rename the MTBEVAL directory or remove it from the path in the
AUTOEXEC.BAT file. MTBEVAL is a demonstration version of the Multimedia ToolBook
product.

ToolBook and the Multimedia ToolBook evaluation edition are developed outside of
Microsoft. For further information regarding ToolBook or the Multimedia ToolBook
evaluation edition, contact Asymetrix at (206) 637-1600.

NOTE: This problem does not occur in Microsoft Windows 95.

Additional query words: 1.00 demo mtbeval mtb ludwig toolbook dev/drv
