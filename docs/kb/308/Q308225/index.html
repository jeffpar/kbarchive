Q308225: Error Message: Windows Installer Internal Error 2889_ISAPATHDLG

Article: Q308225
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0,4.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 16-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Works 6 
- Microsoft Works Suite 2001 
- Microsoft Works 2000 
- Microsoft Picture It! Publishing 2001 
- Microsoft Encarta Reference Library 2002, version 1.0 
- Microsoft FrontPage 2002 
- Microsoft Streets & Trips 2002, version 1.0 
- Microsoft MapPoint 2002 
- Microsoft IntelliPoint software, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install any of the products listed at the beginning of this article,
you may receive something similar to the following error message:

  Windows Installer
  Internal error 2889_ISAPATHDLG

When opening a Microsoft Works 2000 Word Processor document, you may receive the
following error message:

  wkswp caused an IPF in module user32.dll

  -or-

  MSWorks caused an invalid page fault in module wklnac.dll.

When setup for Microsoft Streets & Trips 2002 initializes, you may receive
the following error message:

  Internal error 2889, Customer information, name edit

When starting Microsoft Works Suite 2001, you may receive either of the following
messages:

  MSWORKS caused invalid page fault in module unknown.

  - or -

  Error: Cannot start works.

CAUSE
=====

This behavior can occur if the computer is infected with the W32.Nimda.A@mm
virus.

RESOLUTION
==========

To determine whether your computer has this virus, run antivirus software that
contains the latest virus signature file, and perform a thorough scan of your
computer.

Contact your antivirus software provider to remove this virus.
