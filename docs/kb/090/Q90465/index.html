Q90465: WinLogin Support Policies

Article: Q90465
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11; :1.0
Operating System(s): 
Keyword(s): 
Last Modified: 01-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
- Microsoft WinLogin, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses WinLogin Support Policies for Microsoft Product Support
Services (PSS).

This information applies to version 1.0 of Microsoft WinLogin. For information
about ordering Microsoft WinLogin, call the Microsoft Sales Information Center
(MSIC) at (800) 426-9400, or mail the form supplied with the Windows Resource
Kit (WRK) for the Microsoft Windows operating system version 3.1.

MORE INFORMATION
================

PSS Can Support Only WinLogin Administrators
--------------------------------------------

Microsoft PSS can support only WinLogin setup and usage when the system
administrator is calling PSS. Microsoft requires that the system administrator
calls so that our engineers can work with someone who has the proper tools and
passwords to work with the WinLogin installation, to preserve the WinLogin
security functionality.

This policy DOES NOT affect Windows users who are running WinLogin and are having
Windows usage problems.

PSS Will Not Help Break WinLogin Password Protection
----------------------------------------------------

Microsoft PSS engineers will not help any user, including the system
administrator break the password security in WinLogin. If you have forgotten
your password, please see your system administrator. If you are the system
administrator and have forgotten you password, your only solution is to
reinstall the WinLogin database. This will let you define a new password, but
will also destroy the original database and all data it contains, including user
and workstation definitions and merge rules.

PSS Will Not Help Users Disable WinLogin
----------------------------------------

In keeping with the above password security integrity, Microsoft PSS engineers
cannot help customers, other than the WinLogin administrators, disable WinLogin
nor do engineers tell them how to do it.

For more information about WinLogin, query on the following word in the Microsoft
Knowledge Base:

  WinLogin
