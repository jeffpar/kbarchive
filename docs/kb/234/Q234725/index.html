Q234725: PRB: Microsoft Pocket Guide to Microsoft Outlook 2000 Comments a

Article: Q234725
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 04-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Pocket Guide to Microsoft Outlook 2000 ISBN 1-57231-978-X 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Pocket Guide to Microsoft Outlook
2000, ISBN 1-57231-978-x.

The following topics are covered:

- Page 24: Line Points To Wrong Part Of Screen Shot

- Page 34: Procedure Incorrect

- Page 91: Procedure Incorrect

- Page 95: Incorrect Button Names

- Page 103: Outlook Newsreader Information Incorrect

- Page 127: Lines Point To Wrong Part Of Screen Shot

- Page 149: vCard Procedure Incorrect

MORE INFORMATION
================

Page 24: Line Points To Wrong Part Of Screen Shot
-------------------------------------------------

On page 24, in the AutoPreview section, the line should point to the message from
Jason Gerund.


Page 34: Incorrect Procedure
----------------------------

The second sentence of the Connection Wizard entry is incorrect. It should read:
"To use the Connection Wizard, click the Start button and then choose Programs,
Accessories, Communications, and Internet Connection Wizard."


Page 91: Incorrect Procedure
----------------------------

The last two sentences in the first paragraph on the page (under Starting
NetMeeting) are incorrect. They should read:
"...choosing the View menu's Go To command and the submenu's Internet Call
command. Then choose From Address Book."


Page 95: Incorrect Button Names
-------------------------------

Under Posting Messages, the second sentence should read: "...the New Post toolbar
button."

The third sentence should read: "...and click the Send toolbar button."

Under Posting Messages with Attachments, the first sentence should read:
"...click the Attach File toolbar button."


Page 103: Outlook Newsreader Information Incorrect
--------------------------------------------------

The last sentence of the Outlook Express entry is incorrect. It should read:
"It turns out that Outlook Express is also the newsreader for Outlook."


Page 127: Lines Point To Wrong Part Of Screen Shot
--------------------------------------------------

On page 127, in the Adding a Second Time Zone section, the lines should point to
the lower half of the screen shot.


Page 149: vCard Procedure Incorrect
-----------------------------------

The 4th and 5th sentences in the vCard entry are incorrect. It should read:
"To create a vCard, display the Contacts folder, select the contact for whom you
want to create a vCard, and then chose the File menu's Save As command. When
Outlook displays the Save As dialog box, select vCard Files from the Save As
Type drop-down list box."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
