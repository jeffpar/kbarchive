Q234953: Windows 2000 Beta Training Kit Comments and Corrections

Article: Q234953
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocfix kbdocerr
Last Modified: 09-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Windows 2000 Beta Training Kit ISBN 0-7356-0644-7 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Windows 2000 Beta Training Kit,
ISBN 0-7356-0644-7.

The following topics are covered:

- Page 55: File Systems Accept File Oriented I/O Requests

- Page 278: Cannot Have Duplicate RDNs Within Single OU

- Page 689: Answer to Question 8 Should Say My Documents

MORE INFORMATION
================

Page 55: File Systems Accept File Oriented I/O Requests
-------------------------------------------------------

Add the word "file" to the first sentence on page 55.

Change:
"Accept the oriented I/O requests and translate these requests into
device-specific calls."

To:
"Accept the file oriented I/O requests and translate these requests into
device-specific calls."


Page 278: Cannot Have Duplicate RDNs Within Single OU
-----------------------------------------------------

On page 278, in the first paragraph, change the third sentence from:
"can exist in separate OUs because..."

To:
"can exist in separate DCs because..."

In Figure 12.7 change:
"Users OU"

To:
"Users DC"

Also in Figure 12.7 change:
"Sales OU"

To:
"Sales DC"


Page 689: Answer to Question 8 Should Say My Documents
------------------------------------------------------

On page 689, in the answer to question 8, change the default folder to My
Documents.

Change:
"The default folder for saving a customized console is My Administrative Tools,
which MMC creates when you save the first customized console."

To:
"The default folder for saving a customized console is My Documents."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
