Q193383: Research Organizer: PC-Cillin II Finds Virus in Soccer.ero File

Article: Q193383
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbimu
Last Modified: 12-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 98 Research Organizer, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the virus protection program PC-Cillin to scan your hard disk, you
may receive a warning message that the Soccer.ero file located in the following
folder contains the WORD-Wazzu.BP-Z virus

  <drive>:\Program Files\Microsoft Reference\Encarta Encyclopedia
  \Sample Projects\Soccer.ero

where <drive> is the letter of the drive on which Encarta 98 Research
Organizer is installed.

CAUSE
=====

This behavior can occur if the PC-Cillin virus signature file is outdated.

RESOLUTION
==========

To resolve this issue, TouchStone Software recommends that you upgrade to
PC-Cillin 3.0, or upgrade the PC-Cillin scan engine to the version that supports
pattern file 326 or later.

To obtain the PC-Cillin 3.0 upgrade, call the TouchStone Software order desk at
(800) 932-5566 or visit the PC-Cillin II Internet Virus Lab.

To obtain the PC-Cillin scan engine upgrade, visit the following TouchStone
Software Web site:

  http://www.checkit.com

MORE INFORMATION
================

For more information about PC-Cillin, visit the TouchStone Software Technical
Support Chat Room at the following Web address:

  http://www.checkit.com/chatroom/chatroom.htm
