Q193366: INFO: CAB Files Distributed with Visual Basic 6.0

Article: Q193366
Product(s): Microsoft Visual Basic for Windows
Version(s): 6.0
Operating System(s): 
Keyword(s): kbList kbInternet kbVBp kbVBp600 kbGrpDSVB kbDSupport
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft provides digitally signed cabinets (.CAB files) on the Internet for
download containing core Visual Basic 6.0 files. To download these CAB files,
type "http://activex.microsoft.com/controls/vb6/<xxx.cab>" (without the
quotation marks) (where <xxx.cab> is the name of the CAB file that you
want to download).

MORE INFORMATION
================

Following are the links to the Microsoft Visual Basic 6.0 digitally signed .CAB
files. Navigate to the following links and, when prompted to save or open the
.CAB file, choose save.
