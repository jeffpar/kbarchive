Q193370: Encarta Virtual Globe 99: Program Hangs When You Print Article

Article: Q193370
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbinterop kbprint kbui kbimu
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to print an article from Encarta Virtual Globe 99, the program
may stop responding (hang).

CAUSE
=====

This behavior can occur if Encarta Virtual Globe 99 is configured to print all
linked documents.

RESOLUTION
==========

To resolve this issue, follow these steps:

1. Open an article you want to print from Encarta Virtual Globe 99.

2. On the Options menu, click Print.

3. Click to clear the Print All Linked Documents check box, and then click OK.

MORE INFORMATION
================

Encarta Virtual Globe 99 uses Microsoft Internet Explorer technology to display
and print textual content. One feature of this technology allows you to print
all documents linked to a single article.

If this feature is enabled, up to 180 documents may be printed when you attempt
to print a single article. Encarta Virtual Globe does not respond until all of
the linked documents are printed.
