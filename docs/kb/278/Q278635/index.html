Q278635: MSN Companion Keyboards

Article: Q278635
Product(s): The Microsoft Network
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, versions 1.0, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Each MSN Companion manufacturer will design its own keyboard. However, all MSN
Companion keyboards have some common elements.

MORE INFORMATION
================

- Each keyboard includes the standard typewriter keys (58) for text entry and
  editing.

- It does not include many of the special PC keys (e.g., function keys, numeric
  keypad).

- Each keyboard includes a set of prominent and easily accessible special
  activity keys:

   - start <A0> <A0><A0><A0><A0>Takes the user to the Start Page.

   - mail <A0><A0><A0><A0><A0><A0><A0>Takes the user to e-mail inbox

   - favorites Takes the user to a page that lets them choose, rename, and
     delete a favorite.

   - search <A0><A0><A0>Takes the user to the MSN start page.

   - people <A0><A0><A0>Takes the user to an access point for addresses, messaging, and
     other community-related functions.

   - print <A0><A0><A0><A0><A0><A0> Prints the current page.

   - back <A0><A0><A0><A0><A0><A0><A0>Navigate back thru the list of pages the user has viewed.

- The keyboard also includes a set of four arrow keys (up, down, left, right)
  for scrolling around pages.

- The keyboard also includes an on-board pointing device, e.g., a thumb or a
  navigation pad, for pointing at specific link / field.
