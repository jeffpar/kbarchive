Q61176: File: Footer Cut Off on LaserWriter Using Print Monitor

Article: Q61176
Product(s): Microsoft Fox Miscellaneous Products
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

When using the PrintMonitor under MultiFinder, the footer may shift downwards.
The footer is then cut off, due to the limitations of the LaserWriter.

This problem occurs with the following System configurations:

  System Version        PrintMonitor Version
  --------------        --------------------
  6.0.2                 1.2
  6.0.3                 1.2
  6.0.4                 1.2
  6.0.5                 1.2

This problem is not necessarily verifiable and may be system or document related.
