Q221146: INFO: Description of Visual SourceSafe Versions, Build Numbers

Article: Q221146
Product(s): Microsoft SourceSafe
Version(s): 4.0,4.0a,5.0,6.0
Operating System(s): 
Keyword(s): kbSSafe400 kbSSafe500 kbSSafe600 kbGrpDSSSafe
Last Modified: 15-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual SourceSafe for Windows, versions 4.0, 4.0a, 5.0, 6.0 
- Microsoft Visual SourceSafe 6.0c 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the versioned components that are associated with a Visual
SourceSafe installation. This article also describes the Visual SourceSafe
versions and the build numbers of their components.

MORE INFORMATION
================

The following components are associated with a Visual SourceSafe installation:

- Ssapi.dll
- Ssgui.dll
- Ssscc.dll
- Ssus.dll
- Ssus16.dll
- Ssvb.dll
- Ssexp.exe
- Ssadmin.exe
- Ssarc.exe
- Ssrestor.exe
- Ssint.exe
- Analyze.exe

These components are located in the WIN32 directory under the Visual SourceSafe
installation.

To locate the build numbers for these components, right-click the component in
the WIN32 directory, and then click Properties. In the Properties dialog box,
click the Version tab.

The following table lists the component build number and the version of Visual
SourceSafe that the component is associated with:
