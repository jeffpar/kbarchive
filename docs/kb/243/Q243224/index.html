Q243224: Asheron’s Call: Keyboard Commands Don’t Work After Using ALT+TAB

Article: Q243224
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu msgame
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call, version 1.0 
- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you press ALT+TAB to switch from Microsoft Asheron's Call to another program
or to the desktop, and then press ALT+TAB again to switch back to Asheron's
Call, keyboard commands no longer function correctly in the game.

CAUSE
=====

This behavior occurs because Asheron's Call does not recognize that you have
released the ALT key.

RESOLUTION
==========

To resolve this issue, press and release the ALT key once.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Asheron's Call,
version 1.0 and Asheron's Call: Dark Majesty.

Additional query words: 1.00 msgame ac msngz task switching keys asherons
