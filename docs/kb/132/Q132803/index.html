Q132803: Difficulty Installing System Shock

Article: Q132803
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Add/Remove Programs Wizard in Control Panel to install System
Shock (a program manufactured by Looking Glass and distributed by Origin), the
program does not install properly.

CAUSE
=====

You must run the installation program for System Shock from an MS-DOS prompt.


RESOLUTION
==========

To install System Shock, start an MS-DOS prompt. At the command prompt, type
"D:" (without the quotation marks), type "install" (without the quotation marks)
(in lowercase letters), and then press ENTER.
