Q132667: Icon Spacing Does Not Take Effect Automatically

Article: Q132667
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95 4.00
Operating System(s): 
Keyword(s): win95
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you change the vertical and horizontal icon spacing, the spacing for icons
in folders showing large icons is not adjusted unless the Auto Arrange option is
enabled for that folder.

RESOLUTION
==========

Click Line Up Icons on the View menu to arrange the icons with the new spacing.
Or, enable the Auto Arrange option. To do so, point to Arrange Icons on the View
menu, then click Auto Arrange.

NOTE: To apply a view option to all folders that have not been opened yet, press
and hold down the CTRL key after you make a change when you close a folder.
Folders that have already been opened and customized are not affected.
