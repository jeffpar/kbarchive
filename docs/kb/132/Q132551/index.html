Q132551: Dogs, 500 Nations: Can’t Print Portrait or Landscape

Article: Q132551
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0,95,98
Operating System(s): 
Keyword(s): kbenv kbmm kbprint kbimu kbPrintingkbfaq
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 500 Nations, version 1.0 
- Microsoft Dogs for Windows, version 1.0 
- the operating system: Microsoft Windows 95 
- the operating system: Microsoft Windows 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

Paper orientation is incorrect when you print with Windows 95/98, even though
the Print Orientation setting in the Options menu is correct.

For example, if you click the option to print Landscape, it still prints
Portrait.


RESOLUTION
==========

The printing settings must be configured in Microsoft Windows 95/98, and not in
the multimedia program.

For information about how to change Windows printing settings, see the following
article in the Microsoft Knowledge Base:

  Q130255 Windows 95/98: Installing or Setting a Default Printer Driver
