Q132107: Mac Encarta 1995 Err Msg: Cannot Find Files Needed to Start…

Article: Q132107
Product(s): Microsoft Home Multimedia Titles
Version(s): 1995 edition
Operating System(s): 
Keyword(s): kbbuglist
Last Modified: 02-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to start Encarta, you receive the following error message:

  Encarta cannot find the files needed to start. Either the CD is not inserted,
  or the path to the files is incorrect. To correct this, insert the CD or run
  Installer again

The error message continues even after you try the suggested corrections.

STATUS
======

Microsoft has confirmed this to be a problem in Encarta 1995. We are researching
this problem and will post new information here in the Microsoft Knowledge Base
as it becomes available.

RESOLUTION
==========

Try the following solutions:

SOLUTION 1
----------

Clean the CD.

SOLUTION 2
----------

Remove both the Encarta 95 Preferences folder. To remove the folder, do the
following:

1. Open the System Folder.

2. Open the Preferences folder.

3. Open the Encarta '95 Preferences folder and locate the Encarta '95
  Preferences. Drag this item to the trash.

4. Empty the trash

5. Restart Encarta.

SOLUTION 3
----------

1. Remove both the Encarta '94 and Encarta '95 Preferences folders.

2. Remove extensions that Encarta installs by dragging the following extensions
  to the trash:

  Apple Multimedia Tuner
  FIF Component
  QuickTime
  QuickTime Musical Instruments
  QuickTime PowerPlug
  Sound Manager

3. Remove the Encarta Control Panel by dragging the Sound icon to the trash.

4. Restart and reinstall Encarta. Verify that all extensions are installed, if
  not, Drag and Drop the extensions from the Encarta CD.
