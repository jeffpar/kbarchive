Q301521: MSN Explorer Hangs When Outlook XP Requests Data From The Server

Article: Q301521
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 14-MAR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
- The Microsoft Network Version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

You try to navigate to a Web page in MSN Explorer while Microsoft Outlook XP is
requesting data from a server. This may be when the program opens a public
folder. MSN Explorer will not display the Web page until Outlook has finished.

CAUSE
=====

Known issue.

RESOLUTION
==========

You must wait until Outlook has finished before you can navigate to new pages in
MSN Explorer.
