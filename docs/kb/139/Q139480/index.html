Q139480: ADT95: ADT for Windows 95 Product Information Available

Article: Q139480
Product(s): Microsoft Access Distribution Kit
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbfile kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Access for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Adtdata.exe file contains a six-page document about the Microsoft Access
Developer's Toolkit (ADT) for Windows 95 version 7.0. The document discusses the
many ADT features, such as the royalty-free run-time license, Setup Wizard, OLE
controls, Replication Manager, Office Compatible Technical Kit, and more.

The following file is available for download from the Microsoft Download Center:

  Download Adtdata.exe now
  (http://download.microsoft.com/download/access95/info3/1/WIN98/EN-US/Adtdata.exe)

The Adtdata.exe file contains the following file:

  Adtdata.doc     A document about the Microsoft Access Developer's 

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

Additional query words: Adtdata exe doc
