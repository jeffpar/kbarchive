Q255531: FS2000: How to Add and Remove Interior Cockpit Views

Article: Q255531
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbdisplay kbsetup fsim kbimu msgame
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to add and remove the interior cockpit views in
Microsoft Flight Simulator 2000.

MORE INFORMATION
================

By default, the interior cockpit views for each aircraft are stored in the
following location

  <drive>:\Program Files\Microsoft
  Games\FS2000\Aircraft\<aircraft>\Panel

where <drive> is the drive letter for the hard disk on which Flight
Simulator 2000 is installed, and <aircraft> is the the name of the folder
for an installed aircraft.

For example, the interior cockpit views for the Boeing 737-400 are stored by
default in the following folder:

  C:\Program Files\Microsoft Games\FS2000\AIRCRAFT\B737_400\PANEL

The interior cockpit views are bitmap (.bmp) files, and are named:

- Forward_left.bmp

- Forward_right.bmp

- Left.bmp

- Rear.bmp

- Rear_left.bmp

- Rear_right.bmp

- Right.bmp

- Up.bmp

To delete these views, open the Panels folder for the appropriate airplane and
then delete the files in this list.

For additional information about how to delete files, click the article number
below to view the article in the Microsoft Knowledge Base:

  Q133340 Windows 95/98: How to Delete a File

To add or restore these views, uninstall Flight Simulator 2000, and then
reinstall Flight Simulator 2000 using either the Typical or Custom installation
option.

To remove Flight Simulator 2000:

1. Start Flight Simulator 2000.

2. On the Flight Simulator 2000 startup screen, click Uninstall.

3. Follow the instructions on the screen to remove Flight Simulator 2000.

To reinstall Flight Simulator 2000:

1. Insert the Flight Simulator 2000 Disk 1 CD-ROM into the CD-ROM drive.

2. Follow the instructions on the screen to install Flight Simulator 2000.
  Select the Typical or Custom installation option to install the interior
  cockpit view for each aircraft.

  NOTE: If you select the Custom installation option, click to select the
  "Install interior views" check box on the Custom Install Options screen.
