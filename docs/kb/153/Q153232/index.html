Q153232: Driver for Use with Wangtek 5100 Tape Drive

Article: Q153232
Product(s): Microsoft Windows NT
Version(s): 3.50 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 3.5, 3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

Windows NT does not provide direct support for the Wangtek 5100 tape drive but
it may be possible to use the device with drivers for other Wangtek units.

MORE INFORMATION
================

The driver to use is the Wangtek 525, 250, 9500 driver. This can be added from
Windows NT Setup add/remove tape devices.

For additional information on the use of the Wangtek 5100 with Windows NT, it is
necessary to call Wangtek technical support at (800) 992-9916.


The Wangtek 5100 tape drive is manufactured by a vendor independent of Microsoft;
we make no warranty, implied or otherwise, regarding this product's performance
or reliability.
