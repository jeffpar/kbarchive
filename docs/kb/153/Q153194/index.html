Q153194: WD97: Extended Characters Corrupt After Editing MacWord Document

Article: Q153194
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbenable kbinterop kbdta word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a document in Word 97 after editing the document in Word 6.x for
the Macintosh, extended characters may become corrupt.

CAUSE
=====

There is a problem with the conversion of the Macintosh ANSI extended character
set.


RESOLUTION
==========

To correct this problem, obtain Microsoft Word 97 Service Release 2 (SR-2).

STATUS
======

Microsoft has confirmed this to be a problem in Word 97 for Windows. This
problem was corrected in Microsoft Office 97 Service Release 2 (SR-2).

MORE INFORMATION
================

For additional information about SR-2, please see the following article in the
Microsoft Knowledge Base:

  Q151261 OFF97: How to Obtain and Install MS Office 97 SR-2
