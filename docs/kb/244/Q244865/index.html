Q244865: EST2000: How to Import Contacts From Microsoft Outlook

Article: Q244865
Product(s): Microsoft Automap
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbinterop kbtool kbimu
Last Modified: 27-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Expedia Streets and Trips 2000 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to import Microsoft Outlook contacts into Microsoft
Expedia Streets and Trips 2000.

MORE INFORMATION
================

To import Microsoft Outlook contacts into Microsoft Expedia Streets and Trips
2000:

1. Start Outlook.

2. On the File menu, click "Import and Export."

3. Click "Export to a File," and then click Next.

4. Click "Comma Separated Values (Windows)," and then click Next.

  If the Import/Export engine is not installed on your computer, you receive the
  following message:

  Microsoft cannot start the Import/Export engine.
  This feature is not currently installed. Would you like to install it now?

  Click Yes to install the Import/Export engine.

5. Click Contacts, and then click Next.

6. Type the path and file name of the file you want to export in the "Save
  exported file as" box.

7. Click Next, and then click Finish.

8. Quit Outlook.

9. Start Streets and Trips 2000.

10. On the File menu, click Import Data.

11. In the "Files of Type" box, click "Text Files (*.txt, *.csv, *.tab, *.asc)."

12. Click the file you created in step 6, and then click Open.

13. In the Data Import wizard, click Comma, and then click Next.

14. Click to select the "First row contains column headings" check box, and then
  click Finish.
