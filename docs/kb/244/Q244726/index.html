Q244726: Encarta 2000: Unable to Install Program

Article: Q244726
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0
Operating System(s): 
Keyword(s): kbfile kbsetup kbimu
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Africana 2000 
- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Reference Suite 2000 
- Microsoft Encarta Interactive World Atlas 2000 
- Microsoft Encarta World English Dictionary, version 1.0 
- Microsoft Works Suite 2000 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain an update file that corrects problems you
may experience when you install the U.S. version of any of the programs that are
listed at the beginning of this article.

During installation, the following error message appears even though the correct
CD-ROM is in the drive:

For Windows 95, Windows 98, and Windows Millennium Edition (Me):

  Cannot locate the CD-ROM. Please insert the correct CD-ROM, select OK and
  restart application.

For Windows NT, Windows 2000, and Windowx XP:

  STOP 0x00000A ... IRQ_NOT_LESS_OR_EQUAL ... hal.dll

MORE INFORMATION
================

Microsoft has released an update file that corrects problems you may experience
when you install the U.S. version of any of the programs listed at the beginning
of this article.

NOTE: Do not use this patch on Encarta 2001 products. This patch is designed for
Encarta 2000 products only.

If you experience problems when you install your Encarta program, download and
run the Crow.exe update file, and then install the Encarta program.


The following file is available for download from the Microsoft Download Center:

  Crow.exe
  (http://download.microsoft.com/download/encarta2000ref/enc2k/1.0/WIN98/EN-US/crow.exe)

Release Date: Oct-29-1999

For additional information about how to download Microsoft Support files, click
the following article number to view the article in the Microsoft Knowledge
Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft scanned this file for viruses. Microsoft used the most current
virus-detection software that was available on the date that the file was
posted. The file is stored on secure servers that prevent any unauthorized
changes to the file.

After you download the file:

1. Copy the Crow.exe file to a blank, formatted floppy disk.

2. Quit all programs that are running on your computer.

3. Insert the Encarta Installation and Resources CD-ROM into the CD-ROM or
  DVD-ROM drive.

  If Setup does not start automatically, double-click the My Computer icon on
  the desktop, and then double-click <drive> where <drive> is the
  drive letter of the CD-ROM or DVD-ROM drive.

4. When Setup starts, insert the floppy disk into the floppy disk drive.

5. Click Start, and then click Run.

6. In the Open box, type "a:" (without the quotation marks), and then click OK.

7. Double click the Crow.exe file.

8. Click OK.

9. When you receive the following message, click OK.

  This patch is provided by Microsoft Product Support Services and may not be
  distributed by non-Microsoft personnel.

10. Follow the instructions on the screen to install your Encarta program.

IMPORTANT: Do not install Microsoft Natural Language Search from the list of
components (Microsoft Encarta Reference Suite 2000 and Microsoft Encarta World
English Dictionary only).

If you install Natural Language Search, Microsoft Encarta World Atlas 2000,
Encarta World English Dictionary, and Microsoft Research Organizer may not be
installed.

If you want to install Natural Language Search, install the other components
included with Encarta Reference Suite 2000 first, and then install Natural
Language Search. To do this:

1. Insert the Encarta 2000 Installation and Resources CD-ROM into the CD-ROM or
  DVD-ROM drive.

  NOTE: You may need to run the Crow.exe file before you install Natural
  Language Search.

2. Click Encarta Extras.

3. Click Microsoft Natural Language.

4. Follow the instructions on the screen to install Natural Language Search.

Microsoft has determined that this issue was caused by faulty disks produced at a
third party manufacturing site. If you wish replacement disks, please contact
the Microsoft Order Desk at 800-360-7561.

If you are still unable to install your Encarta program, you may need to
troubleshoot your CD-ROM drive. For additional information about how to
troubleshoot your CD-ROM drive, click the article number below to view the
article in the Microsoft Knowledge Base:

  Q218617 How to Troubleshoot CD-ROM Read Issues
