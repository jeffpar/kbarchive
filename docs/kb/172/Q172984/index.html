Q172984: TI 850/855 Printer Will Not Print Margins Less Than 1 Inch

Article: Q172984
Product(s): Microsoft Windows NT
Version(s): WinNT:4.0
Operating System(s): 
Keyword(s): kbprint
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to print a document with margins less than one inch and use the
Texas Instruments TI 850/855 driver, the document will not print. The document
will spool to the queue and Windows NT will print an informational message in
the event log that the document has printed.

The printer does not produce any output, even though the system seems to be
functioning properly.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
