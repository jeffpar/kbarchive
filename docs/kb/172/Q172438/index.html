Q172438: How to View Connection Status Information on MSN

Article: Q172438
Product(s): The Microsoft Network
Version(s): WINDOWS:2.5
Operating System(s): 
Keyword(s): kbmsn
Last Modified: 12-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

While you are connected to MSN, The Microsoft Network, you may want to view
connection status information such as call duration, connection speed, or the
number of bytes sent and received. This article describes how to view this
information.

MORE INFORMATION
================

To view connection status information while you are connected to MSN, use one of
the following methods:

Method 1
--------

Click the MSN Quick View icon on the taskbar, point to MSN Options, and then
click Connection Settings.

Method 2
--------

Right-click the MSN (or The Microsoft Network) icon on the desktop, and then
click Connection Settings.
