Q85166: ATI 8514 Ultra Hangs Windows 3.1 in 800 x 600 Mode

Article: Q85166
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 30-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

If you want to use 800 x 600 video resolution drivers under Windows version 3.1
with the ATI 8514 Ultra video card, you must have ROM version 1330111330 or
later.

MORE INFORMATION
================

The ROM version is displayed in the startup banner when you first start your
computer. For ROM upgrade details, please contact ATI customer support.

The ATI products included here are manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding these products'
performance or reliability.

For additional information, please see the README.TXT file that comes on the ATI
drivers disk, provided with the video card.
