Q184458: Outwars: Where Does Outwars Store Saved Games?

Article: Q184458
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbui kbimu msgamekbfaq
Last Modified: 23-APR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outwars, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Outwars stores saved games in the Outwars.sav file located inside the
following folder

  <drive>:\Program Files\Microsoft Games\Outwars

where <drive> is the letter of the hard disk on which Outwars is installed.

MORE INFORMATION
================

You can only save a game when you complete a Solo Campaign mission. To save a
game when you complete a Solo Campaign mission, follow these steps:

1. On the debriefing screen, click Save.

2. Click the game slot in which you want to save the game. If you click a game
  slot that contains a previously saved game, you are prompted to confirm your
  selection. Click OK to replace the previously saved game, or click Cancel to
  select another game slot.

3. Type a name for the game you are saving, and then click Save.

Additional query words: 1.00 msgame outwar out wars
