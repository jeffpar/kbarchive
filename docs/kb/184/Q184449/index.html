SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Microsoft Sourcebook for the Help Desk,
Second Edition, ISBN 1-57231-582-2.

The following topics are covered:

- CD-ROM: Instructions For Installing Tools

- Page 81: Table Is Incorrect

MORE INFORMATION
================

CD-ROM: Instructions For Installing Tools
-----------------------------------------

The valuable tools on the CD-ROM mentioned on the back cover are all installed by
the Microsoft TechNet sampler. To install these tools and forms, follow these
steps:

1. Insert the CD-ROM into the drive.

2. If the Setup program does not immediately appear, click the Start button,
  point to Programs and click Windows Explorer. Click the CD-ROM drive to open
  it and then double-click Autorun.exe.

3. Click the Microsoft TechNet button to install the contents of the companion
  CD-ROM.

To access any of the tools, click the Start button, point to Programs and click
Microsoft TechNet.


Page 81: Table Is Incorrect
---------------------------

In the text on pages 80 and 81, the author suggests creating a table similar to
the one shown on the bottom of page 81. When you create the table using the
suggested formula, it will not match the table shown on the bottom of page 81.
The numbers are not correct when the "number of incidents" exceeds 120. The
table should have the following contents:
