Q184355: MSN Err Msg: Iexplore Caused an Invalid Page Fault in Msjava.dll

Article: Q184355
Product(s): The Microsoft Network
Version(s): WINDOWS:2.0,3.01,3.02
Operating System(s): 
Keyword(s): kbenv kberrmsg kbmsn
Last Modified: 07-JAN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 2.0 
- Microsoft Internet Explorer versions 3.01, 3.02 for Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open MSN, The Microsoft Network, you may receive the following error
message:

  Iexplore caused an invalid page fault in module Msjava.dll.

CAUSE
=====

This behavior can occur if some Microsoft Internet Explorer files are missing or
damaged.

RESOLUTION
==========

To resolve this issue, remove and reinstall Internet Explorer. To do so, follow
these steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Add/Remove Programs.

3. On the Install/Uninstall tab, click Internet Explorer 3.x, and then click
  Add/Remove.

4. When you are prompted to confirm the removal of Internet Explorer or one of
  its components, click Yes.

5. Restart the computer.

6. Insert the MSN 2.0 CD-ROM into the CD-ROM drive. If the Auto Insert
  Notification option is turned on, close the Welcome To MSN window that opens.

7. Double-click My Computer.

8. Right-click the MSN (CD-ROM drive) icon, and then click Explore.

9. Double-click the Msnsetup folder.

10. Double-click the Compnts folder.

11. Double-click the Msie301.exe file.

12. Follow the instructions on the screen. When you are prompted to restart the
  computer, click Yes.

13. Download and install Internet Explorer 3.02 with Authenticode 2.0 for
  Windows 95 from the following Microsoft Web site:

  http://www.microsoft.com/ie/download

Additional query words: 3.01 3.02 msnet msnetwork microsoft-net m.s.n.
