Q126657: Mac Wkst: Err Msg: Unable to Retrieve Details for That Item

Article: Q126657
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, version 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use version 3.1 of Microsoft Mail for AppleTalk Networks and address
users in the address list provided by Alisa Systems AlisaMail and attempt to get
details on these users, you will get the following error message:

  Unable to retrieve details for that item

MORE INFORMATION
================

AlisaMail provides connectivity for MSMAIL and VMS mail systems. Alisa appears
on the network as another Mail server. It is not a gateway. The purpose of the
AlisaMail program is to propagate addresses from other mail programs on various
platforms (usually a VMS or UNIX system) using a Directory Synchronization type
program. Because of the various platforms that they interact with, AlisaMail
does not provide details to the mail programs that they are providing addresses
to (for example, MS Mail).

For more information, contact Alisa Systems at (800) 992-5472.

The Alisa Systems product discussed above is manufactured by vendors independent
of Microsoft; we make no warranty, implied or otherwise, regarding these
products' performance or reliability.
