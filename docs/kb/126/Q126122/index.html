Q126122: WIN EXCEL CDK: Corrections and Comments

Article: Q126122
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 20-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Windows Excel 5 Courseware Development Kit ISBN 1-55615-719-3 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains information on known errors, corrections, and comments
relating to the Microsoft Press book "Windows Excel 5 Courseware Development
Kit."

The following topic is covered:

- Grammatical Error in Welcome Document Decompression File List

MORE INFORMATION
================

In addition to a description of the book's problems, this document might also
include sections labeled "Correction" and "Comments." Please note that the
"Correction" section is worded for correcting the book and does not necessarily
address the problem introduced by the book error. The "Comments" section
contains specific information for working around problems.

Grammatical Error in Welcome Document Decompression File List
-------------------------------------------------------------

The Excel 5 Courseware Development Kit page that describes how to decompress
files contains an error when referring to CDK_PART CMP.

Correction:

All text "CDK_PART CMP" in table should be "CDK_PART.CMP." (Note the period just
prior to the .CMP extension is missing from the current syntax.)

Comments:

The Courseware Development Kit (CDK) is a special version of the corresponding
Microsoft Press book "Microsoft Excel 5 Step by Step." The CDK provides the
entire contents of this Step by Step book in electronic format, along with
limited rights to alter and duplicate this material for training purposes. For
more information about the CDK, contact Microsoft Press.

Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later release of this CDK, it may already
contain most or all of the above corrections.
