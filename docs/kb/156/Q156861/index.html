MORE INFORMATION
================

You can not configure this word list in FrontPage versions 1.1 and 1.0.

If you have installed Microsoft Index Server on a Microsoft Internet Information
Server, FrontPage 97 will use the Microsoft Index Server for all its search
capabilities on the Internet Information Server only. The Microsoft Index Server
has its own configuration parameters. For more information about Microsoft Index
Server, see the following Web site:

  http://www.microsoft.com/ntserver/search/index2/ixinstal.asp

Additional query words: search dictionary omit omitted bot WAIS stoplist stop-list stop list
