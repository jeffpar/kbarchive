Q156490: Close Combat: Head-to-Head Play Alert Message

Article: Q156490
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbhowtokbfaq
Last Modified: 08-NOV-2001

1.0
WINDOWS
kbusage kbsetup kbnetwork

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Close Combat for Windows 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you choose to play a Head To Head game in Microsoft Close Combat, you get
the following Alert message:

  Close Combat requires a valid TCP/IP networking connection for head to head
  play. For more information on how to set up a two-player game, refer to the
  Network Readme file.

RESOLUTION
==========

Your computer must be configured to use an IP address to play a head-to- head
game in Close Combat. Please refer to the appropriate Knowledge Base article
listed below for information to ensure your computer is properly configured to
play a head-to-head game.

For additional information on modem to modem head-to-head play, please see the
following article in the Microsoft Knowledge Base:

  Q154729 Close Combat: Play Between Modems Using Dial-Up Networking

For additional information on modem to network head-to-head play, please see the
following article in the Microsoft Knowledge Base:

  Q154796 Close Combat: Playing Modem to Network

For additional information on head-to-head play over a network, please see the
following article in the Microsoft Knowledge Base:

  Q154728 Close Combat: Playing Over a Network

For additional information on head-to-head play over the Internet, please see the
following article in the Microsoft Knowledge Base:

  Q154727 Close Combat: Playing Over the Internet

MORE INFORMATION
================

Head-to-head play is also possible between a computer running Windows and a
Macintosh.

For additional information on head-to-head play using a Macintosh, please see the
following article in the Microsoft Knowledge Base:

  Q156597 Head-to-Head Play on Directly Connected Computers
