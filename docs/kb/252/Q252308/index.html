Q252308: FS2000: Tutorials 12 and 13 Are Missing from Available Flights

Article: Q252308
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbui fsim kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 2000 Professional Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to select a tutorial flight in Microsoft Flight Simulator 2000,
Tutorials 12 and 13 are missing from the list.

CAUSE
=====

This behavior occurs because Tutorials 12 and 13 are not available in Flight
Simulator 2000.

MORE INFORMATION
================

To run a tutorial in Flight Simulator 2000:

1. Start Flight Simulator 2000.

2. Click Fly Now!

3. On the Flights menu, click Select Flight.

4. Scroll down the list of available flights to the list of tutorial flights,
  and then click the tutorial flight that you want.

  Note that in the list of available flights, Tutorials 12 and 13 are missing.

Additional query words: msgame flightsim fltsim fs2k lessons unavailable
