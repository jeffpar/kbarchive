Q155613: AMD Nx586 Processor Incompatible with Windows NT 4.0

Article: Q155613
Product(s): Microsoft Windows NT
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbhw kbsetup kbHardware
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you are installing Windows NT 4.0 on a computer using the Advanced Micro
Devices (AMD) Nx586 processor, the following message is displayed:

  Windows NT requires a 486 or later processor

MORE INFORMATION
================

The AMD Nx586 is based on the 80386 processor instruction set. This processor
does not support the minimum 486 instruction set which is required to use
Windows NT 4.0.
