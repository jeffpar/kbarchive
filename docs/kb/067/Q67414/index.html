Q67414: PIF Files Created by Windows 3.0 Setup from A to F

Article: Q67414
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 11-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the default parameters for program information files (PIFs)
created by the Microsoft Windows version 3.0 Setup program, in alphabetical
order from A to F. These defaults may be modified by loading the PIF file into
the Windows 3.0 PIF Editor, PIFEDIT.EXE.

MORE INFORMATION
================
