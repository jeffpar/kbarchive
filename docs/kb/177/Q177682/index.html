Q177682: Images Not Displayed Correctly After Installing Service Pack 3

Article: Q177682
Product(s): Microsoft Windows NT
Version(s): 4.00
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 10-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 4.0 
- Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install the Windows NT 4.0 Service Pack 3 on a computer with a Matrox
Millennium video adapter, bitmaps may be drawn outside of your program's client
area, and the image may be upside down and mirrored.

CAUSE
=====

This behavior can occur when you are using the Matrox Millennium video adapter
driver included with Windows NT.


RESOLUTION
==========

To resolve this issue, obtain and install the latest driver for you video
adapter from Matrox. To obtain this driver, contact Matrox or visit the
following Matrox Web site:

http://www.matrox.com

For information about how to contact Matrox, see the following article in the
Microsoft Knowledge Base:

ARTICLE-ID: Q60781
TITLE : Hardware and Software Third-Party Vendor Contact List, L-P


STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0.

================

you find the technical support you need. This contact information is subject to
