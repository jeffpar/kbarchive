Q176145: XADM: Admins Can Edit DL Members Even When Hidden from AB

Article: Q176145
Product(s): Microsoft Exchange
Version(s): WinNT:4.0,5.0,5.5
Operating System(s): 
Keyword(s): kbusage
Last Modified: 14-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
-------------------------------------------------------------------------------


SUMMARY
=======



MORE INFORMATION
================

Users who are allowed to edit the membership can use the Microsoft Exchange
Client or Exchange Administrator program to edit the membership.

SYMPTOMS
========

If you create a distribution list (DL) on an Exchange Server computer and choose
the Hide Membership from Address Book option or the Hide from Address Book
option on the Advanced tab, anyone who is allowed to edit the membership
(including the owner and administrators) will still be able to view the DL
membership.

CAUSE
=====

This is by design.
