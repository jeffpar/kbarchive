Q186113: Internet Explorer Input Method Editor 4.0 for Japanese/Korean

Article: Q186113
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): 2.0,3.x,4.0,4.x,5,5.01,5.5,95
Operating System(s): 
Keyword(s): 
Last Modified: 13-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
- Microsoft Internet Explorer versions 2.0, 3.0, 3.01, 3.02, 3.02a, 3.03, 3.03 Service Pack 1, 4.0, 4.01, 4.01 SP1, 4.01 SP2, 4.5, 5, 5.01, 5.5 for Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

Internet Explorer 4.0 users can input Japanese and Korean text into Web forms
and into Outlook Express 4.0 e-mail messages on any language version of Windows
95 and NT 4.0 using Internet Explorer Input Method Editor (IME) 4.0 for Japanese
and Korean.

MORE INFORMATION
================

Current Microsoft Language Packs allow any language version of Internet Explorer
4.0 to display Japanese or Korean characters. With Internet Explorer Japanese or
Korean IME 4.0, you can write those characters, too, without a special keyboard
or a different language browser. Internet Explorer IME 4.0 makes Internet
Explorer 4.0 the only major browser that allows you to read and write East Asian
characters.

After you install the Internet Explorer Japanese IME 4.0 or Internet Explorer
Korean IME 4.0, or both, Windows 95 or Windows NT 4.0 displays the Internet
Explorer IME 4.0 icon on the task bar whenever Internet Explorer 4.0 is active,
or whenever the user clicks in the body of a message in Outlook Express 4.0.

To write in Japanese or Korean characters, the user just clicks the IME icon,
selects Japanese or Korean, and starts typing.

Users can type different languages on the same page. Internet Explorer IME 4.0
also comes with multi-lingual Help.

To download the IME, please see the following Microsoft World-Wide Web site:

  http://www.microsoft.com/msdownload/iebuild/ime5_win32/en/ime5_win32.htm
