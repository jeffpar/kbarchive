Q186211: Cannot Connect to MSN When More Than One Modem Is Installed

Article: Q186211
Product(s): The Microsoft Network
Version(s): WINDOWS:2.5,2.51,2.52,2.6,95,98,98 Second Edition
Operating System(s): 
Keyword(s): kbhw kbtool kbmsn kbHardware
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network versions 2.5, 2.51, 2.52, 2.6 
- the operating system: Microsoft Windows 95 
- the operating system: Microsoft Windows 98 
- the operating system: Microsoft Windows 98 Second Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When more than one modem is installed in your computer, you may be unable to
connect to MSN, The Microsoft Network.

CAUSE
=====

This behaviour can occur if Connection Manager is configured to use a different
modem to connect to MSN than the modem you selected during the MSN Setup.

RESOLUTION
==========

To resolve this issue, configure Connection Manager to use the modem you
selected during MSN Setup to connect to MSN. To do so, follow these steps:

1. Double-click the MSN icon on the desktop.

2. On the MSN Sign-In screen, click Settings, and then in the Connect Using box,
  click the modem you selected during MSN Setup to connect to MSN.

3. Click OK, and then click Connect.

Additional query words: msnet msnetwork microsoft-net m.s.n. 2.50 2.60
