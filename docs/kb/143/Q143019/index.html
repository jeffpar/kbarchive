Q143019: PPT7: Contents of PowerPoint Viewer (32-Bit) Disks

Article: Q143019
Product(s): Microsoft PowerPoint for Windows
Version(s): 7.0
Operating System(s): 
Keyword(s): kbdta kbdiskdir kbdir
Last Modified: 31-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint Viewer for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The PowerPoint Viewer (32-bit) for Windows 95 includes the following files on
two disks.
