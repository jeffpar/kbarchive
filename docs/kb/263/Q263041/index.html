Q263041: XWEB: Err. Mssg.: "Internet Explorer Cannot Download…"

Article: Q263041
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 Service Packs 1, 2, 3 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use Microsoft Internet Explorer to gain access to Outlook Web Access
(OWA), if you click a link for an attachment on a Web page, you may receive the
following error message

  Internet Explorer cannot download from the Internet site
  <FQDN_server_name_or_file_name> the downloaded file is not available.

  This could be due to your security or language settings or because the server
  was unable to retrieve the requested file.

where <FQDN_server_name_or_file_name> is the fully qualified domain name
(FQDN) of the server or the file name.

WORKAROUND
==========

To work around this issue, upgrade your browser to Internet Explorer version 5.5
or later. You can obtain more information about the latest version of Internet
Explorer and download the latest version of Internet Explorer from the following
Microsoft Web site:

  http://www.microsoft.com/windows/ie

MORE INFORMATION
================

This issue may also occur if you use Microsoft Money and you attempt to download
a file from your financial institution's Web site. For additional information
about this error message when you use Money, click the article number below to
view the article in the Microsoft Knowledge Base:

  Q220135 Money Err Msg: Internet Explorer Cannot Download From the Internet
  Site...
