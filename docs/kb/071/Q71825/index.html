Q71825: MS-DOS 5 PACKING.LST (5.25-Inch Disk Set)

Article: Q71825
Product(s): Microsoft Disk Operating System
Version(s): 5.0
Operating System(s): 
Keyword(s): 
Last Modified: 31-JAN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is the packing list from Disk 3 of the Microsoft MS-DOS 5
5.25-inch disk set. The 360K Distribution Disk Layout lists the compressed files
on the disks shipped with the product. The 360K Destination Disk Layout
describes the working disks created when installing on floppy disks.
