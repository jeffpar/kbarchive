Q123947: PRB: Unable to Change Field Type in MODIFY STRUCTURE

Article: Q123947
Product(s): Microsoft Fox Miscellaneous Products
Version(s): MACINTOSH:2.01
Operating System(s): 
Keyword(s): 
Last Modified: 23-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FoxBASE+ for Macintosh, version 2.01 
-------------------------------------------------------------------------------

WARNING: The information in this article has not been confirmed or tested
by Microsoft. Some or all of the information in this article has been taken
from unconfirmed customer reports. ANY USE BY YOU OF THE INFORMATION
PROVIDED IN THIS ARTICLE IS AT YOUR OWN RISK. Microsoft provides this
information "as is" without warranty of any kind, either expressed or
implied, including but not limited to the implied warranties of
merchantability and/or fitness for a particular purpose.

SYMPTOMS
========

The field type cannot be changed in the MODIFY STRUCTURE dialog for a database
what was opened exclusively.

CAUSE
=====

Adobe Type Reunion Manager is installed.

NOTE: Adobe Type Reunion Manager is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.

WORKAROUND
==========

Here are two workarounds:

- Turn off all extensions and then modify the structure.

-or-

- Hold down the shift key while clicking the Field Type popup in the MODIFY
  STRUCTURE dialog to change the Field Type.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
