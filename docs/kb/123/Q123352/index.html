Q123352: BUG: Uninstalling FoxPro Leaves Some Files Behind

Article: Q123352
Product(s): Microsoft FoxPro
Version(s): MACINTOSH:2.6a,3.0b
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 11-FEB-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Macintosh, version 3.0b 
- Microsoft FoxPro for Macintosh, version 2.6a 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use Installation Disk 1 to uninstall a complete copy of FoxPro for
Macintosh, some FoxPro files will be left on the hard drive. The "Microsoft
FoxPro Application program" will remain in the FoxPro folder, and the "Microsoft
FoxPro Setup Data" file will remain in the Setup folder.

RESOLUTION
==========

Delete the files manually by using the mouse to drag the file icon to the trash
can icon.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

Additional query words: vFoxMac FoxMac buglist2.60a
