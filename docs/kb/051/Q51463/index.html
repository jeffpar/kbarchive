Q51463: PPT: How to Copy Slides Between PowerPoint Presentations

Article: Q51463
Product(s): Microsoft PowerPoint for Windows
Version(s): MACINTOSH:2.0,2.01,3.0,4.0; WINDOWS:3.0,4.0,7.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 15-APR-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
- Microsoft PowerPoint for Windows, versions 3.0, 4.0 
- Microsoft PowerPoint for Macintosh, versions 2.0, 2.01, 3.0, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to copy multiple slides from one Microsoft PowerPoint
presentation to another.

MORE INFORMATION
================

To copy multiple slides from one presentation to another, follow these steps:

1. On the View menu, click Slide Sorter.

2. Select the slides to be copied. To easily select multiple slides, hold down
  the SHIFT key while clicking the slides that you want.

  NOTE: You can select all the slides at once by clicking Select All on the Edit
  menu.

3. On the Edit menu, click Copy.

4. Open the presentation in which the slides are to be pasted.

5. On the View menu, click Slide Sorter.

6. Position the insertion point where the slides are to be pasted.

7. On the Edit menu, click Paste.

Additional query words: copy between presentations slides
