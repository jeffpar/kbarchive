Q200065: XFOR: Replication Between French Exchange Server & MacMail Fails

Article: Q200065
Product(s): Microsoft Exchange
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): exc4
Last Modified: 13-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you trying to install the DER on a French AppleTalk Server with French
MacMail, the install screen does not allow input into any field except the Send
directory Hour field. Because of this, directory replication cannot be set up
between MacMail and Exchange Server.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in the latest Microsoft Exchange Server
U.S. Service Pack. For information on obtaining the Service Pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

  S E R V P A C K
