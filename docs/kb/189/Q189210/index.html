Q189210: Running Office 97 Updated Edition Samples Folder Missing

Article: Q189210
Product(s): Microsoft Press
Version(s): 
Operating System(s): 
Keyword(s): kbdocerr
Last Modified: 20-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Running Microsoft Office 97, Updated Edition ISBN 1-57231-889-9 
-------------------------------------------------------------------------------

SUMMARY
=======

Page xxxvii states that the companion CD-ROM contains a \Samples folder with
sample files used in the book. However, the CD-ROM may not contain a \Samples
folder. The first and second printings of "Running Microsoft Office 97, Updated
Edition" did not include this \Samples folder.

MORE INFORMATION
================

Microsoft Press has created a self-extracting executable file containing all of
the "Running Microsoft Office 97, Updated Edition" sample files.

The following file is available for download from the Microsoft Download Center:

  RunOff97.exe
  (http://download.microsoft.com/download/office97std/mspress/2.0/WIN98Me/EN-US/RunOff97.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.


To install these files, follow these steps:

1. Download RunOff97.exe into a temporary folder.

2. Double-click RunOff97.exe to extract the necessary files.

3. If you are asked to confirm the replacement of files, press Y to confirm the
  file replacement.

4. Run Setup.exe to install the files.

5. Delete RunOff97.exe.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
