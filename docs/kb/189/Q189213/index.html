Q189213: Windows NT Tech Support Training Deluxe Edition Corrections

Article: Q189213
Product(s): Microsoft Press

Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- MSPRESS Microsoft Windows NT 4.0 Technical Support Training, Deluxe Multimedia Edition ISBN 1-57231-833-3 
-------------------------------------------------------------------------------

SUMMARY
=======

This article contains information on known errors, corrections, and comments
relating to the Microsoft Press book "Microsoft Windows NT Technical Support
Training, Deluxe Multimedia Edition," ISBN 1-57231-833-3.

Further issues applying both to the Deluxe Multimedia Edition and to the regular
edition of the Microsoft Press book "Microsoft Windows NT Technical Support
Training" are posted in a separate article. For additional information, please
see the following article in the Microsoft Knowledge Base:

  Q178439 MS Windows NT Tech Support Training Comments and Corrections

Contents:

- Back Cover: Systems Requirements Incorrect

MORE INFORMATION
================

Back Cover: Systems Requirements Incorrect
------------------------------------------

The system requirements for this book do not state that two computers are
required for the practice exercises. However, you will need two computers for
the networking exercises in this book.

To complete the lessons, you must have two computers, the first capable of
running Microsoft Windows NT Server 4.0 and the second capable of running both
Microsoft Windows NT Server 4.0 and Microsoft Windows NT Workstation 4.0.

Both computers must have the following minimum configuration:

- 486/33 or higher Intel-based processor

- 16 MB of RAM (32 MB recommended)

- A minimum of 450 MB of available hard disk space on the first computer

- A minimum of 350 MB of available hard disk space on the second computer

- SVGA display adapter and monitor capable of 256 colors

- Microsoft Mouse or compatible pointing device

- Network adapter card and related cables

- One 3.5" high-density floppy disk drive

- CD-ROM drive

- Sound card and speakers (optional)

- Serial port (optional - for the RAS practice)

- Null modem cable for testing RAS (optional)

For those who do not have a null-modem cable, a simulation for RAS is included
with the course.

NOTE: Both computers must have existing Microsoft operating systems such as
MS-DOS, Windows 3.x, or Windows NT 3.x. We recommend that one computer be able
to run multimedia programs so that you can view the video included with the
course.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of the book, it may already
contain the above corrections.
