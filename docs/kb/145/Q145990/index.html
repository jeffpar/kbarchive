Q145990: Windows 95 Update Information Tool (Qfecheck.exe)

Article: Q145990
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): kbfile kbtool win95 kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The Update Information Tool (Qfecheck.exe) is added when you install the Windows
95 Service Pack 1 Update and has the following features:

- System administrators and Microsoft Product Support Service Engineers can use
  this tool to find out which Windows 95 Service Pack 1 components have been
  installed.

- You can use this tool to verify that the versions of the components installed
  match the versions listed in your computer's registry.

For additional information about the Windows 95 Service Pack 1 Update, please see
the following articles in the Microsoft Knowledge Base:

  Q142794 Availability of Microsoft Windows 95 Service Pack 1

  Q143003 Description of Microsoft Windows 95 Service Pack 1 Updates

MORE INFORMATION
================

To run the Update Information Tool, click the Start button, point to Programs,
point to Accessories, point to System Tools, and then click Update Information
Tool.

For information about the Registered Updates and Updated Files Found tabs, click
Help in the Update Information Tool.

Installing the Update Information Tool
--------------------------------------

To install the Update Information Tool, follow these steps:

1. Download the QFECHKUP.EXE file from the Microsoft Download Center to an empty
  folder.

2. Using Windows Explorer or My Computer, double-click the QFECHKUP.EXE file you
  downloaded in step 1.

3. Follow the instructions on the screen.

The following file is available for download from the Microsoft Download Center:

  Download qfechkup.exe now
  (http://download.microsoft.com/download/win95upg/update1/1/w95/EN-US/qfechkup.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
