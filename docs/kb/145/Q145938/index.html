Q145938: Cannot Print Custom Page Sizes with Okidata ML-320 Printer

Article: Q145938
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 95
Operating System(s): 
Keyword(s): win95
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print to an Okidata ML-320 printer (using the Oki9.drv driver) in
Windows 95, you cannot print custom page sizes.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

When you print a custom page size to an Okidata ML-320 printer, a page size of
11 inches is printed regardless of the settings you specify. This affects all
Windows-based programs when you use the Windows 95 Oki9.drv driver.
