Q145606: List of Macintosh Processors

Article: Q145606
Product(s): Microsoft FoxPro
Version(s): MACINTOSH:2.5b,2.5c,2.6a,3.0b
Operating System(s): 
Keyword(s): kb3rdparty kbenv
Last Modified: 08-FEB-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c, 2.6a 
- Microsoft Visual FoxPro for Macintosh, version 3.0b 
-------------------------------------------------------------------------------

SUMMARY
=======

One of the System Requirements to run FoxPro for Macintosh is that the base
processor of the Macintosh is a 68020 or higher, or a PowerPC processor. Below
is a list of Macintosh models and the processor that they use.

NOTE: The products mentioned in this article are manufactured by vendors
independent of Microsoft; we make no warranty, implied or otherwise, regarding
these products' performance or reliability.
