SUMMARY
=======

The following article contains the directory of the 3.5-inch Microsoft Scenes
Stereogram Collection Disk 1 (Disk Assy 213-751-003). The Stereogram Collection
comes on two 3.5-inch disks. The program can be ordered on 5.25- inch disks by
using the 1.2 MB Disk Offer coupon that comes in the package.
