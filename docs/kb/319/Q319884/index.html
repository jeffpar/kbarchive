Q319884: XWEB: "Failed to Connect to Exchange Server" Err Msg Using OWA

Article: Q319884
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): kberrmsg
Last Modified: 06-AUG-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to log on to a Microsoft Exchange Server 5.5 computer by using
Microsoft Outlook Web Access (OWA), you may receive the following error
message:

  Failed to connect to the Microsoft Exchange Server, '<Server_name>'.

RESOLUTION
==========

To resolve this issue, install (or reapply, if it is already installed) Service
Pack 4 (SP4) for Microsoft Exchange Server 5.5.

To obtain this service pack, please visit the following Microsoft Web site:

  http://www.microsoft.com/exchange/downloads/55/sp4.asp
