Q161099: Secured Web Pages Not Loaded with Netscape Navigator 1.x

Article: Q161099
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 2,2.1
Operating System(s): 
Keyword(s): kb3rdparty kbnetwork osr2 win95kbfixlist
Last Modified: 25-MAR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

If Netscape Navigator version 1.x is used to access secured Web pages, the
background may be loaded, but individual graphic items linked to the page are
not loaded. The browser seems to continue to work, but never finishes the
loading of the page.

CAUSE
=====

This behavior can occur with Netscape Navigator version 1.x when you are using
the Microsoft TCP/IP protocol stack.

RESOLUTION
==========

To work around this behavior, use one of the following methods:

- Upgrade Netscape Navigator to version 2.x or later.

- Use a TCP/IP protocol stack from another vendor.


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.
