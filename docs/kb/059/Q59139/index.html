Q59139: Mac Srv Wkst: Time in Summary Window Displayed in 24-Hr Format

Article: Q59139
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 24-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The time appearing next to a Microsoft Mail message in the Summary window
reflects the time when the recipient's Mail server machine received the message.
This time format is the time format of the recipient's server as set in the
Control Panel under General. There is an option for "24 hour" or "12 hour" time.
The Summary window does not reflect any changes until you close and then reopen
the window.

Additional query words: 2.00 2.00a 2.00b 3.00
