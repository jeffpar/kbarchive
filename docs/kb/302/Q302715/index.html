Q302715: "Page Cannot Be Displayed" Error When Going Through Signup In MS

Article: Q302715
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 24-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

You install MSN Explorer 6.1 and choose to sign up for a new MSN Internet Access
account or to dial using an existing MSN Internet Access account. MSN Explorer
dials a toll free number to complete the sign up. If an error occurs in
connecting with the sign-up server, an error message may appear stating "Page
cannot be displayed".

CAUSE
=====

This is a known issue, and is most likely temporary.

RESOLUTION
==========

Close MSN Explorer, re-start MSN Explorer, and to go through Setup and sign up
again.

Additional query words: kbimu; MSN Explorer
