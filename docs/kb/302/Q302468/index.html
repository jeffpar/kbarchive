Q302468: Encarta: Err Msg When You Use Links in Encarta Web Center

Article: Q302468
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 29-JAN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Encyclopedia 2000 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click a hyperlink in the Web Center of any of the programs listed at
the beginning of this article, you may receive one of the following error
messages:

  Encarta Web Links Service
  Microsoft VBScript runtime error '800a000d' Type mismatch:
  'create_encarta_object' /encyclopedia/weblinks/ee98/functions.inc, line 39

-or-

  The site you requested is currently unavailable; please try again later.

CAUSE
=====

This behavior can occur as a result of server-related issues and does not
indicate a problem with your installation of Microsoft Encarta Encyclopedia.

RESOLUTION
==========

To work around this issue, use your Web browser to open the link listed in the
Web Center. To do this, follow these steps:

1. Make a note of (or copy) the Web site address listed in the Web Center search
  results. For example, if you search for "my fair lady" (without the quotation
  marks), the following Web-site address may be returned:

  http://www.foxhome.com/myfairlady

2. Start Microsoft Internet Explorer.

3. In the Address box, enter the Web-site address. For example, type or paste
  "http://www.foxhome.com/myfairlady" (without the quotation marks), and then
  press ENTER.
