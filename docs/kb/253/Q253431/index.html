Q253431: FS 2000: ATIS Frequency Broadcasts Only Current Weather

Article: Q253431
Product(s): Microsoft Home Games
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbdisplay kbui fsim kbimu
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you tune into an airport's ATIS frequency in Microsoft Flight Simulator
2000, you only receive the current weather information.

MORE INFORMATION
================

In Microsoft Flight Simulator 98, you receive additional information, such as
which airport runways are in use, as well as the current weather conditions when
you tune into an airport's ATIS frequency.

Additional query words: 1.00 msgame flightsim fltsim fs2k limited
