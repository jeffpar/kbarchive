Q261237: Visual FoxPro Sample: Selects Printer Using Kodak Image Control

Article: Q261237
Product(s): Microsoft FoxPro
Version(s): WINDOWS:5.0,5.0a,6.0
Operating System(s): 
Keyword(s): kbfile kbsample kbvfp500 kbvfp500a kbvfp600 kbGrpDSFox kbDSupport
Last Modified: 11-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

ImgAdmin.exe is a self-extracting executable file that contains a sample form
that illustrates how to select the printer by using the Kodak Image Admin
Control. The sample also uses the Kodak Image Edit Control as part of the
demonstration.

NOTE: The SET PRINTER TO command has no effect on Kodak Image Edit Control
PRINTIMAGE method.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

  ImgAdmin.exe
  (http://download.microsoft.com/download/vfox60/sample5/6/win98/en-us/ImgAdmin.exe)

Release Date: May-24-2000

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

The ImgAdmin.exe file contains the following files:

+---------------------+
| File Name    | Size | 
+---------------------+
| ImgAdmin.scx | 2KB  | 
+---------------------+
| ImgAdmin.sct | 10KB | 
+---------------------+
| ReadMe.txt   | 1KB  | 
+---------------------+

1. Download and decompress the sample.

2. Start Visual FoxPro.

3. Run the form (DO FORM IMGADMIN).

4. Click the command button labeled Get Image File (.TIF) and choose an image.

5. Click the command button labeled Printer Selection and choose a printer.

6. Click the command button labeled Print Image to print the image.
