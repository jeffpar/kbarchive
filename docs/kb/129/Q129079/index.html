Q129079: You Need Arabic Word to Read or Print Arabic SDK Documents

Article: Q129079
Product(s): Microsoft Windows Software Development Kit
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): 
Last Modified: 06-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Software Development Kit (SDK) 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The 16-bit Arabic Windows SDK and SDK documents are currently available on the
Microsoft Developer Network (MSDN) Development Platform (level II CDs).

The Arabic SDK documents stored in the ARA_SDK\DOCS\DOCS directory cannot be read
or printed by the U.S. version of Microsoft Word. They can only be read or
printed by the Arabic version of Microsoft Word. Also, the Arabic version of
Microsoft Word can be run only under Microsoft Arabic Windows, not U.S. Windows.
Microsoft Arabic Windows version 3.1 is available on MSDN Development Platform.

Additional query words: 3.10 word processor middle east me
