Q129386: PPT7: Internet Assistant for PowerPoint Available Online

Article: Q129386
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbfile kbinterop kbtool kbdta kbGraphxLink
Last Modified: 10-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Internet Assistant add-in for Microsoft PowerPoint 95 is now available from
Microsoft. The Internet Assistant add-in for PowerPoint can change a PowerPoint
presentation into a set of linked Hypertext Markup Language (HTML) documents.
HTML is the language of the World Wide Web. With Internet Assistant, you do not
need to work in HTML to create Web-ready documents. Internet Assistant does all
the HTML coding for you.

The following file is available for download from the Microsoft Download Center:

  DownloadDownload Pptia95.exe now
  (http://download.microsoft.com/download/powerpoint95/utility3/1/WIN98/EN-US/Pptia95.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

MORE INFORMATION
================

Installing the Internet Assistant for PowerPoint 95
---------------------------------------------------

After downloading Pptia95.exe, double-click this file to extract the following
files to the folder of your choice:

Files         | Description
======================================================================
Ia4ppt95.doc  | Documentation in Microsoft Word 95 for Windows format.
----------------------------------------------------------------------
Ia4ppt95.exe  | The installer program for the Internet Assistant.
----------------------------------------------------------------------
Ia4ppt95.htm  | Documentation in HTML format, you can read this with 
              | any Web browser.
----------------------------------------------------------------------
Readme.txt    | Instructions on how to install the Internet Assistant.

Follow the instructions in the Readme.txt file to install the Internet Assistant
for PowerPoint 95.

How Does the Internet Assistant Add-in Alter PowerPoint 95?
-----------------------------------------------------------

The Internet Assistant add-in adds the Export As HTML command to the File menu in
PowerPoint. You can create an HTML version of your presentation by clicking this
command, typing a name for your project, and then clicking OK.

What Does Internet Assistant for PowerPoint 95 Do?
--------------------------------------------------

When you click the Export As HTML command, two HTML versions of each slide are
created: one graphic version and one text version. The text version is useful
for Web browsers that can't view graphics -- in addition, a text version of a
slide is easier for a reader to "flip through" than a graphic version of a
slide. The graphic version of the slide retains the color scheme and appearance
of your original presentation.

Because the Internet Assistant add-in creates "client side image maps" of your
original slides, both versions of the slides maintain the PowerPoint interactive
settings that jump to other slides or documents. For example, if clicking an
oval on slide 1 takes you directly to slide 5 in the original presentation, that
functionality is retained in the HTML version of the slide.
