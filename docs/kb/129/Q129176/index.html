Q129176: Demonstration Mode Conflicts with Media Vision

Article: Q129176
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0,1.0a
Operating System(s): 
Keyword(s): win31
Last Modified: 21-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 500 Nations, version 1.0, on platform(s):
   - the operating system: Microsoft Windows NT, on platform(s):
      - the operating system: Microsoft Windows 
- Microsoft Oceans for Windows, version 1.0 
- Microsoft Dogs for Windows, versions 1.0, 1.0a 
- Microsoft World of Flight for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

To access the Demonstration mode in any of the products listed above, you must
press and hold down the CTRL key and ALT key while you press the D key
(CTRL+ALT+D).

This action conflicts with the CTRL+ALT+D key combination Media Vision sound
cards, Soundman 16, and Soundman Wave use to adjust sound card volume with the
Sound Mixer Control program.

WORKAROUND
==========

To control the audio volume on the Media Vision card, Soundman 16, and Soundman
Wave, press the CTRL+ALT+D key combination one time to start the Demonstration
Mode Setup. In the Demonstration mode dialog box, use the CTRL+ALT+D key
combination as many times as necessary to adjust sound card volume. Then press
OK to close the Demonstration Mode Setup.

of Microsoft; we make no warranty, implied or otherwise, regarding these
products' performance or reliability.
