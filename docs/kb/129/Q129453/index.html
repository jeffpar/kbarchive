Q129453: Explorapedia: Error Messages When Running Explorapedia

Article: Q129453
Product(s): Microsoft Home Kids Products
Version(s): 1.00
Operating System(s): 
Keyword(s): kbenv kberrmsg kbhw kbsound kbHardware
Last Modified: 24-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Explorapedia series: World of Nature for Windows, version 1.0 
- Microsoft Explorapedia series: World of People for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the error messages that you may receive when you run
Explorapedia and the Microsoft Knowledge Base articles that address the specific
problem.

NOTE: Explorapedia error messages that reference a number beginning with 8 or 4
refer to a problem with sound.

MORE INFORMATION
================

If you receive the following error message(s):

  Explorapedia error 800xxxxxx-x (or 48xxx-xx)

  An error occurred while starting Explorapedia. Restart Windows and try again.
  If that fails to correct the problem, run Setup from the Explorapedia CD-ROM
  again.

Please see the following article(s) in the Microsoft Knowledge Base:

  Q124812 Kids: Hangs With Ensoniq Sound Card

If you receive the following error message(s):

  Explorapedia error 80048212-0
  An error occurred while starting Explorapedia. Restart Windows and try again.
  If that fails to correct the problem, run Setup from the Explorapedia CD-ROM
  again.

Please see the following article(s) in the Microsoft Knowledge Base:

  Q126745 Explorapedia: ErrMsg: Error 8## (or 4##) When Starting Explorapedia

If you cannot hear Tad's narration, but you can hear other sounds in the program.
Please see the following article(s) in the Microsoft Knowledge Base:

  Q125957 Explorapedia: Cannot Hear Tad's Narration

If you receive the following error message(s):

  Cannot open .TDF file. Run setup again.

  -or-

  <Program Name> is unable to locate a necessary file. Make sure the
  <Program Name> is in your CD-ROM drive and try again. If the disk is in
  your drive and you continue to get this message, run setup again.

  -or-

  Cannot find file C:\Mskids\Leopard\Leopard.tdf

Please see the following article(s) in the Microsoft Knowledge Base:

  Q126006 Explorapedia: ErrMsg: Cannot Open .TDF; Unable to Locate

If you receive the following error message(s):

  Invalid TrueType font detected. An application used a TrueType font that
  caused an error in Windows.

Please see the following article(s) in the Microsoft Knowledge Base:

  Q131108 Kids: Invalid TrueType Font Detected

If you receive the following error message(s):

  Error accessing d:\ drive.

  -or-

  Internal error XXX...

  -or-

  Block verification error.

  -or-

  File copy operation failed...

  -or-

  Out of memory

  -or-

  Program Error: Terminating Application (This occurs with MSB Dinosaurs when
  accessing the Clues Book.)

Please see the following article(s) in the Microsoft Knowledge Base:

  Q124969 Kids: ErrMsg: Error Messages When Accessing CD-ROM Drive
