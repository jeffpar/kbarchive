Q300152: Troubleshooting Passport Using the Microsoft Windows NT Event Mo

Article: Q300152
Product(s): Microsoft Developer Network
Version(s): 1.3,1.4
Operating System(s): 
Keyword(s): 
Last Modified: 04-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Passport, versions 1.3, 1.4 
-------------------------------------------------------------------------------

SUMMARY
=======

You should review the Windows NT event log for information, warnings, and errors
that may have occurred during the operation of Passport features.

MORE INFORMATION
================

To view the Windows NT event log on your Web server, click Start, point to
Programs, point to Administrative Tools, and then click Event Viewer. View the
Application events on the server.

For additional information, compare any Passport-specific events with the event
reference on the following Passport Software Development Kit (SDK) Web site:

  Passport Manager Windows NT Events
  http://www.passport.com/sdkdocuments/sdk14/reference/operations/ntevents.htm

This information will assist you in locating the specific section of the SDK
document that is relevant to your issue.
