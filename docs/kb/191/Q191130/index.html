The information in this article applies to:

SUMMARY
=======

The following table lists fixed problems in Visual Basic for Windows, version
6.0. This is not a comprehensive list and is current as of August 7, 1998. For
more information about each issue, query in the Microsoft Knowledge Base on the
article identifier (ID).

For a more current fix list, go to the Support Highlights for Visual Basic at the
following location:

- http://support.microsoft.com/support/vbasic
