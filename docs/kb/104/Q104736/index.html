Q104736: System Hangs When DoubleDisk Gold Is Installed

Article: Q104736
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:6.0,6.2
Operating System(s): 
Keyword(s): 
Last Modified: 22-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 6.0, 6.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

This information applies to both Microsoft DoubleSpace and Microsoft DriveSpace.
For MS-DOS 6.22, use DRVSPACE in place of DBLSPACE for commands and filenames.

DoubleDisk Gold version 6.0 installs and functions correctly with MS-DOS 6.0.
However, after you install DoubleDisk Gold version 6.0 under MS-DOS 6.2, your
system may stop responding (hang) after displaying the following error message:

  DoubleDisk is examining your system

You may need to restart your computer several times to allow the installation
program to continue. After DoubleDisk Gold is installed, you may encounter
several incompatibilities with MS-DOS 6.2 that cause your system to hang.

RESOLUTION
==========


STATUS
======

Vertisoft has confirmed this to be a problem with DoubleDisk Gold version 6.0
and is working on a compatible version.

MORE INFORMATION
================

DoubleDisk Gold is manufactured by Vertisoft, a vendor independent of Microsoft;
we make no warranty, implied or otherwise, regarding this product's performance
or reliability.
