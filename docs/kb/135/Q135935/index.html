Q135935: MS Plus! Setup Causes GP Fault When DriveSpace Is Left Running

Article: Q135935
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Plus! for Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you run Microsoft Plus! for Windows 95 Setup while Windows 95 DriveSpace is
running and you do not shut down DriveSpace when you are prompted to, you
receive an error message similar to the following:

  DrvSpace caused a general protection fault in module USER.EXE at...

RESOLUTION
==========

Restart Windows 95 and run Microsoft Plus! Setup again. Make sure to shut down
DriveSpace when you are prompted to.
