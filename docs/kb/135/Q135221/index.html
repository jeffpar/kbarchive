Q135221: Cannot Display 256 Colors on Lexmark AR10

Article: Q135221
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are running Windows 95 on a Lexmark AR10 486slc computer, the Cirrus
Logic video drivers do not display the screen correctly if you are using 256 or
more colors. The screen may appear smaller than you expect and garbled.


RESOLUTION
==========

To work around this problem, use 16 colors, or install the video drivers
included with the Lexmark AR10 computer.
