Q135822: PPT7: Write-up Fails If "Show Field Codes" Turned On in Word

Article: Q135822
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbinterop kbole kbdta
Last Modified: 16-APR-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
- Microsoft Word for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use the Write-up feature in PowerPoint for Windows 95, version 7.0, while
the Show Field Codes option is active in Word for Windows, version 7.0, you may
encounter the following:

- Your presentation is not transferred as text to Word as expected.

- The insertion point does not move, though the menus may be available.

- The program appears to have stopped responding.

This problem does not occur if you use PowerPoint 97 for Windows and Word 97 for
Windows.

RESOLUTION
==========

You must hide field codes in Word 7.0 before using Write-up. To do this:

1. In Word for Windows, click Options on the Tools menu.

2. Click the View tab.

3. Clear the Show Field Codes check box.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
