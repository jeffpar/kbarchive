Q320484: XCCC: "HTTP 500.x" Error Messages in Outlook Web Access 5.5

Article: Q320484
Product(s): Microsoft Exchange
Version(s): 5.5
Operating System(s): 
Keyword(s): 
Last Modified: 31-JUL-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 
- Microsoft Outlook Web Access, version 5.5 Service Packs 1, 2, 3 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use Outlook Web Access (OWA), you may receive any of the following "HTTP
500" error messages:

  HTTP/1.0 500 - Internal Server Error

  -or-

  HTTP/1.0 500 - Server Error (a dynamic link library (DLL) initialization
  routine failed.)

  -or-

  HTTP 500.100 - Internal Server Error - ASP error
  Internet Information Services

RESOLUTION
==========

To resolve this problem, obtain the latest service pack for Exchange Server 5.5.
For additional information, click the following article number to view the
article in the Microsoft Knowledge Base:

  Q191014 How to Obtain the Latest Exchange Server 5.5 Service Pack

STATUS
======

Microsoft has confirmed that this is a problem in OWA 5.5. This problem was
first corrected in Exchange Server 5.5 Service Pack 4.
