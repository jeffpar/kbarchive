Q320013: Encarta Online Deluxe: "Does Not Recognize Passport" Err Msg

Article: Q320013
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbimu
Last Modified: 28-JUN-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Reference Library 2003, version 1.0 
- Microsoft Encarta Reference Library 2003 - DVD Edition 
- Microsoft Encarta Encyclopedia Standard 2003 
- Microsoft Encarta Encyclopedia Deluxe 2003 
- Microsoft Encarta Reference Library 2002 - DVD Edition 
- Microsoft Encarta Reference Library 2002, version 1.0 
- Microsoft Encarta Encyclopedia Standard 2002, version 1.0 
- Microsoft Encarta Encyclopedia Deluxe 2002, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you join Club Encarta and then use Microsoft Encarta Online Encyclopedia
to view an Encarta Online Deluxe article, you may receive an error message
similar to the following when you try to sign in to Encarta Online Deluxe:

  Encarta Online Deluxe does not recognize this Passport sign-in.

  If you have more than one Passport, your access to Encarta Online Deluxe may
  be associated with another Passport. Remember that your Hotmail and MSN
  e-mail addresses are also Passports.

CAUSE
=====

This behavior may occur for any of the following reasons:

- You have Encarta Encyclopedia Standard. Articles that are marked "Deluxe"
  require Deluxe access. Deluxe access is available only to those who have
  Microsoft Encarta Encyclopedia Deluxe or Microsoft Encarta Reference Library.

  -or-

- The Passport is not registered with Club Encarta. You may have more than one
  Passport (one that is registered and one that is not).

  -or-

- The Passport is not registered for use with Online Deluxe. You may have more
  than one Passport (one that is registered and one that is not).

RESOLUTION
==========

To resolve this issue, use one of the following methods, as appropriate to your
situation.

Encyclopedia Standard Users
---------------------------

Obtain Encarta Encyclopedia Deluxe or Encarta Reference Library. With Encarta
Encyclopedia Deluxe or Encarta Reference Library, you are eligible to view
Encarta Online Deluxe articles. For information about how to obtain these
products, please visit the following Microsoft Web site:

  http://encarta.msn.com/products/deluxe/signup.asp

Encyclopedia Deluxe or Reference Library Users
----------------------------------------------

If you have Encyclopedia Deluxe or Reference Library, you can re-sign up for
Online Deluxe. This is done by Joining (or rejoining) Club Encarta, and signing
up for the optional Encarta Online Deluxe membership. To join, follow these
steps:

1. Connect to the Internet.

2. In Encarta Encyclopedia Deluxe or Encarta Reference Library, on the Help
  menu, click Join Club Encarta.

3. Click Join Now!, and then click Sign-up Now!.

4. Returning users are prompted for their e-mail address. If you are prompted,
  enter the same e-mail address you entered the last time you accessed the Club
  Encarta page.

5. Verify your personal information, and then click Next.

6. On the Thanks page, click Sign-in, and sign into Passport. This sets your
  Online Deluxe membership with the current Passport ID.

NOTE: If Step 6 is not completed, even though you have successfully joined Club
Encarta, you will not have access to Encarta Online Deluxe.

MORE INFORMATION
================

Encarta Online Deluxe is an optional benefit to Club Encarta members. Even
without access to Encarta Online Deluxe, you can still view all of the standard
articles.

For more information about the Encarta Online products, visit the following MSN
Web site:

  http://encarta.msn.com
