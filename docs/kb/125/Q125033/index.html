Q125033: Writer/Artist: Unable to Record Sounds on Power Macintosh

Article: Q125033
Product(s): Microsoft Home Kids Products
Version(s): MACINTOSH:1.0,1.1
Operating System(s): 
Keyword(s): 
Last Modified: 29-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Creative Writer for the Macintosh, versions 1.0, 1.1 
- Microsoft Fine Artist for the Macintosh, versions 1.0, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to record sounds in Creative Writer or Fine Artist on a Power
Macintosh computer, Spike barks, and Creative Writer displays the following
error message:

  There's not enough room for this sound!

  You need to free some space.

1. Try to save your work--hold down "(Cmd key) and press S" (without the
  quotation marks)

2. Quit the program--hold down "(Cmd key) and press Q" (without the quotation
  marks)

3. Restart your computer.

Note that you receive this error message regardless of the amount of RAM you have
available or the amount of space on your hard disk drive.

CAUSE
=====

You receive this error message if you are using a Power Macintosh.

RESOLUTION
==========

There is no known resolution to this problem.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
