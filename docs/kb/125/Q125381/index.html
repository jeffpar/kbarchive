Q125381: Diamond Stealth 64 at 16.7 Million Colors Fails VGA Test

Article: Q125381
Product(s): Microsoft Home Multimedia Titles
Version(s): 1994 edition,1994-1995 edition
Operating System(s): 
Keyword(s): 
Last Modified: 11-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Complete Baseball for Windows, version 1994 edition 
- Microsoft Complete NBA Basketball for Windows 1994-1995 edition 
- Microsoft Bookshelf for Windows 1994 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When using the Diamond Stealth 64 video card configured for 16.7M colors, the
Multimedia Setup program returns one of the following error messages:

  - Requires a VGA or better display adapter.

  - Requires a VGA monitor supporting at least 256 colors.

RESOLUTION
==========

1. Change the display color depth to 256 or 64K colors

2. Run the Setup program for the applications listed at the top of this article.

3. Change the video display back to 16.7M colors.

MORE INFORMATION
================

This behavior has been reported with the Diamond Stealth 64 display driver
version 1.06 and Diamond InControl Tools (version 1.21 or 1.30).

In Windows 95, the color depth can be changed by following these steps:

1. Use the right mouse button to click the desktop and select Properties.

2. Click the Settings tab.

3. Select a different Color palette.

4. Click OK and restart Windows when prompted.

In Windows 3.1, the color depth can be changed by following these steps:

1. Run Control Panel and double-click the InControl Tools Icon.

2. Select the Display Option.

3. In the Color Depth list box, choose the desired depth.

4. Choose OK and restart Windows when prompted.

At color depths of 256, 64K or 16.7M, the Multimedia applications will run
correctly. Only the Setup program will fail when set to 16.7M colors.
