Q125379: 1995 Windows Encarta: Media Gallery Truncates Drop-Down Menu

Article: Q125379
Product(s): Microsoft Home Kids Products
Version(s): 1995 edition; WINDOWS:3.1x
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
- the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When the installed video driver is set to 640x480 resolution (without regard to
number of colors), the Media Gallery shifts the enlarged image on the screen
upwards and the drop-down menu does not appear. You will not be able choose
Close with the mouse.

NOTE: This problem does not occur in Windows 95.

RESOLUTION
==========

To close the enlarged screen:

- Press the ESC key.

  -or-

- Press and hold down the ALT key, and press the F4 key one time, and then
  release all keys.

STATUS
======

Microsoft has confirmed this to be a problem in the Encarta 1995 edition for
Windows. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.

================
