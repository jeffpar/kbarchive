Q195624: Virtual Globe 99: Empty Dialog Box Appears When Starting Program

Article: Q195624
Product(s): Microsoft Home Multimedia Titles
Version(s): 1.0
Operating System(s): 
Keyword(s): kberrmsg kbinterop kbimukbfaq
Last Modified: 24-APR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Encarta Virtual Globe 99, you may experience one of the
following symptoms:

- An empty dialog box with an OK button may appear on the screen.

- You may receive the following error message:

  Encarta Virtual Globe has detected that your Internet Explorer security
  settings are set to 'Custom'. Please set the security setting to 'Medium'.
  Otherwise, some portions of Encarta Virtual Globe will not run correctly.

When you click OK, Encarta Virtual Globe starts and appears to run correctly.
However, you may experience this symptom again any time you click a command on
the Features menu.

CAUSE
=====

This behavior can occur if the security setting in Microsoft Internet Explorer
is set to Custom or High.

RESOLUTION
==========

To resolve this issue, do one of the following methods:

Method 1

Set your Internet Explorer security setting to Medium. To do this, follow these
steps:

1. Quit Encarta Virtual Globe.

2. Click Start, point to Settings, and then click Control Panel.

3. Double-click the Internet icon.

4. On the Security tab, click Internet Zone in the Zone box.

5. Under Internet Zone, click Medium (More Secure), and then click OK.

6. Close Control Panel, and then restart Encarta Virtual Globe.

Method 2

Reset Internet Explorer Security Settings to Default. To do this, follow these
steps:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Internet Options.

3. On the Securitytab, click the Iternet zone option, and then change the
  security level to default.

4. On the Securitytab, click the Local Intranet zone option, and then change the
  security level to default.

5. On the Securitytab, click the Trusted Sites zone option, and then change the
  security level to default.

6. On the Securitytab, click the Restricted Sites zone option, and then change
  the security level to default.

7. Click OK, and then exit the Control Panel.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
99, version 1.0.
