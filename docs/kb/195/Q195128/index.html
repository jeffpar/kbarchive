Q195128: WD97: Converting WordStar 7.0 Files to Word for Windows Format

Article: Q195128
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta kbconversion winword word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

WordStar versions 5.0, 5.5, 6.0, and 7.0 share the same file format; however,
with each new version, additional features were added to the format. WordStar
4.0 has a different file format from WordStar 5.0, 5.5, 6.0, and 7.0.

Microsoft Word 97 supports WordStar 3.3, 3.5, 4.0, 5.0, 5.5, 6.0, and 7.0 file
formats. Word versions 7.0 and later also support importing WordStar 1.0 and 2.0
for Windows documents.

For additional information about obtaining the WordStar supplemental converter
for use with Word 97, click the article number below to view the article in the
Microsoft Knowledge Base:

  Q212265 WD: Additional Text Converters and Image Filters Available in
  Microsoft Office Converter Pack

MORE INFORMATION
================

New WordStar 7.0 features:

- Macros: WordStar 7.0 has a new macro system, which includes the following
  capabilities:

   - You can record a macro.

   - You can use the Basic-like macro language to create and edit macros to
     perform complex functions. And you can use the Single Step command to run
     a macro step-by-step for easy debugging.

   - You can run a macro using a hot key or the Play command.

- Integrated graphics conversion: WordStar converts the graphic into a .pix
  file format when you open a graphic in WordStar.

- Fax support: You can use a new command to create .pcx fax files.

- Windows Clipboard support: You can copy blocks of text up to 64 KB to and
  from the Microsoft Windows Clipboard.

- Mouse support: You can use a mouse in WordStar 7.0.

- EMS support: WordStar uses expanded memory (EMS) when it's available. You can
  set the level of EMS support using WSCHANGE (CC2G). When sufficient EMS is
  available, you can now run Preview, Inset, the thesaurus, or background
  printing when you have two windows open.

- New screen design: Redesigned pull-down menus, New Editing screen with style
  bar the mouse can change, vertical scroll bar, and so on.

- Hide Dot commands: Toggle command tags on and off (like the Word Show All
  feature).

- Print and Print Merge printing: The Print option allows you to choose between
  printing and merge printing when you print a file.

- Write a block to ASCII: You can choose to copy a block of text when you
  select Edit, Copy, To Another File in ASCII format.

- Re-mark or unmark a block: This feature marks the block that was marked
  before you marked the current one.

- Improved support for colors and shading: The list of colors or shades that
  appears shows correct choices for your printer.

- Preview improvements: Preview supports for more monitors and video modes.

- Improved paper bin support: Paper bins supported by your printer are listed.

- Go to a style tag: Go to the next paragraph style tag.

- Support for many new printers: Support for more than 550 printers.

- More WSCHANGE options. New WSCHANGE options have been added.

- TelMerge improvements: TelMerge supports the COM3 and COM4 ports on most
  systems.

- Star Exchange improvements: Also available are several new capabilities and
  support of additional file formats (dBASE IV, Lotus 1-2-3 version 3.0, and
  Apple Macintosh format).
