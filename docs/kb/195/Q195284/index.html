SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Running Microsoft Outlook 98, ISBN
1-57231-840-6.

The following topics are covered:

- Page 135-136: Mail format Must Be Set To HTML

MORE INFORMATION
================

Page 135-136: Mail format Must Be Set To HTML
---------------------------------------------

Page 135-136, "Sending a URL in a Message"

The dialog box on page 135 and steps 1 - 5 on the following page do not match
what you see on your screen unless your Outlook mail format is set to HTML.

To set the mail format to HTML, follow these steps:

1. On the Tools menu, click Options.

2. On the Mail Format tab, click HTML from the "Send in this message format"
  list.


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
