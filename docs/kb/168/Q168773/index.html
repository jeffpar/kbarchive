Q168773: SMSINST: Changes to Wizard Dialogs Do Not Appear Immediately

Article: Q168773
Product(s): Microsoft Systems Management Server
Version(s): WINDOWS:1.0; winnt:1.2
Operating System(s): 
Keyword(s): kbnetwork smsinst
Last Modified: 17-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server version 1.2 
- Microsoft Systems Management Server Installer version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Changes made to the Wizard dialog boxes, such as adding a radio button or making
a font change, are not reflected during the setup.

WORKAROUND
==========

The changes to the dialog boxes will be seen the next time you launch the
Systems Management Server Installer.
