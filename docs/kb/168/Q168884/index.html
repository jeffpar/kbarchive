Q168884: Service Receives TIMEOUT Error While Attempting To Load

Article: Q168884
Product(s): Microsoft Windows NT
Version(s): 3.51,4.0
Operating System(s): 
Keyword(s): kbnetwork
Last Modified: 25-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 3.51, 4.0 
- Microsoft Windows NT Server versions 3.51, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

A service will receive a timeout error if it takes more than 80 seconds to load.
A service timeout error can occur because of system problems that affect the
startup of a particular service, a third-party service, or perhaps a group of
services.

MORE INFORMATION
================

This is consistent in Windows NT 3.51 and Windows NT 4.0 Workstation and Server.
This is a hard-coded value and at this time is not user configurable.
