Q168858: XGEN: How to Obtain Exchange Server 5.0 U.S. Service Pack 2

Article: Q168858
Product(s): Microsoft Exchange
Version(s): winnt:5.0
Operating System(s): 
Keyword(s): kbusage exc5 kbgraphxlinkcritical
Last Modified: 21-DEC-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

THIS U.S. SERVICE PACK 2 IS FOR MICROSOFT EXCHANGE SERVER 5.0 ONLY!!

Microsoft Exchange Server version 5.0 U.S. Service Pack 2 is a cumulative build
of all fixes to Microsoft Exchange Server version 5.0.

For additional information about which bugs are fixed by this update, click the
article number below to view the article in the Microsoft Knowledge Base:

  Q168735 XGEN: README.TXT: Microsoft Exchange 5.0 U.S. Service Pack 2

Available Formats for Microsoft Exchange Server 5.0 Service Pack 2
------------------------------------------------------------------

Service Pack 2 is available as single self-extracting archive files for Intel
(x86), DEC Alpha, and Macintosh-based computers.

Ordering Microsoft Exchange Server 5.0 Service Pack 2
-----------------------------------------------------

If you do not have access to the electronic services listed below, you can order
Service Pack 2 (CD-ROM or Disk Set) from the Microsoft Sales Information Center
(MSIC) at (800) 426-9400.
