Q140665: Macintosh Cinemania 96: Manual Installation Instructions

Article: Q140665
Product(s): Microsoft Home Multimedia Titles
Version(s): 1996 edition
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Cinemania for the Macintosh 1996 edition 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article tells how to complete a manual installation for the 1996
edition of Microsoft Cinemania for the Macintosh.

MORE INFORMATION
================

Manually Installing Cinemania
-----------------------------

To manually install Cinemania, do the following:

1. Create the following folder on your hard drive:

  Cinemania 96

2. Insert the Cinemania compact disc into the CD-ROM drive.

3. Copy the following items from the Cinemania disc to the folder you created in
  step 1:

   - Cinemania 96
   - Microsoft Multimedia Catalog
   - Read Me

4. Locate and open the Install Files folder on the compact disc.

5. Locate and open the System Files folder on the compact disc.

6. Copy all the items in the System Files folder to your System folder on the
  hard drive. When you are prompted to place items into specific folders, click
  OK.

7. Close the System Files folder on the compact disc.

8. Locate and open the Fonts folder on the compact disc.

9. Copy the MS Reference font and the Arial font from the disc to the System
  Folder on your hard drive. When you are prompted to place items into the
  Fonts folder, click OK.

10. Close the Fonts folder on the compact disc.

11. Locate and open the Lists folder on the compact disc.

12. Copy the six items from the Lists folder to your Macintosh desktop. When you
  are prompted with the alert indicating items cannot be moved to the desktop,
  click OK. The following items are then copied to the Macintosh desktop:

   - Cool New Cinemania 96 Media
   - Ebert's All-Time Ten Greatest
   - Film Comment Editor's Favorites
   - Maltin's 10 Favorite Movies
   - Movies to Own
   - Movies to Rent

13. Close the Lists folder on the compact disc.

14. Close the Install Files folder on the compact disc.

15. Close the Cinemania 96 window.

16. On your hard drive, locate and open the System Folder.

17. Locate and double click the Preferences folder.

18. Create the following folder:

  Cinemania Folder

19. Copy the six items listed in step 12 above from your desktop to the
  Cinemania Folder.

20. Close all your windows and restart your computer.

21. Start Cinemania from the hard drive.
