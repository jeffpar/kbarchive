Q140491: Multimedia Sound Card Support Files Are Missing or Corrupt

Article: Q140491
Product(s): Microsoft Home Multimedia Titles
Version(s): 3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 01-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf 1996-97 for Windows 
- Microsoft Encarta 96 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install a Microsoft multimedia program, you receive the following error
message:

  Setup has detected that your computer is missing one or more of the necessary
  components for enjoying the full range of features of <Microsoft
  multimedia title>.

  For information on the problem and how to correct it, click Help. When you're
  ready to continue Setup, click OK.

  No sound card detected. No MIDI output device.

  <OK><HELP>


CAUSE
=====

The manufacturer-provided sound card software is not properly installed.

RESOLUTION
==========

To correct the problem, use one of the methods below to reinstall the sound card
drivers.

Method 1
--------

Reinstall the sound card drivers following the manufacturers instructions. This
usually involves running the installation program on the manufacturer- provided
disks. For more information, contact your hardware vendor.

Method 2
--------

Manually add and configure the sound card drivers as follows:

1. Double-click Control Panel, usually found in the Main group of Program
  Manager.

2. Double-click Drivers to see the list of installed devices.

3. If drivers for the sound card are already present, remove them with the
  Remove button, restart Windows, and then begin at Step 1 again.

4. Click Add. The Add window appears with Unlisted Or Update Driver selected.
  Click OK.

5. Insert the manufacturer-provided disk and click OK. Follow the on screen
  instructions. For information on how to configure the drivers, please see
  your sound card user's guide or contact the manufacturer.

MORE INFORMATION
================

When you install the software from your sound card manufacturer, files
supporting different types of sound are installed to your system.
