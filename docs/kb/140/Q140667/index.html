Q140667: Dinosaurs and MSB Ocean: Unable to Print to a Canon BJC-610

Article: Q140667
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0,1.0a
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbmm kbprint kbtlc kbPrinting kbHardwarekbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Dinosaurs for Windows, versions 1.0, 1.0a 
- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print from one of the programs listed at the top of this article to a
Canon BJC-610 printer using the vendor provided BJC-610 printer driver, the
Windows Printing System may display the Printing screen for 1 or 2 seconds, the
printer's online light may blink one time, and then nothing prints.


RESOLUTION
==========

Use one of the following color printer drivers:

- Canon BJC-600 printer driver versions 1.0 through 2.1

  -or-

- Epson LQ 2550 or Epson LQ 860 printer driver

When you are using emulation on the Canon BJC-610, set Dip Switch 11 to ON for
Epson emulation and OFF for Canon emulation. Refer to the printer manual for
further information on dip switches.


STATUS
======

Microsoft has confirmed this to be a problem in the programs listed at the top
of this article. We are researching this problem and will post new information
here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

The Canon BJC-610 printer driver consists of the Microsoft Windows Printing
System. When a print job is sent to the printer, the Printer Status screen is
displayed for the length of the print job.

You may be able to solve this problem by using the BJC-600 printer driver that
ships with Windows 95. For more information about installing a printer driver in
Windows 95, see your Windows printed documentation or online Help.

The third-party product discussed here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.

REFERENCES
==========

For more information about using, installing, or configuring the Canon printer
driver, please contact your hardware vendor, or contact Canon Technical Support
at (714) 437-7210.
