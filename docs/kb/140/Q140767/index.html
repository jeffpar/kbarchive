Q140767: Explorapedia: Clicking Taskbar Does Not Maximize Program

Article: Q140767
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0,95
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Explorapedia series: World of Nature for Windows, version 1.0 
- Microsoft Explorapedia series: World of People for Windows, version 1.0 
- the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Explorapedia in Windows 95, you cannot restore Explorapedia from
the taskbar if you minimize the program before you type in your name or select
your name from the list. In some cases, Explorapedia may not appear on the
taskbar.

RESOLUTION
==========

To resolve this problem, do one of the following:

Method 1
--------

With your right mouse button, click the Explorapedia icon on the taskbar, and
then click Restore.

-OR-

Method 2
--------

If the Explorapedia icon does not appear on the taskbar, do the following:

1. Press and hold down the CTRL key while you press the ALT and DEL keys
  (CTRL+ALT+DEL).

2. On the Close Program list, select Explorapedia.

3. Click End Task.

4. Restart Explorapedia.

The program should now run successfully.

MORE INFORMATION
================

This behavior does not occur in Microsoft Windows 3.x.
