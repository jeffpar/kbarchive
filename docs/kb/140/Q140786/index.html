Q140786: Which Products Qualify for PowerPoint for Windows 95 Upgrade

Article: Q140786
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kb3rdparty kbsetup
Last Modified: 16-APR-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

To install the competitive upgrade of PowerPoint for Windows 95, you must have
one of the products listed in this article.

MORE INFORMATION
================

During PowerPoint for Windows 95 Setup, the Microsoft Compliance Checking
Protocol searches for these products:

- Microsoft PowerPoint for Windows, versions 2.0, 3.0, and 4.0

- Microsoft Access, version 2.0

- Microsoft Word for Windows, versions 1.1a, 2.0, and 6.0

- Microsoft Excel for Windows, versions 2.0, 3.0, and 5.0

- Microsoft Office for Windows, versions 3.0, 4.x

- Harvard Graphics for Windows, version 1.1, and 3.0

- Harvard Graphics for MS-DOS, version 2.3

- Lotus 1-2-3, versions 1.0, 2.0, 3.1, 4.0, and 5.0

- Borland Quattro Pro, versions 1.0, 3.0, 4.0, and 6.0

- Lotus SmartSuite for Windows, version 3.0

- Lotus AmiPro for Windows, version 3.1

- Novell Perfect Office, version 3.0

- WordPerfect Presentations for Windows, version 2.0

- WordPerfect Presentations for MS-DOS, version 2.0

- WordPerfect for Windows, versions 6.0, 6.0a, and 6.1

- Freelance for MS-DOS, version 4.0

- Freelance for Windows, version 1.0

- Aldus Persuasion for Windows, versions 2.0, 2.1, 3.0

- Borland AmiPro, versions 2.0, 3.0, and 3.01

- Legacy, version 2.0

- WordStar, versions 1.5 and 5.5
