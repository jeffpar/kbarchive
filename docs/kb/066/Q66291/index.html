Q66291: Flight Simulator: Aircraft & Scenery Designer Features

Article: Q66291
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

Microsoft Flight Simulator Aircraft & Scenery Designer version 4.00 is
designed to work with Flight Simulator version 4.00. The package includes static
scenery design, dynamic scenery design, enhanced aircraft design, and four new
aircraft.

Aircraft & Scenery Designer is not an update to Flight Simulator; it is a
separate add-on product. For information on how to purchase Aircraft &
Scenery Designer, contact the Microsoft Information Center at 1 (800) 426-9400.

MORE INFORMATION
================

Aircraft & Scenery Designer includes the following features:

  Static scenery design:

     Create and edit custom scenery files or add to existing scenery.
     Static scenery design includes the following items:
       Roads
       Rivers
       Buildings, objects
          14 building shapes
          Control towers, towers
          Windsocks
          Bridges, suspension bridges
          Coniferous trees, deciduous trees
          Automobiles
          Fuel boxes
          Thermal generators
       Mountains
       Lines
       Polygons
       Runways
          Approach light systems (MALSR, SSALR, MALSF, SSALF,
          ALSF-I, ALSF-II)
       Navigational aids
          NDB, VOR, ILS, marker beacons
       Flight timing gates (for air racing)

  Dynamic scenery design:

     Create moving ground and air traffic. Choose from the following
     dynamic models:
        Cessna Skylane
        Gates Learjet
        Boeing 767
        Grumman F14
        Schweizer SGS 1-36
        Automobile (red, blue, or white)
        Airport fuel truck
        Sailboat

  Aircraft design:

     You can adjust more specifications than in Flight Simulator
     4.00. Flight characteristics and aerodynamics are adjustable using
     many new parameters.

  New aircraft:

     Four new types of aircraft are included:
     Boeing 747-400 airliner (enhanced instrument panel on EGA/VGA)
     Beechcraft Starship business jet
     Piper Cherokee Archer
     Cessna 182 Seaplane
