	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains a list of articles in the Microsoft Knowledge Base for
	bugs that are fixed in Systems Management Server (SMS) 2.0 Service Pack 2 (SP2).
	Service packs are cumulative. This means that the bugs fixed in a service pack
	are also fixed in subsequent service packs.
	
	In addition to the bugs listed in the "More Information" section of this article,
	the latest service pack for SMS 2.0 fixes the problems listed in the following
	Microsoft Knowledge Base article:
	
	  Q235991 SMS: Systems Management Server 2.0 Service Pack 1 Fixlist
	
	For additional information about obtaining the latest service pack for Systems
	Management Server 2.0, click the article number below to view the article in the
	Microsoft Knowledge Base:
	
	  Q236325 How to Obtain the Latest Systems Management Server 2.0 Service Pack
