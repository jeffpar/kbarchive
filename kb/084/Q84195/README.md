## Q84195: Hardware Options When Installing Windows 3.1

	Article: Q84195
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 27-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the hardware options offered during the installation of
	Microsoft Windows operating system version 3.1. If the Setup program offered any
	other choices for hardware, then you have a OEMx.INF file located in your
	WINDOWS\SYSTEM directory, or you are not using Microsoft Windows but an OEM
	(original equipment manufacturer) version of Windows. Additional support must be
	obtained from the manufacturer.
