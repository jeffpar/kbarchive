	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is the packing list from Disk 3 of the Microsoft MS-DOS 5.0a
	Upgrade 5.25-inch, 360K disk set. The Upgrade 360K Distribution Disk Layout
	lists the compressed files on the disks shipped with the product. The Upgrade
	360K Destination Disk Layout describes the working disks created when installing
	on floppy disks.
