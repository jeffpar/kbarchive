	Steps to View Contents of .CAB Files
	------------------------------------
	
	1. Copy EXTRACT.EXE to your hard drive from Disk 1.
	
	2. Put a disk with the .CAB file you wish to view in your floppy drive.
	
	3. Type the following at either an MS-DOS prompt or by choosing Run from the
	  File menu in Program Manager:
	
	  " EXTRACT /D <drive_letter>:\<filename>.CAB" (without the
	  quotation marks)
	
	For example, to view the contents of the VFP10.CAB file on Disk 10, with the
	floppy disk in drive A:, use this command:
	
	     EXTRACT /D A:\VFP10.CAB
	
	For more information about .DMF formatted floppy disks and .CAB files, please see
	the following articles in the Microsoft Knowledge Base:
